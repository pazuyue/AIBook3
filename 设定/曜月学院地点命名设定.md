# 《三月女神今天也只是普通学生》曜月学院地点命名设定

---

## 一、文件定位
这份文件统一曜月学院内部的重要地点命名，适配"正反都市"核心设定。

适用范围：
1. 正文场景描写
2. 第一卷大纲和细纲
3. 穿行、训练、比赛、封锁战的地点落名
4. 后续继续扩学院地图时的命名参考

优先原则：
1. 地点正式名优先按本文件来
2. 学生日常口语可以在正式名基础上自然缩写
3. 同一个地点别同时跑出三四套叫法
4. 正面和反面的同一位置可以有不同称呼

---

## 二、总命名原则
曜月学院的地点命名，统一分四层理解：

1. **正式旧称**
   - 用于留档、地图、导师口令、封锁记录

2. **制度用名**
   - 用于课表、穿行安排、训练公告、对抗场通知

3. **学生简称**
   - 用于日常交流和临场报动向

4. **战场称呼**
   - 用于章节标题、事件命名和读者记忆点强化

### 写法原则
1. 正式旧称要偏古旧，带异世界学院感
2. 制度用名要清楚，别太飘
3. 学生简称要顺口，方便喊
4. 战场称呼要有画面，最好一眼就记住
5. 反面地点在正面名称基础上加"反"字前缀或"影"字后缀

## 四、穿行路线命名

### 1. 银辉线
- **正式定位**：教学外围和操场边的基础穿行线
- **气质**：亮、稳、规整
- **学生简称**：银线
- **适用口径**：新生穿行、基础报动向、外圈明线

### 2. 铜锈线
- **正式定位**：器材库、实验楼北墙和旧巷之间的过渡带
- **气质**：旧、冷、带残留感
- **学生简称**：铜线
- **适用口径**：旧墙巡查、残体热手战、贴墙推进

### 3. 反面线
- **正式定位**：反面浮现期的高危穿行线
- **气质**：沉、冷、侵蚀失真
- **学生简称**：反线
- **适用口径**：反面穿行、怪异封堵、反面核心突入

### 4. 铁壁线
- **正式定位**：器材库外缘、外墙和后勤封锁带
- **气质**：硬、守、后勤秩序重
- **学生简称**：铁线
- **适用口径**：物资外环、防线巡查、封锁协防

## 五、训练场与公开对抗场命名

### 1. 复盘教室
- **对应旧称**：复盘教室
- **用途**：战后拆解、站位纠错、口令统一
- **气质**：亮、硬、压迫感重，不给人躲闪空间

### 2. 银阶回击场
- **对应旧称**：对抗场复训区
- **用途**：复盘后立刻重打、模拟再演、失误矫正
- **气质**：空、直，讲究打一遍错一遍就再来一遍

### 3. 银阶对撞庭
- **对应旧称**：银阶对抗场
- **用途**：公开对抗、晋位赛、实测首秀
- **气质**：亮、正、公开、全场都看着

### 4. 观战高席
- **对应旧称**：观战席
- **用途**：公开比赛和演武旁观、导师观察、同届围观
- **气质**：高、压、视线聚拢感很强

### 5. 积分晶屏前
- **对应旧称**：积分屏前
- **用途**：赛后结算、排名揭示、气氛收束
- **气质**：冷、亮，适合落结果

### 6. 穿行演武场
- **对应旧称**：穿行演练区
- **用途**：反面穿行演武、混战救场、公开训练
- **气质**：开阔、正反边界分明、适合多线混战

### 7. 鸣铁靶列
- **对应旧称**：训练标靶区
- **用途**：复盘课、拆招课、定点训练
- **气质**：硬、直，适合打节奏

### 8. 第七旧训楼
- **正式旧称**：旧训楼
- **制度用名**：第七旧训楼
- **学生简称**：旧训练楼
- **战场称呼**：反旧训楼、旧训楼反层
- **定位**：曜月学院早期废弃训练建筑，现已封存，是第一卷旧训练楼传闻与局部危机的核心场景
- **用途**：校园传闻、旧地图不一致、门牌错位、多出的门、正规核查任务、巡廊骸危机、撤离线断裂、异核与记录异常
- **气质**：废弃、熟悉又不对劲，像学生每天路过但没人愿意靠近的旧楼
- **写法提醒**：第一卷正文可让学生口语称"旧训练楼"；导师、任务栏、封锁令、档案记录优先使用"第七旧训楼"；进入反面或异常夹层后使用"反旧训楼"或"旧训楼反层"。它不是正在使用的银阶试炼场系列训练场地，而是废弃楼、校园传闻和异常入口。

## 六、反面区域命名

### 总则
反面是正面的镜像，但常规危险主要来自侵蚀污染、怪异聚集和感知失真。反面地点命名在正面名称基础上统一加"反"字前缀，学生口语可以用"影"字后缀。

### 1. 反教学区（暗堂教学区）
- **正面对应**：教学区
- **反面特征**：侵蚀失真，走廊方向感不可靠，黑板上的字会因污染残响自己动

### 2. 反实战区（影阶试炼场）
- **正面对应**：实战训练区
- **反面特征**：重力可能偏移，训练设施变成攻击性存在

### 3. 反宿舍区（暗月居）
- **正面对应**：住宿区
- **反面特征**：缝隙最密集区域，走廊无限延伸，门后可能不是自己的房间

### 4. 反以太区（逆以太中枢）
- **正面对应**：以太设施区
- **反面特征**：以太逆流，能量不稳定，适合设伏但极度危险

### 5. 反面核心
- **正面对应**：无（反面独有）
- **反面特征**：反面最深处，与三月权柄底层规则同源的规则中枢。越深入核心，侵蚀越强；只有核心、潮涌或终局级场景才允许出现空间、时间、因果错位

### 6. 反面边缘
- **正面对应**：正反重叠区
- **反面特征**：旧封锁区和反面高风险区。正面不表现反面异常；真正进入反面后才出现路径失真、实体敌人和侵蚀放大

### 1. 回纹书梯

- **用途**：追逐、上攀、卡位

### 2. 塔顶封存阁
- **对应旧称**：塔顶封存室
- **用途**：存放封存手册、守卫记录、旧档原件
- **气质**：高、静、锁得很死

### 3. 抄录侧厅
- **对应旧称**：旧档案塔抄录区
- **用途**：公开副本查档、低风险翻档
- **气质**：暗、静、纸页感强

### 4. 月匣档案库
- **对应旧称**：学院深层封存区
- **用途**：存放反面建造相关高密级档案、早期穿行制度记录、古脉秘史相关封存物
- **气质**：深、冷、封，带反面规则残留感

### 5. 月衡观测台
- **用途**：学院地下深处的旧时代观测遗迹，曾用于观测高位月相法则运行、记录三月断联前后的异常数据
- **气质**：古、静、精密，像一座废弃后仍在低频校准月相残响的观测站
- **写法提醒**：月衡观测台不是档案库，也不是反面核心。它负责"观测、记录、校准"的秘史证据功能；月匣档案库存放文本与封存物，反面核心是规则中枢。第一卷不得进入或正式命名月衡观测台，只能以"旧月/月相残片""旧仪轨残响"等低曝光表述轻触。

## 九、使用要求

后续正文、大纲、细纲统一守住：

1. 正面和反面地点不要混用称呼
2. 同一地点不要混用三四种叫法
3. 章节标题可以用战场称呼，正文里优先用正式旧称
4. 学生对话里可以用简称，但导师、留档、封锁令优先用正式名
5. 反面地点在正面名称基础上加"反"字前缀，学生口语可以用"影"字后缀
6. 涉及学院地下深处的旧时代观测遗迹时，正式地点名统一使用"月衡观测台"；若只是残留痕迹，不直接称地点，写作时用"旧月/月相残片""旧仪轨残响"等低曝光表述

## 十、反面战场动线设计原则

反面战斗的动线设计，统一遵循以下原则：

1. **从边缘到核心**：反面战斗动线必须层层深入，让读者清楚感觉到"越深入越危险"
2. **正反切换**：高手可以在战斗中短暂"切层"——从正面闪避到反面，或从反面突袭到正面
3. **危险递进**：越接近反面核心，路径错位、隐藏裂缝、怪异密度与长期侵蚀累积速度越高；核心能力和术式在短时战斗中正常运作，不写环境自动造成感知迟钝或神秘输出失真；空间/时间/因果错位只作为核心级异常
4. **险点设计**：每场反面战斗至少设置一个"慢半拍就会被吃掉"的险点
5. **收口位置**：每场反面战斗的收口必须落在明确的位置，不能模糊收束
