# 反面 vs 绝区零空洞设定对照参考

> 本文件用于写作和审稿时快速区分《三月女神今天也只是普通学生》的「反面」与绝区零式「空洞」表达，避免术语、空间逻辑和抗性机制串味。若本文件与 `设定/当前硬口径.md`、`设定/术语口径统一表.md`、`设定/世界观设定.md` 冲突，以前三者为准。

## 一、核心差异

| 项目 | 本书反面 | 绝区零式空洞风险写法 |
|---|---|---|
| 来源 | 上古封印松动后，上古黑暗侵蚀世界的物质化显化 | 像由某种组织或力量主动制造的空间 |
| 形成方式 | 被动显化，不是主动编织或制造 | 主动生成、投放、编织、扩张 |
| 空间关系 | 与正面共享地理位置，接管区域空间被搅乱成迷宫（一切都是真实的，但布局错乱），外层有蛋壳包裹，迷宫中隐藏空间裂缝（肉眼不可见，需靠近后用观测仪或探知类能力发现；不一定是出口，可能通向其他区域或死路） | 像在城市旁边另开一个独立副本 |
| 入口痕迹 | 入口凭空出现、肉眼可见；反面退去后不留灾害残留（白灰/冷凝/气味），但保留空间错位残留（建筑歪斜、走廊偏移、档案对不上） | 关闭后留下白灰、裂纹、气味、低温、壳层等痕迹 |
| 危害机制 | 空间搅乱成迷宫（真实但布局错乱）+蛋壳包裹困人+长期侵蚀（停留过久核心不稳定）。术式正常运作、感知正常、战斗不受削弱。裂缝不一定是出口，有陷阱属性（不慎踩入会被传送，另一端不可预测），但也可能是机会（通向目的地附近可抄近路）；不会密集分布不影响战斗。需探知类能力（瞬间分析）或反面观测仪（三到五分钟计算）判断去向 | 毒雾式污染、以太抗性数值化、固定副本灾害 |
| 资源定位 | 反面资源需采集、封装、净化、鉴定、分级、监管使用 | 反面等同以太唯一产地 |
| 认知呈现 | 前中期公开认知是「凭空出现、原因不明」 | 角色过早知道完整底层 |

## 二、禁用表达

以下表达原则上不进入正文和规划，除非是在明确说明「旧稿问题」：

- 反面被某人制造、编织、投放。
- 城市被吸进反面、被吸进异界。
- 反面入口关闭后留下白灰壳层、干燥裂纹、冷凝水痕、墙体温度、气味、气息、怪声。但不禁止空间错位残留（建筑歪斜、走廊偏移、门方向变化、档案对不上）。
- 以太抗性数值、抗性分数、装备抗性等级、防护百分比。
- 以太污染、反面以太污染。
- 反面是所有异常的总源头。
- 普通信息、论坛图、地图文字让主角指尖发凉或身体回响。

## 三、推荐替代表达

| 风险表达 | 替代表达 |
|---|---|
| 反面被制造/编织 | 反面被动显化、封印松动后上古黑暗侵蚀世界 |
| 城市被吸进反面 | 原世界该区域被反面规则改写覆盖 |
| 空洞入口残留 | 反面退去后不留灾害残留（白灰/冷凝/气味）；证据落在空间错位残留（建筑歪斜、走廊偏移）、记录、权限、路线、门牌、访问缺口 |
| 以太抗性数值 | 抗性三要素：核心偏向契合度、规则认知深度、防护工具辅助 |
| 反面污染 | 反面侵蚀（长期停留后核心不稳定）、空间搅乱成迷宫+蛋壳包裹（裂缝不一定是出口，需探知类能力瞬间分析或反面观测仪三到五分钟计算判断去向） |
| 资源来自反面 | 反面资源是以太结晶等多种形态之一，需正规处理后才能使用 |

## 四、第一卷执行提醒

1. 第一卷学院安全区正常运转，危险来自校外、反面、怪异事件和学院监管下的旧训练楼封印区。
2. 旧训练楼不是普通废楼，也不是校园里随地异常的证明，而是学院监管下的高风险例外。
3. 第一卷证据优先落在异常访问缺口、公共终端缺口、任务记录不一致、门牌/地图/备案错位和违规训练记录，不靠入口物理残留。
4. 影子训练场骗局的诱惑来自正规反面训练确实有效，陆沉将其偷换成「更深压力能让术式提前成型/成长」。
5. 污染进阶不是幻觉，也不是外挂；它真的会让未成型者短暂看见第一式，让已成型者短暂摸到下一形态，但形态由反面压力参与催生，长期扭曲神秘核心成长方向。
