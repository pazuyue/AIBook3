# 第一卷执行导航

> 本目录负责《银序一班篇》的实际生产。现行正文章号、已完成进度和下一章任务只以“第一卷当前写作状态”为准。

## 一、写作前读取顺序

1. [当前硬口径.md](../../当前硬口径.md) + [第一卷当前写作状态.md](第一卷当前写作状态.md)  
   先确认当前覆盖项、禁写边界、唯一现行正文章号、章↔节拍映射、上一章结尾、下一章任务和活跃伏笔。
2. [write_brief_cache.md](write_brief_cache.md)  
   当前章人工写作卡；属于运行态缓存，只约束本次写作，不得改写设定事实。
3. [第一卷写作设定包.md](第一卷写作设定包.md) + [第一卷口径卡片.md](第一卷口径卡片.md)  
   第一卷可见范围、人物声轨、场景执行和快速检查。若与当前硬口径或状态文件冲突，以当前硬口径和当前写作状态为准。
4. [第一卷分卷大纲.md](第一卷分卷大纲.md)  
   卷定位、阶段功能、角色弧、核心冲突与卷末达成。
5. [第一卷事件链.md](第一卷事件链.md)  
   节拍因果、事件前后关系和后续规划母版。
6. 必要时读取专项设定、人物文件和两份能力归档。

## 三、冲突规则

1. 实际正文章号、已写事实：当前写作状态为准。
2. 卷级规划四件套互相冲突：分卷大纲 → 事件链 → 节奏表 → 钩子统计。
3. 规划文件中的 `ChN` / `第N章` 默认表示规划节拍，不表示现行正文章号；只有状态文件的映射表能把两者对应起来。
4. 写作设定包和口径卡片只能压缩或收紧全局设定，不能创建另一套能力、术语或世界机制。
5. 缓存只服务一次流程，不进入长期裁决链。
