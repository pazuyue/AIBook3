## 说明
这张表专门管全书五条核心秘史怎么按卷往前推。

这里只记三件事：
**第几卷、通过什么人/事件/遗迹、揭到什么程度。**

不在这里写太细，是为了把悬念留给各卷大纲和正文自己去开。

## 关键场景定义

**月衡观测台**：学院地下深处的旧时代观测遗迹，原为上古时期监测高位月相法则运行状态的观测站。观测台废弃后，残存的法则碎片和旧仪轨残响仍会对特定位格者产生共鸣反应。

> **载体偏差备注**：第一卷实际落地时，"月衡观测台"的叙事功能只由月相仪白光延迟和旧训练楼记录/权限异常轻触承载，不正式命名地点。后续卷如需启用月衡观测台作为独立场景，需注意与第一卷已完成的"月相仪异常回应"做好区分，避免功能重复。

## 二、神秘体系起源

### 写作者统一解释

神秘体系起源必须按四层理解，后续揭露只是认知逐层升级，不是推翻前一层：

1. **边界来源**：三月女神曾把上古混乱超凡整理成可理解、可传承的秩序雏形，因此神秘体系带有“被设计过的边界”。
2. **形成机制**：三月隐没后，世界以残留秩序、碎裂权柄和自身修复机制长出供凡人使用的替代规则，因此它也是“世界自行长出的体系”。
3. **本体定性**：这套替代规则在高层本质上是三月权柄碎裂后的低维投影，所以天然存在无法越过原件的上限。
4. **非直接创造**：三月没有逐一创造所有神秘核心、能力和术式；个体神秘仍由人的认知、执念与共鸣形成。

| 卷 | 揭露事件/途径 | 揭露程度 |
|----|---------------|----------|
| 第一卷 | 入学实测、第一堂课、实战测试、积分任务和旧训练楼核查这些校园场景里自然展示神秘体系怎么运作；苏弥从旧档案、任务记录和权限缺口里发现旧训练楼事件不像普通事故 | 先让读者看见这套体系存在、能打、能分层，同时点出"旧训练楼异常不是纯自然事故" |
| 第二卷 | 方导师把神秘三要素往更深一层讲开 | 把"神秘体系不是自然法则，而是被设计出来的规则"这层先点出来 |
| 第三卷 | 月衡观测台的旧仪轨残响通过课堂、旧设施故障和日常异常反复露边；不正式进入观测台深层 | 积累"神秘体系底层规则可能和三月权柄同源"的证据，只让角色觉得现有教材解释不完整 |
| 第四卷 | 主角在高位战斗中神秘术式首次触顶——无论怎么提升，神秘体系的输出存在一个无法突破的绝对上限；方导师结合自身研究确认这不是个人瓶颈，而是体系天花板 | 确认神秘体系其实是三月隐没后，世界自己长出来的替代性规则，天然有上限 |
| 第五卷 | 主角在公会任务中术式反复触顶，体系上限成为实战瓶颈；方导师在公会层面接触到更多旧时代研究资料，开始怀疑"上限不是缺陷而是设计" | 体系天花板从"现象"推进到"可能是故意的"——但具体为什么故意的，尚不确定 |
| 第六卷 | 远征中接触旧时代封印点，主角在链接三月法则时直接感知神秘体系的形成机制 | 最终揭开：神秘体系其实是三月权柄碎裂后的低维投影——它天然有上限，因为投影不可能高于原件 |
| 第七卷 | 三月归一，完整权柄重新接通，神秘体系在新秩序中归位 | 不再大段解释神秘体系本质；重点落在选择：它作为"凡人可用的低维接口"被保留，并被重构到不再失控 |

## 三、怪异诞生

### 写作者统一解释

怪异诞生必须区分“终极规则原因”和“近因诞生路径”：

1. **终极规则原因**：上古黑暗对三月法则的腐蚀，使世界中出现了可以持续生成腐化异常的规则缺口。
2. **近因诞生路径**：晋升失败、负面能量汇聚、反面侵蚀、特殊地域、非法实验和幕后者逆向工程，都可以成为具体怪异的直接诞生原因。
3. **非直接同源**：不是每一只现世怪异都直接来自上古黑暗残层或反面；多数怪异只通过世界已被腐蚀的底层条件间接关联终极源头。

| 卷 | 揭露事件/途径 | 揭露程度 |
|----|---------------|----------|
| 第一卷 | 通过巡廊骸和旧训练楼异常夹层先让低阶反面实体站上台面；异核和记录/权限异常只作为跨卷线索，不确认人为投放机制 | 先展示反面实体确实存在、有明确战斗性，同时保留"异常背后有问题"的疑问 |
| 第二卷 | 苏弥在旧图书塔翻到怪异诞生记录 | 确认怪异不是自然产物，而是"神秘提升失败 + 负面能量汇聚"的结果 |
| 第三卷 | 学院日常任务中遭遇带有古老特征的异常怪异残型，外观和已知怪异分类对不上 | 把怪异往更古老、更危险的源头轻推一层，但不命名"原始怪异"，不正式确认源头 |
| 第四卷 | 幕后推手展示怪异逆向工程能力——不仅能制造，还能精确控制怪异的等级、类型和腐蚀方向 | 确认怪异的诞生机制可以被逆向工程——怪异不是随机腐化产物，而是可以被定向设计的规则武器 |
| 第五卷 | 公会任务中遭遇更复杂的怪异形态——某些怪异的出现位置和时机明显是被精确投放的，而且等级远超"神秘提升失败"的解释范围 | 怪异诞生的"随机腐化"解释已经不够用了——存在更古老、更系统的诞生源 |
| 第六卷 | 远征中发现怪异的源头与上古黑暗的直接关联线索——某些旧时代封印点周围的怪异形态明显是"被腐蚀的三月法则碎片" | 怪异诞生的源头指向上古黑暗对三月法则的腐蚀，但具体机制尚未完全确认 |
| 第七卷 | 上古黑暗源头和怪异关系在终局中被验证 | 第六卷已基本揭示源头关联，第七卷只验证最终定性：怪异是上古黑暗侵蚀三月法则后的腐化产物，是世界规则出问题的证据 |

### 三月继承者暗线揭示边界

- 第一至三卷只埋旧月残片、旧碑铭、残缺短句、异常同频感和极模糊外界前史梦影，不确认主角与三月权柄的真实关系。
- 第四至五卷可以让主角和核心圈层意识到她与三月高位法则不是普通承载关系，但不能写成旧人格苏醒，也不能写成三月女神转世。
- 第六卷确认过去的三月女神主动死亡、拆解位格、留下封印和规则遗嘱；同时确认她没有安排自己复活，也没有把未来当成回归容器。
- 第六卷同时确认归一之躯来源：它是三月世界在漫长封印中自行孕育的空白神性承载体，不是旧神遗体，也不是来源世界肉身。
- 第七卷确认主角是被三月世界残余权柄因相性吸引来的外界继承者。她接过三月，是继承守护与选择权，不是恢复旧神记忆。

## 核心定性备忘
这几条是最后不能写歪的底层定性：

- **上古黑暗**：它不是一个具体boss，而是法则失衡必然长出来的结果。终局战的本质不是"打死一个东西"，而是主角重新立住秩序，让失衡回到平衡。
- **三月隐没**：这是主动牺牲，不是被动失败。三月选择离开，是为了保住世界；主角接过权柄，是接着把这件事做完。过去的三月女神已经死去，主角不是旧神复活容器，也不是三月神性核心在未来重新长出的新人格，而是被三月世界残余权柄因相性吸引来的外界继承者。
- **八律起源**：八律是三月创世时按分工造出来的，不是自然演化。八律秘史真正的冲突，会落在"守护使命"和"自主意志"之间。
- **神秘体系**：神秘体系不是独立长出来的一套完整上位体系，它本质上是三月权柄碎裂后的低维投影，所以它天然有上限。
- **怪异诞生**：怪异不是自然界随便冒出来的东西，而是上古黑暗腐蚀三月法则后的腐化结果。怪异本身，就是世界规则出问题的证据。
- **五线关系**：五条秘史不是五条独立的故事线，而是同一个创世→守护→侵蚀→隐没→投影→腐化闭环的不同切面。任何一条线的揭露，都必须考虑对其他四条线的连带影响。终局不是五条线分别收束，而是五条秘史线和七条故事线在同一个事件里同时归位。
