## 一、世界概貌

故事发生在第八纪元「恒月纪元」。

这个世界表层秩序仍在运转：百城、学院、八律、公会、远征组织和普通社会都还存在。人们上课、工作、交易、执行任务，也会争资源、争名额、争话语权。深层危机则持续加重：怪异事件变多，旧时代遗迹外翻，反面活跃度上升，世界更高层的神性秩序长期处在松动边缘。

写作基调不是末世废土，而是「秩序仍在，但秩序正在付出代价」。普通日常必须真实存在，危机才有重量。

### 世界特征

- 表层秩序：百城、学院、八律和公会维持文明运行。
- 深层危机：怪异、反面、旧时代遗迹、神秘失控和现实利益冲突同时存在。
- 核心矛盾：普通社会想继续生活，高位规则却在松动。
- 主角定位：哥伦比娅不是旧神复活，也不是旧神容器，而是被三月世界残余权柄因相性吸引来的外界继承者。

## 二、三层超凡体系

本世界的超凡结构分三层，不得混写。

| 层级 | 内容 | 正文边界 |
|---|---|---|
| 主流超凡层 | 神秘核心、核心技、通用术式、以太载体、学院训练、公会任务 | 大多数战斗和成长都发生在这一层 |
| 异常生态层 | 怪异、反面、侵蚀体、旧时代遗迹、神秘失控事件 | 不是所有异常都来自反面 |
| 神性月权层 | 三月权柄、三月女神、归一之躯、月相锁定、终局裁定 | 前中期只低频外泄，不作为普通神秘体系上级来写 |

### 执行红线

1. 神秘体系不是月亮体系的下级，也不是三月权柄创造出的公共职业系统。
2. 普通神秘者以唯一神秘核心承载个人法则，以核心技表达个人能力；可教学、可复现的公共能力才称为术式。
3. 以太是世界通用能量载体，不是神秘核心，也不是反面专属物质。
4. 月相可以影响反面浮现、边界稳定、旧月残片和神秘输出稳定性，但不能写成所有普通神秘者的统一能量来源。
5. 常规成长链固定为「核心回响→核心显相→真名解放→界临→立序」，分别对应神秘雏形、神秘者、深律者、域主、秩序者；详细阶段与晋升条件以《神秘设定》为准。
6. 通用术式分攻式、禁式、护式、行式、察式、净式、阵式七类，纳入七十二标准式谱；编号表示构筑复杂度与许可等级，不等同于单纯威力排名。
7. 哥伦比娅的三月权能不属于常规神秘体系。旁人可以误判为稀有冰霜核心或无吟唱术式，真实发动则是零核心激活、零以太调用、零术式构筑。

## 三、非反面异常生态

反面是重要主线，但不是所有异常的总源头。每卷至少保留一条不依赖反面和三月的异常或现实冲突线，避免世界观收窄成「一切都是反面」。

| 来源 | 写法锚点 | 与反面的边界 |
|---|---|---|
| 自然怪异 | 公会委托、城市边缘事件、民间传闻 | 可以和反面无关，不默认同源 |
| 神秘失控事件 | 术式走火、核心崩解、旧术式反噬、训练事故 | 是神秘体系自身风险，不等于反面侵蚀 |
| 旧时代遗迹 | 旧设施、旧档案、旧训练法、失效封印 | 公开认知里不直接绑定三月或反面 |
| 八律与古脉冲突 | 古老传承、圣物许可、地脉资源、氏族责任 | 现实利益和历史责任驱动，不必然有反面参与 |
| 城市与公会利益冲突 | 委托竞争、资格考核、资源分配、政治摩擦 | 可以没有怪异，也可以只有常规怪异参与 |
| 反面相关事件 | 入口、侵蚀、局部浮现、旧训楼反层、封印边界 | 只解释明确参与的部分，不外扩成万能答案 |

### 跨卷非反面异常线

| 卷 | 必保留线 | 用途 |
|---|---|---|
| 第一卷 | 核心显相/开发焦虑、训练资源压力、陆沉核心崩解 | 校园制度压力与个人选择，不全靠反面解释 |
| 第二卷 | 自然怪异或学院深层权限任务 | 扩展学院以外的异常生态 |
| 第三卷 | 学院日常中的术式走火、旧校规复核、八律同学传统冲突 | 日常旧痕与变身认同线托底 |
| 第四卷 | 学院准降临危机外的旧设施和高层认知冲突 | 中段撞线但不全揭底 |
| 第五卷 | 公会派系、考核、经济和政治摩擦 | 公会立足线的现实阻力 |
| 第六卷 | 八律领地遗迹、古脉政治、远征清理 | 远征真相线的多来源冲突 |
| 第七卷 | 百城与八律的新秩序重排 | 终局后的现实余波 |

严禁反写：非反面异常不能被统一改成反面的衍生物。它们可以和上古黑暗有更深层终极关联，但在叙事机制上不通过反面渠道。

## 四、三月女神与主角真实身份

上古时期，三月女神是三位独立存在：

| 月相 | 权柄方向 | 叙事关键词 |
|---|---|---|
| 霜月 | 现在、静止、冻结当前状态 | 眼前一瞬、终止推进、冷白反馈 |
| 恒月 | 过去、稳固、补回旧态 | 保场、钉局、修复、承接 |
| 虹月 | 未来、分歧、终焉与错认 | 斩断未来、制造误判、切断连锁 |

三位女神以死亡完成上古封印，并主动拆解位格，把三月权柄碎片留在封印内部作为承重锚点。三位女神已经死去，没有复活计划。

哥伦比娅不是旧三月女神复活，不是旧神人格容器，也不是三月神性核心重新长出的新人格。她是外来灵魂，被三月世界残余权柄因相性吸引，降临到三月世界自行孕育出的归一之躯中。她是历史上第一个一人承接三月全部权柄的存在。

### 归一之躯

归一之躯不是来源世界原身，不是旧女神遗体，也不是有原人格的身体。它无灵魂、无原人格、无旧神记忆，只提供承载三月归一的神性接口。它读取外来灵魂本相后，显现为哥伦比娅的外貌。

写作边界：

1. 第一卷正文不得直出「归一之躯」「三月权柄」「霜月碎片」等底层身份词。
2. 异常只写成体感、仪器误判、旁人无法解释的结果。
3. 主角知道自己有异常，也知道要收住；不写成懵懂失控或被老师真正看穿。

## 五、反面真实底层

反面不是任何角色、组织或力量主动制造/编织出来的空间。

真实底层：三月女神以死亡完成的上古封印因长期松动而衰退，被封印的上古黑暗侵蚀世界。侵蚀在世界表面物质化后，形成与正面不同步的异常规则空间，即「反面」。

### 承重锚点

三月权柄碎片是封印的承重锚点。承重最强的位置，正是松动最快、侵蚀最强的位置。因此三月权柄碎片与反面底层规则同源，这里的「同源」指承重结构与侵蚀面共享规则基础，不是反面由权柄主动制造。

### 权柄回收的矛盾效应

主角逐步收回权柄时，会出现矛盾现象：

1. 收回单个权柄碎片等于让该承重锚点暂时脱开封印，局部松动速率上升。
2. 局部松动会让反面活跃度、侵蚀强度或局部浮现风险上升。
3. 只有全部回收并三月归一，才能重新锚定整个封印，彻底重置松动。

半途而废最危险：松了一半的锁比完全锁死或完全打开都更不稳定。它不是反面有意识地变强，而是规则层面的物理连锁。

### 阶段性松动

松动不是匀速的，关键节点会加速：

- 权柄回收时，承重锚点脱开，局部松动加速。
- 月相旬切换时，完整力量对承重锚点的支撑强弱变化，弱时松动加剧。
- 上古黑暗存在活跃周期，某些纪元阶段更容易侵蚀世界。

第一卷至第四卷不主动回收权柄，只存在月相切换和活跃周期带来的轻度加速；第五卷开始权柄回收推进，加速机制正式进入剧情主线；第七卷归一时重新锚定。

## 六、反面空间口径

反面与正面共享地理位置。被反面接管/改写的区域不是被吸到另一个世界，而是在原世界那一块位置上空间被搅乱了——一切都是真实的，建筑、路、地形都还在，但整个布局像一个被打碎又重新拼接的迷宫。眼睛看到的就是实际存在的，没有幻觉。反面会像一层蛋壳一样把整个区域罩住——你能看到外面，但出不去。迷宫中隐藏着不可见的空间裂缝——肉眼不可见，感知不到，不靠近不会知道它在那里，普通人走到面前也只会觉得是一面墙。裂缝还有陷阱属性——如果不慎踩入，会被传送到裂缝另一端，而另一端的环境不可预测，可能是迷宫的其他区域，可能是空中，可能是水下。但裂缝也可能是机会——有些通向目的地附近，踩对了就能抄近路。裂缝不会密集分布，不足以影响正常的战斗和行动，但你不知道踩下去的是陷阱还是捷径。裂缝不一定是出口——有些通向迷宫的其他区域，有些是死路，只有极少数通向外面。发现和分析裂缝需要反面观测仪（类似萝卜的工具，近距离对准裂缝分析，三到五分钟得出结果）或空间探知类能力（如林见星的星丝，十步外即可感知，十秒内分析去向）。判断裂缝通向哪里是在反面中生存的核心技能。进去容易，出来难——不是出不来，是找不到对的裂缝。反面内核心能力与通用术式均正常运作，感知和战斗不受即时削弱；侵蚀是长期停留的累积后果，不是即时效果。反面退去后不留灾害残留（白灰/冷凝/气味），但保留空间错位残留（建筑歪斜、走廊偏移、档案对不上）。

### 入口硬约束

1. 反面入口在正面现场凭空出现，肉眼可见。公众能看见，不一定理解本质。
2. 反面退去后不留下白灰壳层、干燥裂纹、冷凝水痕、墙体温度、气味、气息、怪声等可被取样的灾害残留。但允许空间错位残留——建筑歪斜、走廊偏移、门方向变化、实地校验对不上、档案对不上。

### 学院安全区

曜月学院和学院城建立在安全区。宿舍、教室、食堂、正规训练场、任务平台默认代表秩序、保护、训练和日常。旧训练楼等封印区是学院监管下的高风险例外，不等于学院建在高风险区。

### 反面训练

正规受控反面实训可以有效训练学生：

- 轻度耳压、方向感微偏、空气质感变化是基础体感，不是攻击。
- 反面内核心能力与通用术式均正常运作，战斗不受即时削弱。即时危险来自空间扭曲（迷路、重力异常、建筑位移）和反面怪异本身。
- 适配度差异体现在对空间扭曲的适应速度上，高适配学生调适更快，低适配学生更吃力。

影子训练场骗局之所以有效，是因为正规反面训练真的有用。诱导链把「导师监管下的安全训练」偷换成「更深压力一定更有效」，再把学生带向污染显相、错误真名与伪解放。

## 七、侵蚀与怪异化

反面内部存在侵蚀。侵蚀不是毒雾，也不是普通脏污，而是长期停留后反面底层规则对神秘核心造成的累积影响。短时间在反面内战斗不受侵蚀影响；侵蚀是时间累积的后果，不是即时效果。

第一卷使用边界：

1. 普通学生最多写到轻度或中度侵蚀，能被救回来。
2. 第一卷允许一例完整怪异化，作为旧训练楼局部封印危机的直接触发点。
3. 不让普通学生频繁怪异化，否则学院安全秩序会失真。
4. 完整怪异化案例不可复用，周砺结局按当前硬口径执行：被镇压后未死亡，但完全怪异化不可逆，转入封存收容。

## 八、反面认知梯度

反面的真实来源和各角色的认知层级必须分开。正文只能调用角色当前层级，不得把真实底层提前写成角色已知事实。

| 层级 | 持有者 | 可用语言 | 卷次边界 |
|---|---|---|---|
| 公开认知 | 民众、普通学生、绝大多数神秘者 | 凭空出现、原因不明、跟旧月残片/旧封印有某种关联但没定论 | 第一卷至第三卷公开场合唯一口径 |
| 学院高层私见 | 院长、季悬灯、方导师级 | 旧时代封印、法则性力量、可能与三月权柄同源等推测 | 第四卷起可在私下短暂出现 |
| 核心团队推测 | 苏弥、季悬灯、林见星等 | 权柄回收=松锁=升级、异核投放像某种法则性力量在操作 | 第四至第六卷逐步形成 |
| 主角独有认知 | 哥伦比娅 | 反面与三月权柄存在同源关联，但无法提前确认完整真相 | 第一卷以 Ch1 一次性被动外泄和 Ch11-13 主动低频反馈留下体感疑问，第五卷后逐渐接近 |
| 真实底层 | 世界观设定 | 上古封印松动，上古黑暗侵蚀世界，反面被动显化 | 第七卷归一时确认 |

严禁：

- 第一卷角色直接说出「反面是封印松动导致的上古黑暗侵蚀世界」。
- 把反面写成任何角色主动制造、编织或投放。
- 把真实底层作为公开档案、课堂教材或普通调查结论提前出现。

## 九、归一之躯的同源反馈潜能

### 接近同源结构时的低频反馈

三月承载者的归一之躯接近同源规则结构时，具备产生低频反馈的潜能。可见形式可以是脚下短暂结霜、指尖发白、掌心骤冷等低外显，数秒内消失，不留永久物理痕迹。

这不是反面入口发出信号被她接收，而是归一之躯与同源承重结构之间存在底层咬合。第一卷除 Ch1 冰封的一次性被动外泄外，不用这一机制解释 Ch11-13；那三次均是哥伦比娅主动选择放出极低频霜月反馈后的残留。

### 接触侵蚀残留时的排斥潜能

当归一之躯直接接触正面人体上附着的反面规则侵蚀残留时，霜月碎片具备瞬时排斥潜能，可以短暂冻停或静止接触面上的侵蚀扩散。但第一卷 Ch13 的冻停是哥伦比娅主动出手压制，不是本能触发；被动排斥是否在后续卷显形，按对应卷硬口径决定。

边界：

1. 无论主动调用还是未来可能出现的本能排斥，都不是净式、核心技或常规术式。
2. 效果是暂停或冻停，不是清除。
3. 第一卷 Ch11-13 只能以主动低频反馈出现，并必须被压回 S 级感知、稀有冰霜核心或无吟唱术式的误判空间。
4. 第一卷作者层不得把 Ch11-13 写成“身体自动反应”；她知道自己在藏，也知道自己只放了一点，但不知道这股力量的底层来源。

## 十、月相锁定机制

女神态本身无切换代价，三套月相女神态各自独立，可用即满功率。前期限制来自锁定机制和身份暴露风险。

| 阶段 | 锁定程度 | 说明 |
|---|---|---|
| 第一卷 | 霜月碎片低频反馈 | 不展开女神态，只允许伪装态、冷白反馈和 S 级适配误判 |
| 第二卷 | 完全锁定 | 当前月相旬只能展开对应月相女神态 |
| 第三卷 | 相邻借用 | 可短暂借用相邻月相女神态，时间极短且不可连续 |
| 第四卷 | 单线自由 | 可在任何月相旬展开任意单条月相女神态，同一时间只能维持一条线 |
| 第五卷 | 自由切换 | 不再受月相旬限制，可在战斗中切换 |
| 第六卷 | 完全解除 | 可同时维持多条线 |
| 第七卷 | 三月归一 | 不再有切换概念 |

## 十一、三月女神态外观

三套女神态不是换色，而是法则层面的显化。三套外观共享体型和面部特征，但视觉风格差异极大，目击者很难立刻联想到是同一人。

### 霜月女神态

- 基调：白、冰蓝、冷银。
- 服装：巫女式剪裁，宽袖束腰，裙摆层叠如冰晶裂片。
- 发色：银白渐变至冰蓝末梢。
- 气质：清冷肃寂，像月下霜原上行走的巫女。
- 法则显化：霜纹沿皮肤蔓延，呼吸间可见白雾。

### 恒月女神态

- 基调：金白，暖光流转。
- 服装：祭司长衣式剪裁，庄重而有流动感，金色纹路自肩部向全身蔓延。
- 发色：淡金至纯白。
- 气质：庄严温暖，破晓时分的创世之光。
- 法则显化：金色光纹在皮肤下隐现，周围空间微微扭曲。

### 虹月女神态

- 基调：猩红、暗紫、虹光。
- 服装：华艳而危险，不对称剪裁，裸露处有猩红纹样如血管蔓延。
- 发色：深红渐变至暗紫。
- 气质：艳丽危险，落日前最后一道虹光。
- 法则显化：猩红裂纹沿皮肤蔓延，空气中弥漫虹色光尘。

## 十二、见神认识层级

「见神」不是单纯物理可见，而是从目击外观到理解位格的多层认识。

| 层级 | 表现 | 叙事含义 |
|---|---|---|
| 目击外观 | 看见高位形态、强光、法则显化 | 低阶角色会解释为未知高位神秘或异常强者 |
| 承受神性 | 能在不崩溃的前提下承受神性压迫 | 需要高位战力、特殊抗性或保护 |
| 识别异常 | 知道这不是普通神秘体系能解释的东西 | 方导师、苏弥等可逐步接近 |
| 理解位格 | 理解三月权柄、神性法则和旧神牺牲 | 秘史意义上的真正「见神」 |
| 与神同行 | 能与神性位格对话、并保留自我判断 | 终局核心圈层层级 |

第一卷普通学生可以看见哥伦比娅的异常表现，但只能误判为 S 级适配、稀有冰霜核心、无吟唱术式或罕见核心技。不能让他们直接理解位格。

## 十三、高位强者制衡

本世界不是高位强者随意解决一切的结构。高位存在有能力，但有职责、权限、情报、代价和社会结构约束。

1. 学院负责教育、保护和受控训练，不替学生完成所有选择。
2. 公会负责承接任务、远征、委托和社会层面的风险，不直接接管学院日常。
3. 八律承担古脉责任和历史传承，不是万能行政机构。
4. 高位战力不能在低风险事件中随意明牌，否则会破坏社会秩序、暴露更深真相或引发政治后果。
5. 第一卷危机里，学生必须先完成学生能完成的部分：救人、保出口、保证据和低调收口。封印边界镇压由方导师接手。

## 十四、上古黑暗与终局敌人分层

上古黑暗不是普通怪异，也不是某个可被早期角色调查确认的组织。它是被三月女神以死亡封印在世界背侧的高位失衡。

| 层级 | 表现 | 卷次作用 |
|---|---|---|
| 代理层 | 幕后推手、陆沉、旧时代笔记、白色金属书签、数据归档格式 | 第一卷至第五卷通过行为痕迹推进 |
| 现象层 | 反面活跃、局部浮现、准降临、怪异升级 | 第二卷至第五卷逐步加压 |
| 规则层 | 封印松动、承重锚点脱开、权柄回收导致局部升级 | 第四卷起被推测，第五卷实战验证 |
| 真相层 | 上古封印衰退，上古黑暗侵蚀世界 | 第六卷接近，第七卷确认 |
| 终局层 | 三月归一后重新锚定封印，完成终局裁定 | 第七卷解决 |

终局不是打死一个普通敌人，而是完成三月归一后的世界级重新锚定。主角可以成为唯一执掌过去、现在、未来的创世级神明，但不得写成无代价、无限制、跨世界随意重写一切。

## 十五、写作总边界

1. 日常空间默认安全，不把宿舍、教室、食堂、正规训练场写成处处漏风。
2. 反面不是所有异常总源头，非反面冲突必须保留。
3. 月亮体系不是普通神秘体系的上级词，普通神秘者不靠月相供能。
4. 三月真实底层不能提前进入第一卷公开正文。
5. 主角异常优先写成主动降档、输出收束、能力外观遮掩和旁人误判；真实底层始终是零核心激活、零以太调用、零术式构筑，不写成失控反噬。
6. 任何涉及第一卷现行边界的判断，以 `设定/当前硬口径.md` 和 `设定/大纲内容/第一卷/第一卷当前写作状态.md` 为优先裁决源。
