## 一、项目定位

这个变身流的奇幻小说。

主轴固定为：
**隐藏身份 + 无敌流 + 异世冒险 + 古脉秘史 + 神秘学 + 热血战斗**

整体基调固定为：
**日常、神秘、热血、奇幻一起走，不拆开写。**

***

## 二、主角设定

主角灵魂来自与三月世界无关的外界，在前世死亡后因月相、寂灭、重生和过去未来闭环的相性，被三月世界残余权柄吸引，降临到三月世界自行孕育的「归一之躯」中。归一之躯读取其灵魂本相，显现为哥伦比娅·希珀塞莱尼娅的外貌。

她是当前异世界唯一的三月女神承载者，也是三月女神死后被三月世界承认的外来继承者；她能执掌三月相关权柄，不是因为旧神复活，而是因为归一之躯提供了承载接口，外来灵魂提供了三相闭环相性。
主角能力必须始终围着三月女神设定走，**不额外乱加无关能力**。

用词口径：
- **权柄**用于宏观权限、位格和主线进度层面，如三月权柄、权柄回收、权柄解锁期。
- **权能**用于具体能力落地层面，如过去复写权能、改变未来权能、静止世界权能、生死操控权能。
- 两者不得混用：主角接过的是三月权柄，战斗中具体调用的是某项权能。

这个项目明确是变身文。

这次降临不是简单换壳，而是：
**原本男性灵魂视角，被三月世界吸引进一具无灵魂、无旧人格、可承载三月归一的女性神性空白躯体里。**

### 归一之躯口径

主角所占据的身体不是原世界原身，不是旧三月女神遗体，也不是三月女神预留的复活容器，而是三月世界在漫长封印中自行孕育出的空白神性承载体。

1. **无原人格**：归一之躯没有原主灵魂、原主记忆或旧神人格，不存在夺舍、覆盖或旧人格苏醒问题。
2. **外貌来源**：外来灵魂降临后，归一之躯读取其灵魂本相，才显现为哥伦比娅·希珀塞莱尼娅的外貌；这不代表三月世界与原世界存在因果联系。
3. **能力来源**：战斗直觉、无吟唱、超规格判断和三月权柄反馈，来自归一之躯与三月权柄接驳，不来自前世战斗经验。
4. **开篇异常**：灵魂降临、归一之躯长期沉睡苏醒、霜月碎片被动外泄三重叠加，造成第一章一次性冰封；后续不可复现同等规模。
5. **主题边界**：主角不是三月女神复活，也不是三月神性核心长出的新人格，而是外界来客、三月继承者、非旧神复生者。

### 穿越前世界口径

穿越前主观世界统一定为：近现代低神秘/无显性超凡世界，生活质感接近现代地球。该世界与三月世界无世界观关联。

穿越前主观身份统一定为：十七岁心脏病少年，长期病弱，身体虚弱到快走几步都会心跳骤乱，常年在病房、针管、心电监护仪和输液架之间生活；不是高位者、神、军人、杀手、长期战斗者或神秘体系修炼者。

更外层前史口径：主角灵魂曾在另一个无关联世界以哥伦比娅·希珀塞莱尼娅的形态经历"过往寂灭，向未来重生"的命运结构。这段前史只解释灵魂为何与三月权柄相性极高，不给当前三月世界提供攻略、情报或力量来源；前中期只可作为极模糊梦境、陌生熟悉感或名字外貌既视感存在。

穿越前知识边界：

1. 有现代生活常识，理解校园、食堂、宿舍、网络、论坛、视频、游戏等概念；但现实校园生活经验受长期住院限制，更多来自短暂就学、网络和旁观。
2. 可以对"哥伦比娅"这个名字、外貌或角色印象有模糊熟悉感，但不能把它当成当前异世界攻略。
3. 不懂曜月学院制度、神秘体系、百城八律、反面、术式、月相权柄和当前世界秘史。
4. 战斗直觉、无吟唱、超规格判断和三月权柄反馈，全部来自归一之躯与三月权柄接驳，不来自前世经历。

### 穿越前人格锚点

穿越前的主角只提供人格底色和现代人惯性，不提供战斗履历、超凡知识或前世大谜团。

1. 性格底色偏冷静、观察优先，遇到陌生环境会先收集信息再行动；不是社牛，但也不是阴郁厌世型。
2. 有普通现代男性的身体边界感和性别惯性，也有长期病弱身体带来的谨慎和自我保护习惯；因此穿越后面对健康、轻盈的女性身体，长发、校服、盥洗、宿舍距离感和被注视感都会持续不适应。
3. 道德底线正常，面对学生被诱导、受伤或被利用时，会先按普通人的保护冲动反应，而不是高位神明的漠然旁观。
4. 习惯用现代常识理解校园制度、课程安排、宿舍生活和同学关系；但因为长期病弱，真正参与普通校园生活的经验不足，当现代常识、病房生活惯性与神秘学院规则冲突时，会出现短暂停顿和重新校准。
5. 对"哥伦比娅"的熟悉只停留在名字、外貌和模糊角色印象层面，不会使用玩家攻略口吻，不把当前异世界当游戏副本。
6. 穿越前没有战斗训练，也没有健康身体带来的运动经验，所以主观意识并不擅长战斗；战斗中的精准、冷静和超规格判断来自归一之躯本能与三月权柄反馈。
7. 不写成嘴臭吐槽役、油滑乐子人或重度互联网梗输出者；她可以有现代人反应，但整体气质仍要和哥伦比娅当前的清冷、安静、判断准保持一致。
8. 前世记忆在正文中的功能是制造适应差、身份认同差和少量现代常识错位，不承担解谜外挂、情报外挂或战斗外挂功能。

正文禁写：前世只有权柄、规则、战斗和黑暗；前世长期战斗；前世掌握神秘体系；前世是神、救世者、高位者、军人、杀手、运动健将或任何特殊战斗职业。

所以前中期必须长期写她在适应下面这些变化：

1. 身体感知变化
2. 性别身份变化
3. 社交位置变化
4. 气质和外在观感变化
5. 归一之躯天然自带的神性疏离感

主角这条线不是普通升级流，而是：
**外来继承者以普通学生身份隐藏自身，在校园、公会和异世冒险里，一步步接过三月女神死后留下的权柄、责任和选择权。**

## 三、世界观设定

穿越目标世界的大框架已经定死。

当前确认的关键基础如下：

1. **纪元**：第八纪元（恒月纪元）
2. **天象**：双日三月轮空
3. **古脉**：百城八律（城邦律、魔女律、圣林律、血契律、亡名律、妖血律、灵脉律、海脉律），全部属于人族内部的守律传承，不写成八个生物种族
4. **整体气质**：表面秩序还稳着，深层神秘危机和热血战斗一直并存

## 四、神秘等级体系

这条线先卡死几个原则：

1. **神秘和月相没有直接从属关系**
2. **世上大多数超凡者走的都是神秘体系，不是月亮体系**
3. **每个人的神秘都不一样，能力方向、代价和战斗风格都很个人化**

神秘等级固定为五级：神秘雏形→神秘者→深律者→域主→秩序者。

完整等级描述和晋升条件见《神秘设定》"神秘等级体系"和"神秘成长体系"段落。

***

## 五、怪异等级体系

怪异等级固定为：

1. **普通级**：单一能力、单一核心、单一弱点
2. **危险级**：有完整战斗意识、稳定杀招和多段连招
3. **灾难级**：会开领域、会群体精神侵蚀、会带大范围失控
4. **禁忌级**：已经上升到规则改写、时空扭曲、存在抹除级威胁

正文对外只使用这四个官方等级；作者内部和任务分层可以使用**下位 / 中位 / 上位 / 阈值**细分。阈值不是第五级，而是"已经显露高一级特征、但核心还没完全稳定"的过渡状态。

| 官方等级 | 内部分段 | 使用口径 |
| ----- | ----- | ----- |
| 普通级 | 普通下位 / 普通中位 / 普通上位 | 单一异常到可连续伤人的低阶怪异 |
| 危险级 | 危险下位 / 危险中位 / 危险上位 | 有杀招、会伏击、可巢化或区域化的中阶怪异 |
| 灾难级 | 灾难阈值 / 灾难下位 / 灾难中位 / 灾难上位 | 从刚晋升、核心不稳，到城域/百城危机和灭国前兆 |
| 禁忌级 | 禁忌阈值 / 禁忌下位 / 禁忌中位 / 禁忌上位 | 从规则共振前兆，到秩序者级对手、文明级和纪元级威胁 |

怪异等级不能和神秘等级、战力层级直接画等号。怪异等级看的是**单体战力、扩散速度、规则侵蚀、处理窗口、情报缺失后的社会后果**；神秘等级看的是神秘者对规则的掌握深度。

### 怪异段位与神秘等级处置门槛

完整的 14 段位处置门槛表（最低接触门槛 / 稳定处理门槛）以《神秘设定》"怪异细分段位与神秘等级对齐"段落为唯一来源，本文件不再复制。

压缩口径：普通级由神秘雏形到神秘者处理；危险级由神秘者到深律者处理；灾难级需要域主/区域霸主级以上压阵；禁忌级以秩序者级为起点，强禁忌必须多方联合。

补充规则也一起定死：
怪异整体遵循**越强越拟人化**的倾向。低等级更像异常现象、污染或本能活动体；高等级更容易长出稳定形体、独立智慧、明确恶意和复杂行动逻辑。

怪异成长体系（吞噬同化、侵蚀蜕变、规则共振三条路径及成长阶段门槛）见《神秘设定》"怪异成长体系"段落。

## 六、战力层级体系

战力层级固定为：

1. 凡人境
2. 神秘雏形境
3. 神秘者境
4. 区域霸主级
5. 世界影响级
6. 月亮级（主角专属上限）

### 战力层级与神秘等级对应关系

> **注意**：下表为参考对应，非严格绑定。同等级内因相性、经验、规则理解差异巨大，战力层级可上下浮动。

| 战力层级  | 参考对应神秘等级        | 区间说明 |
| ----- | ------------- | --- |
| 凡人境   | 无神秘等级         | 无超凡能力 |
| 神秘雏形境 | 神秘雏形          | 刚觉醒，能力还不稳 |
| 神秘者境  | 神秘者           | 核心能力成型，已是标准战力 |
| 区域霸主级 | 深律者到域主之间   | 下段≈深律者中期，中段≈深律者后期到域主初期，上段~巅峰≈域主中后期 |
| 世界影响级 | 域主到秩序者     | 下段≈域主后期，中上位≈秩序者 |
| 月亮级   | 主角专属，超出常规神秘体系 | 仅三月权柄持有者可达 |

### 写法规范

人物能力发展文件中写成长路径时，**两套术语同时标注**，格式为"神秘等级（战力层级）"。

示例：深律者后期（区域霸主级中段）、域主初期（区域霸主级上段）、秩序者（世界影响级中上位）。

避免只写一套术语导致读者产生"跳级"感。

### 补充规则

1. 世间常规战力还是按**神秘等级 + 个人神秘核心**来判断
2. 月亮级不是大众修行终点，而是主角独有的神性位格上限
3. 只有世界影响级以上的少数存在，才真正见过神，或者有资格理解神
4. 绝大多数生命识别不了主角真实位格，只会把她当成异常强的天才、怪物或特殊神秘者
5. 区域霸主级已经是足以灭国的层次
6. 世界影响级则是真正能把文明级、纪元级后果打出来的层次

## 七、三月设定

1. **恒月（黄色）**
   - 核心法则：过去复写、存在与创造
   - 掌管：过去
2. **猩红虹月（深红色）**
   - 核心法则：改变未来、败线剪定与心智虚实干涉
   - 掌管：未来
3. **清冷霜月（白色）**
   - 核心法则：静止世界、定格现在、生死边界与能力潮汐
   - 掌管：现在

说明：恒月"掌管过去"的上位权能是读取已经发生的痕迹，选定某个已经成立过的状态，把它重新接回当前战场。高阶状态下，恒月可以让"曾经完整""曾经活着""曾经未崩塌""曾经被钉住"这类过去事实重新覆盖现在，也可以改写某个关键过去节点，让当前局面按新的因果往下走。

说明：虹月"掌管未来"的上位权能是改变未来。它不是普通预知，而是先看见未来分支，再删掉中间过程，让局面直接抵达对自己有利的既定结果。被她否定的未来会失去确定到达的资格，敌人本该成功的反击、逃脱、连招和翻盘线，都只能停在"可能发生"之前。

说明：霜月"掌管现在"的上位权能是让当前时刻停住，极限状态下可以静止世界。它静止的对象包括时间流动、物理运动、能力运转、生命状态、伤势变化、空间崩塌和规则推进；绝对零度与能量停滞只是战斗层外显，不是霜月权能本体，也不是单纯冰属性。

### 三月规则补充

1. 每旬轮换一轮，周而复始
2. 三月同时高悬会引发规则剧变
3. 主角最终能完整掌控三轮月亮的力量
4. 三月权柄只服务主角个人主线和世界高层真相，不拿来当世间常规神秘体系的总源头
5. 三月本质上是力量、法则和权柄的集合，**不能写成会说话、能直接交流的拟人化存在**

### 三月命名规则

三月正式名分别为恒月、猩红虹月、清冷霜月。正文和大纲中，恒月直接用简称（因为全称就是恒月），猩红虹月和清冷霜月首次出现时用全称，后续可用简称虹月、霜月。设定文件内部统一用全称。

3. 接受自己不是暂住这具身体，而是必须真的以这具身体和这个身份活下去

## 十、爽文核心写法

这本书明确走的是**隐藏身份的无敌流爽文**。

核心逻辑就一句话：
**主角其实不用任何人配合也能赢，但她平时不全打，是为了维护普通学生身份边界，也是为了让队友成长。她真出手，只因为队友顶不住了。**

### 具体原则

1. 主角本质上开局就是天花板级存在
2. 前期不是弱，而是还没和异世界规则完全接驳
3. 她平时以普通学生身份活动，低调是为了避开不必要的麻烦和目光
4. 队友能打就让队友打，真打不了了她再出手，每次出手都得拉出明显反差爽点
5. 每解锁一轮月亮权柄，都要带出一次明显的剧情推进和爽点爆发
6. 主角平时大多数时候都处于伪装状态，只有在队友扛不住、常规手段收不了场，或者需要高光镇压时，才短暂开更接近女神位格的状态
7. 就算开了高位状态，对外也要尽量包装成特殊神秘、未知遗物、异常共鸣或极端战斗表现，不能轻易把真实身份亮穿
8. 战斗不能长期依赖“大家一起上、镜头平均分”的团体战来撑场面，更适合多写单人战、双人战和局部协同战
9. 真正的决定性高光和收束，尽量交给一个明确角色完成；主角负责卷内最重的终局收口，配角也要各自有单独扛起一拍的时刻

### 爽点不只来自战斗碾压

变身反差也是核心爽点之一：

1. 她心里还留着过去的习惯，外面却已经是让人移不开眼的少女和女神容器
2. 她越想把自己当普通人，越容易在言行、气质和外貌上拉出反差
3. 她在适应女性身份的过程中，会先别扭，再慢慢长出掌控感和魅力

### 成长-代价悖论的叙事取向（已确认：爽文优先）

主角权柄回收会导致反面短暂松锁升级，这个机制的**叙事处理方向是爽文优先**，不是道德代价流。

**具体执行口径**：

1. 反面松锁升级是**逼主角出手的外部压力**，不是主角成长造成的道德罪责
2. 写法重心放在"局面被逼到了必须她出手"，而不是"她变强直接造成了这些伤亡"
3. 世界本就在走向失衡，主角的权柄回收是唯一真正能锁死反面的路——幕后推手的判断在这个层面是对的，但他是反派，代价由他承担，不由主角背负
4. 权柄解锁的每一次爆发，首要目感是**反差爽点和高光镇压**，代价感服务于"她平时为什么要藏"，不反过来压制爽感
5. 不把主角写成"每次变强都要内疚"——她的内驱力是完成自己的使命，不是赎罪

## 十一、关系体系设定

主角团的关系固定走**稳定累积型羁绊**。

### 明确禁止

1. 不搞主角团分裂
2. 不搞核心队友背刺
3. 不搞误会式长期决裂
4. 不拿内部撕裂当主要冲突推进手段

### 允许存在的张力

1. 磨合
2. 认知差
3. 身份带来的震撼
4. 外部压力下的短期意见差异

但最终方向必须一直守住：
**关系越走越热，信任越攒越厚，队伍越打越稳。**

## 十二、主角团标准结构

1. **室友 / 青梅型同伴**：负责陪伴感、日常感、最早无条件信任主角
2. **战斗型伙伴**：负责热血战斗和并肩作战
3. **术式归档型伙伴**：负责资料归档、任务记录、核心频率判断、术式库存火力、档案和秘史追查
4. **古脉代表型伙伴**：负责把八律和百城主线接进来
5. **高位引路人**：少数世界影响级以上强者，能隐约看懂主角位格，但主要站在保护、观察、见证的位置

阶段口径：第一卷四人组暂时由哥伦比娅、林见星、白砚河、苏弥构成；古脉代表型伙伴在第一卷尚未正式纳入四人组，第二卷起逐步加入。

### 四人组战斗分工

四人组是群像战斗结构，每个人都要有能直接落在战场上的价值，不能只做旁白、解说或后勤。

1. **哥伦比娅：异常压制 / 王牌位**
   - 平时隐藏实力，关键时刻压住失控、旧月回应、怪异异常和队友撑不住的局面
   - 她负责把不可能的局面压回可处理范围，但不能每场一开始就全包
2. **白砚河：近战破阵 / 主输出位**
   - 正面冲锋、撕开防线、承担热血硬碰硬爽点
   - 他负责把敌人和战场打出缺口
3. **林见星：断线瞳导航 / 机动护场位**
   - 负责看见命中线、断裂线、危险路径和撤离线断点，用星丝术护人、牵路、稳住队形
   - 她解决“队伍怎么活着抵达正确位置”
4. **苏弥：资料归档 / 核心频率判断 / 术式库存火力位**
   - 前期核心职责不是解析反面底层规则，而是处理任务记录、权限缺口、核心频率、术式库存和战后证据
   - 平时把低阶封堵术、净化术、束缚术、攻击术抄录归档进封印书；战斗时翻页激活，形成弱化版的术式覆盖
   - 她可以打，但强在“准备好的多术式连续释放”“标记撤离路线”“短时稳定核心频率”，不是站在后面讲解敌人弱点
   - 第一卷不能让她直接读懂高位规则、反面底层结构或主角位格；只能通过核心频率、术式印、记录异常、权限缺口逐步发现“有人动过手脚”

### 四人组互动温度分工

四人组不能全员冷处理。团队日常互动需要形成稳定温度结构：

1. **哥伦比娅：冷面反差**
   - 本人不主动搞笑，但会因为过度认真、频道错位、身份边界维护和女性身份不熟练产生反差笑点
   - 她的温度不靠话多，而靠顺手护人、记住细节、接受善意时的短暂停顿
2. **林见星：生活暖场**
   - 负责把队伍拉回日常，递水、夹菜、占座、照顾身体状态
   - 可以轻声吐槽，但不尖锐；她的存在让团队有“有人在等你回来”的生活感
3. **白砚河：直球冲撞**
   - 负责笨拙喜剧和行动派张力，嘴硬、抢菜、听不懂专业话但会先冲出去
   - 他的笑点来自直、冲、不绕，不来自装傻；战斗越硬，日常越容易粗糙出错
4. **苏弥：冷静吐槽**
   - 负责记录式幽默和信息反差，一本正经说扎心话，把别人情绪当观察样本
   - 她的温度藏在“我记下来了”和“不问，但我知道”里，不写成纯工具人

**写作要求**：每章至少让其中一人承担一次轻微暖场、冷幽默或反差缓冲，除非该章是卷末高压战。暖场不能破坏主线压迫感，只负责让队伍像真正一起生活、一起上学、一起打架的人。

## 十三、剧情阶段划分

1. **银序一班篇（第一卷）**
   - 重点：适应归一之躯、隐藏身份低频外泄、307宿舍与四人桌成型、课堂/食堂/积分任务建立校园感、旧训练楼传闻升级为第一次局部危机；不展开女神态
2. **深潜篇（第二卷）**
   - 重点：深入学院深层权限、接触反面核心秘辛、神秘等级推进、怪异升级、异常累积
3. **日常篇（第三卷）**
   - 重点：变身认同"从别扭到自然"完成、团队从战友变生活伙伴、日常中异常越来越难遮
4. **裂痕篇（第四卷）**
   - 重点：身份半暴露、团队信任危机、学院准降临危机/局部浮现失控、学院危机倒逼提前毕业
5. **公会·立足篇（第五卷）**
   - 重点：学院毕业后正式进入公会、在公会体系里重新找到位置、幕后者部分暴露
6. **远征·真相篇（第六卷）**
   - 重点：公会远征、八律领地、身份进一步暴露、三线法则联动、幕后者完整动机
7. **归一·终局篇（第七卷）**
   - 重点：三月归一、终局战、身份彻底公开、世界重构

## 十四、整体创作原则

1. 始终把变身后的适应感和身份反差放在前台
2. 始终让异世冒险、古脉秘史、神秘学三条线保持平衡
3. 始终让主角显得强势、神秘、高位，但不轻易明牌
4. 始终把大冲突放在外部世界，而不是主角团内部崩裂
5. 后续所有大纲、分卷、人物设计，都以本文件为当前版本依据
6. 常规角色的战斗、成长、职业和冲突，优先围着个人神秘核心设计，不统一往月亮体系上靠
7. 怪异事件通常都要落到明确怪异敌人、实体载体或可对抗目标上，别只剩气氛异常却没有实战压力
8. 主要角色的具体战力落点和能力成长，补充参考《主要角色战力定位表》《主要角色能力发展表》

## 十五、变身核心写法

1. 变身感必须一直在，不能开头提一句后面就消失
2. 前期要持续写身体和身份的不适应，比如动作习惯、穿着体验、被注视感、称呼落差
3. 中期要写她开始熟练利用女性身份和外在气质去完成隐藏、社交和反差压制
4. 后期要写她真正认下“她自己”，把变身从被动接受推成主动掌控
5. 这条线的基调不是痛苦撕裂，而是带着新鲜感、反差感、掌控感和高位魅力的渐进式适应
