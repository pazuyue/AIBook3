## 一、总原则
1. **平时大家用什么打**：这个世界正常战斗还是靠“神秘体系”，不是人人都能碰到三月那一套
2. **三月是主角专属线**：三月权柄只服务主角和高位冲突，不拿来替代全世界的普通战斗体系
3. **敌人要能看得见、打得着**：重要敌人不能一直只靠概念吓人，得有实体、动作、受击反应
4. **打戏要写得看得见**：尽量写拳、刀、枪、火、冰、雷、护盾、冲击、封锁这类能直接脑补出画面的东西
5. **强弱要看得出来**：一个角色强不强，要能从破坏力、速度、抗打、范围、控场、连战力这六项里直接看出来
6. **不是纯数值比大小**：同层次交手，看的不只是面板，还要看相性、脑子、配合和临场反应
7. **越强的怪异越会动脑子**：高等级怪异不只是更能打，也更会设套、骗招、埋坑
8. **团体战不能写成默认配置**：多写单人战、双人战、接力战和局部协同，少写从头到尾平均分镜头的大团战
9. **一场战斗最好只有一个绝对高光位**：其他人可以托、接、补、救，但不要人人都抢同一场的光
10. **真正的收束要落到具体人身上**：局面可以靠团队撑住，但最后一拍最好由一个明确角色拿下，主角负责卷内最重的终局落幕

## 二、战斗六维

| 维度 | 描述 | 分层/示例 |
|------|------|-----------|
| **破坏力** | 说白了就是一招能打多狠 | 能打翻人 → 打烂门墙 → 拆楼 → 打崩一片街区 |
| **速度** | 看谁先出手、追不追得上、躲不躲得开 | 普通人 → 比人快一截 → 冲刺带残影 → 快到肉眼难追 |
| **抗打性** | 看能挨多少、挨了会不会倒 | 扛钝击、扛斩击、扛爆炸、扛精神侵蚀 |
| **范围** | 看一招能打多大地方 | 打一个人 → 打一小片 → 清一间房 → 压一整片区域 |
| **控制力** | 看能不能让对面动不了、打不顺 | 打断、减速、束缚、封印、压制 |
| **连战力** | 看能打多久，全力出手后还能不能接着打 | 连战能力、持久战能力、全力出手后的状态 |

### 高位战力四维标尺

区域霸主级与世界影响级不能只用“能毁多大”判断。高位战斗统一从四个维度记录，避免把单次大招、持续灾害、政治威慑和规则权限混成一个概念。

| 层级 | 单次破坏 | 持续灾害 | 战略影响 | 规则权限 |
|---|---|---|---|---|
| **区域霸主级下段** | 能摧毁要塞、重创城域关键节点 | 可在局部战场持续压制常规部队 | 能决定一座城域级事件的成败 | 具备界临雏形或局部规则压制，但仍可被同级战术拆解 |
| **区域霸主级中段** | 能毁城、裂山或改写大型地形 | 可制造跨城区持续灾害 | 能迫使城邦调整防线与资源配置 | 界临成熟，可稳定改变主战场条件 |
| **区域霸主级上段至巅峰** | 能威胁国境要塞群或造成灭国级前兆 | 可让灾害跨城扩散，常规力量难以止损 | 一人足以改变国家级决策与公会格局 | 能压制多重界临，逼近高阶秩序门槛 |
| **世界影响级下段** | 单次出手未必追求大范围破坏，但可击穿国家级防线 | 可形成大国级持续灾害或长期封锁 | 本人存在即改变多国与八律决策 | 能直接制定部分局部战场规则 |
| **世界影响级中上位** | 可引发洲陆级灾变或纪元级后果 | 灾害可跨区域长期维持并自我延续 | 足以改变世界格局与文明存续判断 | 能稳定建立高阶秩序，常规界临无法正面对抗 |

使用原则：

1. 单次破坏范围大，不代表规则权限一定高；规则权限高，也不代表每次出手都要毁城。
2. “灭国级”优先描述战略后果与持续灾害，不默认等于一击抹去国家。
3. 高位角色的正文压迫感优先通过封锁、秩序、处置权限与他人反应表现，避免频繁堆叠地形毁灭。

## 四、神秘等级与实战门槛
这一段可以简单理解成：

- **神秘等级**：更像修炼进度，代表你走到哪一步了
- **战力层级**：更像实战能打到什么程度

两边有关联，但不是完全一回事。

| 神秘等级 | 实战能力 | 对应战力层级 |
|----------|----------|--------------|
| **神秘雏形** | 刚摸到超凡门槛，但还不够稳 | 神秘雏形境 |
| **神秘者** | 能稳定战斗了，已经算成熟战力 | 神秘者境 |
| **深律者** | 开始碰规则层的东西，已经不是普通强者 | 深律者·稳解≈区域霸主级下段；深律者·深解≈区域霸主级中段 |
| **域主** | 能完成界临，让个人法则接管一整片主战场 | 新晋域主≈区域霸主级中段到上段；成熟至顶尖域主≈区域霸主级上段到巅峰，接近世界影响级门口 |
| **秩序者** | 常规神秘体系的顶点，能直接定一部分战场规则 | 世界影响级（域主到秩序者） |

其中“新晋、成熟、顶尖”只描述同为域主时的实战积累与界临完成度，不是新的正式内部阶段；域主在神秘等级上仍不细分。

## 补充：个人核心与通用术式战斗口径

### 1. 战斗四层

战斗描写必须先分清四层：

1. **神秘核心**：个人唯一能力本体，决定核名、核形、核能、成长上限与代价。
2. **核心技**：本人对核心能力的具体应用，不能教学或完整复制，不走术式回路。
3. **通用术式**：学院、公会可教学、可记录、可复现的标准技术，使用七十二标准式谱。
4. **高位状态**：真名解放、界临、立序，以及主角体系外的女神态、视界和创世态。

简单口径：

- 核心回答“只有这个人能做到什么”；
- 核心技回答“这个人怎样把独有能力送进战场”；
- 术式回答“受训者共同掌握哪些公共技术”；
- 解放、界临与立序回答“个人法则推进到了多深”。

### 2. 核心技、真名解放、界临与立序

| 层级 | 战斗表现 | 写作判断 |
|---|---|---|
| **核心回响** | 从危险或情绪中的被动反馈，推进到固定条件下短暂、局部显出核心技雏形 | 能力能够重复出现，但还不能跨环境稳定成立 |
| **核心显相** | 核名、核形与核能稳定，形成个人核心技 | 本人能主动让能力发生 |
| **真名解放** | 核心动词、形态和代价同时质变 | 本人理解能力真正是什么 |
| **界临** | 核心法则覆盖战场，出现范围、锚点、默认效果和反制方法 | 战场开始自动执行其能力逻辑 |
| **立序** | 唯一主序在条件成立时拥有局部优先权 | 世界暂时承认其核心规则 |

核心技可以喊名，但喊名不是吟唱，不参与公共回路，也不存在“节省吟唱导致核心技缩水”。解放语同样不是吟唱，而是人物对核心真意的自我确认。

正文标记格式：

```text
【裂锋·破甲。】                    # 核心技
【让万物显出断处——真名解放，裂锋·万物有隙。】  # 解放语
【界临·无间断界。】                # 界临
```

第一次真名解放必须同时写出动词质变、形态变化和代价显露。界临必须写清范围、锚点、默认效果与反制方式。立序必须来自核心本质，并带明确条件和边界。

### 3. 七十二标准式谱

术式只指公共构筑，分类与序号按《神秘设定》执行：

- 攻式、禁式、护式、行式、察式、净式、阵式；
- 标准格式为 `【分类·序号，术式名。】`；
- 序号代表注册结构、复杂度与权限区间，不是绝对威力；
- 低序号用得准，可以打断高序号；
- 日常叙述可省略编号，首次登场、教学、档案和正式喊名使用完整格式。

示例：

```text
【禁式·十二，锁步。】
【攻式·二十七，贯星。】
【护式·九，偏甲。】
```

### 4. 术式吟唱

吟唱只属于通用术式。它不是站着背咒，而是借声音、动作、呼吸和媒介把标准回路补完整。

| 方式 | 本质 | 优点 | 代价 | 常见场景 |
|---|---|---|---|---|
| **全吟唱** | 完整构筑标准回路 | 威力、稳定性、范围和附加效果最完整 | 慢，容易被打断 | 决胜窗口、守点、清场、阵地战 |
| **节省吟唱** | 省略部分节点，只保留主要功能 | 快，适合常规实战 | 威力、范围、持续时间缩水 | 抢节奏、移动战、连招 |
| **无吟唱** | 不经声音直接完成全部回路 | 最快 | 门槛极高；强行使用会崩式、偏移或反噬 | 顶尖术式专家、预设媒介或特殊装备 |
| **预先构筑** | 战前把回路存入书页、符签、阵盘或设备 | 临场近似瞬发 | 库存有限，必须提前准备 | 苏弥封印书、阵地防御、职业小队 |

普通学生默认使用“简易吟唱 + 标准术式名”。只报术式名而效果完整，必须有高熟练压缩、预先构筑、媒介辅助或明确代价。

节省吟唱示例：

```text
【三点定足，越线止步。禁式·十二，锁步。】
```

全吟唱示例：

```text
【以三点定足，以四隅封路；越界者止，受缚者留。禁式·十二，锁步。】
```

吟唱观感可以参考高节奏咒文的分类感、限制感、递进感、落名感与战术感，但文本、分类和序号体系必须完全使用本书原创内容。

### 5. 写作硬规则

1. 林见星的断线瞳与星丝、白砚河的裂锋、苏弥的归档与封印书、月枝的寒潮、方导师的压线、季悬灯的悬灯全部属于核心能力，不叫术式。
2. 核心技不写术式构筑、回路闭合、全吟唱、节省吟唱或无吟唱。
3. 只有公共的攻击、束缚、防护、机动、感知、净化和阵列技术才写术式与吟唱。
4. 同一角色可以组合两条轴：先用行式近身，再用核心技破甲；先用护式争取时间，再完成真名解放。
5. 核心显相或解放不等于立刻升级；必须经过稳定化和实战消化，才能成为常规状态。
6. 影子训练场强行催出的是污染显相、错误真名或伪解放，不是术式升级。
7. “展开”不再作为普通术式完成度；个人高位结构优先写界临，主角体系外仍可写女神态展开、视界展开和创世态。

### 6. 主角的体系外例外

哥伦比娅真实力量既不经过普通神秘核心转化，也不经过通用术式回路。对她使用“无吟唱”只能是旁观者误判，不能作为作者层真实机制。

- 霜楔、霜华、霜印等伪装能力，对外被判断为罕见冰霜核心的核心技；
- 她喊名只提供战斗锚点和身份伪装，不参与发动；
- 学院也可能把部分异常误判成静默术式、异常适配或遗物辅助；
- 真正异常证据是零核心激活、零以太调用、零术式构筑，却仍然产生结果；
- 女神态、三月视界与创世态不属于真名解放、界临或立序，不能硬套常规成长链。

第一卷战斗喊名仍只使用《主角能力发展设定》已经放行的短名【霜楔。】【霜华。】【霜印。】；第二阶段以后新增能力按对应角色文件执行。

主角伪装态的核心不是压住体内力量，而是主动降档：她把输出、精度、能力外观和战场结果收进“冰霜核心天才或静默术式特例”能够解释的范围。不能写成反噬、异物感、寄生感或失控风险。

### 7. 连续性归档

- 已出现的核心技、解放语、界临名与表现，记录在《核心技与解放归档》；
- 已出现的标准术式、编号、吟唱与发动层级，记录在《术式吟唱归档》；
- 核心技和术式都可以使用【】标记，但必须依靠命名格式和上下文区分。

## 五、怪异等级的实打实表现

| 等级 | 特征 | 能力表现 | 破坏规模 | 应对要求 |
|------|------|----------|----------|----------|
| **普通级** | 能力单一，弱点也比较明显 | 寄生、惊吓、低级精神干扰、简单扑杀、小范围幻象 | 房间级、走廊级 | 神秘雏形境基本能处理 |
| **危险级** | 已经会打，会连招，也有稳定杀招 | 高速扑杀、持续精神侵蚀、范围压制、藏影、精神攻击 | 建筑级到城区级 | 神秘者境才能稳着打 |
| **灾难级** | 会形成规则场，会控整片场，也很会动脑子 | 会诱导、设伏、分兵、拖垮一整片区域 | 城域级到灭国级前兆 | 深律者·深解只能碰边，真正处理要域主/区域霸主级以上 |
| **禁忌级** | 能改规则、扭曲时空，甚至抹掉存在 | 不只是强，而是已经像怪异中的王 | 灭国级到灭世天灾级 | 秩序者级起步，强禁忌需要多方联合和特殊条件 |

> **关键口径**：怪异等级不是人类战力层级的等号。灾难级怪异可以造成城域、百城级危机，甚至形成灭国前兆，但完整灾难级不等于秩序者。能稳定和秩序者正面博弈的，主要是禁忌级怪异。

### 怪异危险标签

怪异等级回答“总体有多危险”，危险标签回答“它具体怎么危险”。正文或任务简报在报出等级后，至少补充一个主要标签；复杂怪异可以有一个主要标签和一至两个次要标签。

| 标签 | 主要危险 | 处置重点 |
|---|---|---|
| **高速扑杀型** | 短时间内造成连续伤亡 | 预警、卡位、限制第一轮突进 |
| **区域绑定型** | 与建筑、地形、载体或人群绑定 | 先拆载体与区域条件，再处理本体 |
| **精神侵蚀型** | 放大情绪、扭曲认知、瓦解队伍判断 | 精神防护、轮换确认、缩短暴露时间 |
| **扩散寄生型** | 通过接触、媒介或群体传播扩大事件 | 隔离、封锁、追踪传播链 |
| **规则共振型** | 正在向更高等级跃迁，危险与脆弱并存 | 抢共振窗口，禁止拖成长战 |
| **隐匿诱导型** | 依靠伪装、误导、替身或路径陷阱杀人 | 情报校验、标记、禁止单独追击 |
| **高耐久统御型** | 本体难杀，并能驱使其他怪异或单位 | 斩首、断供、拆统御链 |

同等级怪异因标签、载体、场地与情报条件不同，可以需要完全不同的队伍配置。标签不新增怪异等级，也不改变对应处置门槛。

### 怪异等级内部分段

正文对外仍说普通级、危险级、灾难级、禁忌级；作者内部用分段控制压迫感和任务难度。

完整的 14 段位定位与处置门槛表以《神秘设定》"怪异细分段位与神秘等级对齐"段落为唯一来源，本文件不再复制。写战斗时按该表核对"谁能最低接触、谁能稳定处理"，再结合场地、载体、情报与危险标签调整队伍配置。

### 怪异成长对战斗表现的影响

怪异不是生来就固定在某个等级，它的成长状态会直接影响战斗表现。同一个怪异，成长阶段不同，打起来完全是两回事。

| 成长状态 | 战斗表现变化 | 实战判断要点 |
|----------|-------------|-------------|
| **吞噬同化活跃期** | 能力开始叠加，攻击方式变多，但核心还不稳，被打断吞噬节奏会露出空档 | 截断它的吞噬对象，逼它吃不饱，核心就会紊乱 |
| **侵蚀蜕变进行中** | 开始和区域绑定，越打越难拆，但它对载体的依赖也在加深 | 净化载体比硬打怪异本体更有效，载体一断它就大幅削弱 |
| **规则共振前兆** | 局部时空开始不稳，规则出现裂缝，怪异本身也在承受共振压力 | 共振期是怪异最脆弱也最危险的时刻——打断它可能直接湮灭，但没打断就是禁忌级降临 |
| **成长完成稳定期** | 核心已经稳在新等级，能力完整，弱点更少 | 只能正面硬碰，按对应等级的应对要求来打 |

> **战斗写法要点**：遇到正在成长的怪异，优先判断它卡在哪条成长路径上，再决定是截吞噬、断载体还是抢共振窗口。这比无脑硬打更有效，也更好看。

### 事件联合评估模板

怪异等级、公会灾级与反面事件等级不能互相替代。任务设计与处置权限统一按以下字段联合评估：

| 字段 | 回答的问题 | 示例 |
|---|---|---|
| **敌人等级与危险标签** | 面对的主要敌人有多强、如何杀人 | 危险上位·区域绑定型 |
| **事件规模** | 已经影响多少人、多少区域、多少设施 | 单栋建筑、城区、城域、百城 |
| **反面状态** | 是否存在反面浮现、潮涌或降临 | 无反面参与 / 局部浮现 / 潮涌 |
| **情报完整度** | 是否知道核心、载体、弱点与撤离线 | 完整、部分、严重缺失 |
| **处置权限** | 谁可以接触、谁可以稳定处理、谁负责封锁善后 | 学生监管任务、导师带队、公会王牌压阵 |
| **公会灾级** | 整个委托需要什么规模的组织响应 | 狼灾、虎灾、鬼灾、龙灾、神灾 |

联合评估示例：

| 场景 | 联合评估 | 处置结论 |
|---|---|---|
| 学院受控实训 | 普通中位·高速扑杀型；房间级；无反面异常升级；情报完整 | 可作为导师监管下的新生小队任务 |
| 城区旧设施异常 | 危险上位·区域绑定型；城区级；局部反面浮现；核心位置未知 | 即使敌人未到灾难级，也应按高风险虎灾或鬼灾前置处理 |
| 百城封印点潮涌 | 灾难中位·规则共振型；百城级；反面潮涌；情报严重缺失 | 龙灾级组织响应，需要区域霸主级以上压阵 |
| 城域反面降临 | 敌人等级可能未确认；城域级以上；反面降临；撤离线失效 | 先按神灾前兆封锁，不得等待单体怪异等级确认后再升级响应 |

## 七、好看的打戏要怎么写
1. **先让人知道敌人可怕在哪**：这东西到底是快、硬、诡异，还是会控场，要先说清
2. **谁干什么要清楚**：谁顶前面、谁保人、谁找弱点、谁拿最后一拍，要让读者看得明白
3. **别把每场都写成平均分戏的大团战**：大多数战斗更适合单人、双人或局部协同，镜头集中，高光才会硬
4. **别让主角一上来全包了**：先让队友打，打不住了再让她收，爽点才会更大
5. **真正的大高光要留关键时刻**：越级战、卷末战、队友全倒、身份快露的时候再狠狠干一波
6. **打完一定要有结果**：谁死了，谁伤了，核心碎没碎，局面有没有稳住，都要交代清楚

## 八、主角团标准战斗配置

| 角色类型 | 职责 |
|----------|------|
| **主角** | 队伍最后的底牌。平时让队友练，真顶不住了她再把战场收掉 |
| **战斗型伙伴** | 顶前排、打正面，是队伍里最亮眼的明面主力 |
| **室友型同伴** | 负责稳团队、保精神状态、救场和保人 |
| **术式归档型伙伴** | 负责资料归档、任务记录、术式库存火力、核心频率判断和战后证据保存 |
| **古脉代表型伙伴** | 负责复杂地形、遗迹、特殊区域里的作战支持 |
| **高位引路人** | 负责压住失控局面、封战场、挡高层风险 |

### 主角团出手原则
1. 这张配置表是为了让每个人有稳定职责，不是要求每场都把所有人一起摆上场
2. 常规战斗优先考虑：**单人战 > 双人战 > 三人局部协同 > 四人完整小队战**
3. 四人一起上的场面，优先用于巡夜、守楼、封锁、救场、卷中攻坚这类必须体现规模感的章节
4. 就算是四人战，也尽量只留一个绝对高光位：有人负责开局，有人负责稳住，有人负责送出机会，有人负责最后收掉
5. 卷末和终局战可以先用团队把局面打到临界点，再由主角或某个明确角色单独把最后结果拿下来

### 主角团双轨战斗风格对照表

| 角色 | 个人核心主轴 | 通用术式习惯 | 战斗观感 | 写法提醒 |
|------|--------------|--------------|----------|----------|
| **主角** | 对外伪装为罕见冰霜核心；真实为三月权柄 | 很少真正构筑术式，必要时让旁观者误判为静默术式 | 别人还在调用核心或构筑术式，她已经开始定结果 | 作者层不能把她写成“无吟唱术式专家” |
| **白砚河** | 裂锋核心技贴脸找裂、破甲、追斩 | 只用少量行式、护式补位，不擅长复杂阵式 | 贴脸爆发、硬冲硬撕、正面破防 | 裂锋不吟唱；公共术式才有节省吟唱 |
| **林见星** | 断线瞳读织、星丝写织，控怪救人 | 察式、护式与撤离类行式适配较好 | 看线、铺线、拉回队友、缝合阵型 | 星丝不叫术式；第一卷禁用后期界临 |
| **苏弥** | 归档与封印书负责读取、保存、调用 | 把标准术式预先构筑进书页，战时连续翻页 | 个人核心管理公共弹药，准备越充分越强 | 必须分清“书是核心、页中结构是术式” |

### 主角团双轨风格细化

#### 1. 主角
- **平时状态**：权柄结果被主动压成冰霜核心技外观，偶尔被误判为静默术式
- **对外伪装**：尽量让别人以为她拥有罕见冰霜核心、极高适配，或者能够静默触发少数术式
- **真正不能乱开的部分**：完整高位展开只能留给卷末高光
- **主动降档**：她不是压住体内失控力量，而是主动控制输出上限、动作精度和能力外观，让结果落在旁人能解释的范围内
- **写法要求**：别老直接说她强多少，靠结果、旁人误判和压迫感去写；不要用“体内涌动、按回去、钻出来、反噬失控”描写伪装态出手

#### 2. 白砚河
- **平时怎么打**：裂锋附着、破甲拳、裂痕追斩全部直接沿核心逻辑发动，讲究冲上去就撕开
- **公共补位**：可以用低序行式抢位置、护式扛第一拍；复杂术式会拖慢他的近战节奏
- **解放方向**：从寻找已有裂线，推进到让潜藏的断裂可能显现，再到主动造裂
- **写法要求**：白砚河要写出核心技爆发感，不要给裂锋补吟唱或公共回路

#### 3. 林见星
- **平时怎么打**：她用断线瞳看见空间裂缝线、异核能量线、攻击牵引线和生命衰弱线，再用星丝核心能力剪断、拉回、钉住、缝上
- **为什么这样写**：她是丝线控怪救人位；怪要扑谁，她先断线，人要掉队，她先拉住
- **什么时候推进高阶状态**：第二卷后段允许星罗归网雏形；第三卷触及真名深化；第四卷以后进入织境界临；第五卷以后使用终式不坠之线
- **常用招式**：第一卷只用 **星丝牵 / 断线剪 / 出口线 / 预铺线 / 星丝缝合**；第二卷后段用 **破线流星 / 星罗归网雏形**；第三卷用 **七星回缝**；第四卷以后用 **星网坠牢**；第五卷以后用 **终式·不坠之线**
- **写法要求**：动作必须落在线上：攻击牵引线被剪断，队友被丝线拉回，裂开的伤口被星丝临时缝住，空间裂缝线用实线验证。不要只写“她看见路径/稳住护场”。

#### 4. 苏弥
- **平时怎么打**：封印书是“归档”的核心显相；她把低阶禁式、净式、攻式、护式提前构筑成页，战斗时翻页激活
- **为什么这样写**：她前期不是规则解读员，而是资料归档和术式库存位；战斗价值来自“带了什么术式页”“能不能稳住队伍节奏”和“能不能判断核心频率”
- **什么时候进入书阵界临**：归档积累、理解深度与书页库存都达到域主门槛后，所有已归档页才能进入即时调用状态
- **常用表达**：止裂、净尘、缚章、短炮属于书中预存标准术式；连续翻页、复写、组合、封存与调用方式属于她的核心应用。缚章登记为【禁式·十四，缚章。】
- **写法要求**：她要有“翻页即出手、术式连续落下”的火力感，但单页弱于原施术者；库存用完就必须撤或换打法

### 高位角色核心与术式风格

| 角色 | 个人核心主轴 | 通用术式习惯 | 核心气质 | 写法重点 |
|------|--------------|--------------|----------|----------|
| **方导师** | 压线核心技快速辨界、封回、反震 | 熟练使用职业封锁与救援术式补位 | 硬、准、短 | 她一出手，散乱战场重新出现边界 |
| **季悬灯** | 悬灯核心从稳场推进到灯界与立序 | 高位阵式只作辅助与善后 | 高位封锁、秩序压场 | 灯还悬着，这片地方就不能彻底失控 |

### 1. 方导师
- **平时怎么打**：压线核心技讲究快压、快断、快把越界目标按回秩序；必要时用标准封锁术式补阵
- **为什么这样写**：她不是来炫自己多强的，她是来带队、护学生、把烂局拉回来
- **什么时候进入界临**：后续达到域主后，界线才会成为整片战场的默认结构；第一卷只使用局部核心技
- **写法要求**：别写得太玄太花，她的强在经验老、判断准，一出手场面就不乱

### 2. 季悬灯
- **平时怎么打**：以悬灯核心直接立灯、照秽、镇形；公共阵式只辅助封路和善后
- **为什么这样写**：他最可怕的不是火力，而是封锁、切断、压场
- **什么时候进入灯界**：只有在必须隔绝外面视线、封死灾情、挡更高层东西的时候
- **写法要求**：他动作不用多，但每一下都很重。别写成热血老师替学生一路清怪，要写成他给整个战场套了一层更高的规矩

### 高位敌人双轨风格储备

后续卷若需要写高位敌人，优先写“完整战斗体系压制”，而不是单纯火力更大。

- **个人轴**：核心技成熟、真名解放清晰，甚至具备稳定界临。
- **公共轴**：高序术式、全吟唱、预设阵地和多人协同。
- **核心气质**：纪律、站位、阵型和战场权限。
- **写法重点**：强在整套体系完整，能把新生刚学会的配合全部压回去。
- **使用限制**：第一卷不使用高位敌人压场。

### 卷内高位战执行提醒

本文件只保留高位战的通用口径：高位角色下场时，重点写封锁、压场、截断、秩序感和战场权限差，不要写成单纯火力更大。

第一卷只写旧训练楼传闻升级为局部危机、巡廊骸、撤离线断掉和四人第一次协同；具体执行口径统一见《第一卷分卷大纲》《第一卷事件链》。

### 双轨对杀镜头顺序

#### 1. 第一步：先看清危险
- **先让读者知道怕什么**：这招打中会怎样，谁先看出来，谁先提醒队友
- **别一上来就放决胜技**：先抢一个先手，把对面的起手打断
- **常见动作**：报动向、骗走位、打断抬手、挡视线、逼对面停一下
- **写法要求**：开打前几句就说清“这一拍输掉会出大事”

#### 2. 第二步：先抢位置，再打伤害
- **位置比拳头更重要**：谁被逼到墙角、楼梯口、窄走廊，谁就先吃亏
- **招式要和场地绑在一起**：为什么能封路、卡人、困人，都要写明白
- **常见动作**：逼角落、切队形、断支援、抢高处、抢掩体
- **写法要求**：别写成两个人站平地对轰，要让地形参与战斗

#### 3. 第三步：逼对面出底牌
- **压制的关键**：不是特效更大，而是把对面逼到真名解放、界临或高序术式全吟唱；也可以先锁定、束缚、定住目标，逼出完整术式的价值窗口
- **怎么判断局势**：
  - 对面被逼真名解放或界临：说明个人核心表层打法已被破
  - 对面被逼高序术式全吟唱：说明必须赌完整公共构筑
  - 对面只能用节省吟唱硬扛：说明在拼命拖时间
  - 对面强抢无吟唱失败：说明节奏被你打乱了，也说明对方已经被逼到冒险
- **写法要求**：核心状态或吟唱层级变化后，场面必须真正改变，不能只换台词和特效

#### 4. 第四步：每一击都要有代价
- **别只写“吐血”**：要写清到底坏了什么，比如手断了、护盾碎了、阵型散了
- **好看的受伤**：被钉住、被打断、被逼退、被压得不敢抬头
- **写法要求**：重击后场面要变，不能打一圈还是原样

#### 5. 第五步：结尾要“定输赢”
- **合格收口**：不是多打一拳，而是让胜负直接定下来
- **常见动作**：钉核心、断后手、冻关键一拍、封退路、救回队友
- **写法要求**：最后一招要让读者明白“这场已经结束了”

### 常用收口模板

#### 1. 钉杀型收口
- **适用角色**：主角恒月线、封印处理位、早期穿行制度相关术式
- **标准顺序**：先封住位置 -> 再断掉动作 -> 最后钉住核心
- **读感重点**：重点不是打飞，而是把敌人定死在最难受的一瞬间

#### 2. 断线型收口
- **适用角色**：主角虹月线、苏弥、高速刺杀位
- **标准顺序**：先看穿后手 -> 再切掉翻盘机会 -> 最后斩断主连招
- **读感重点**：重点不是伤害更高，而是对面后续招全断了

#### 3. 冰霜压制型收口
- **适用角色**：主角冰霜强攻线、冰系压制位、护场角色
- **标准顺序**：先用强输出打慢速度 -> 再以低温破坏能量流 -> 最后截断关键反击点
- **读感重点**：像整场被寒潮压到失速，只剩最后一击还能动

#### 4. 星丝救场收口
- **适用角色**：林见星、丝线救援位、团队快被打散的战斗
- **标准顺序**：先用断线瞳看见命中线/断裂线 -> 再用核心技“星丝”剪断攻击或拉回队友 -> 最后把队形重新缝回战线
- **读感重点**：不是站在后面兜底，而是红线被剪断、白线被续上，快输的局面被她硬拉回能打

#### 5. 封页覆盖收口
- **适用角色**：苏弥、封印处理位、术式库存型角色
- **标准顺序**：先确认敌人/核心/撤离点 -> 再翻页激活防护或压制术式 -> 最后用低阶术式连续覆盖关键位置
- **读感重点**：不是站在后面解说，而是符页一页页落下，把继续恶化的战场压回可处理范围

### 战斗写崩禁区
- **别这么写 1**：打了很多招，但位置、目标、代价都没变化
- **别这么写 2**：用了高序术式全吟唱或界临，却只是字多了，场面没升级
- **别这么写 3**：本来是控场角色，却每场都变成硬拼火力
- **别这么写 4**：主角高位出手只剩“更大更亮”，没有“接管战场”的感觉
- **别这么写 5**：明明已经收尾，还硬拖两三轮
- **别这么写 6**：为了抬敌人，故意把队友写笨

### 主角团战斗逻辑

核心原则：**主角不需要配合，配合是为了队友成长。**

主角一个人就能打赢任何仗，但她不会一上来就替所有人清场。她让队友打，是因为队友需要实战才能成长；她出手兜底，是因为队友扛不住了。配合不是赢的条件，而是队友练级的过程。

队友的配合逻辑随能力成长而升级，每升级一次意味着他们能独立处理更强的敌人，主角需要出手的频率就越低。

**群像剧补充原则**：每个阶段必须至少有一场"主角不在场或主角不参与，配角独自解决问题"的段落。配角之间也要有独立于主角的战斗默契，不能只围着主角转。

第一卷队友成长、四人第一次协同和旧训练楼局部危机执行口径，统一见《第一卷分卷大纲》《第一卷章节目录版节奏表》。

## 九、主角的特殊规则
1. **伪装原则**：主角大部分情况下必须维持普通学生伪装
2. **战斗观感**：明面上的战斗观感应是「强得离谱但仍能被理解成神秘体系天才」
3. **高位展开**：只有在重大危机、关键战役、剧情高点时，才允许短暂展开接近女神位格的力量
4. **身份保密**：即便展开女神状态，对外也要包装成特殊神秘、未知遗物、异常共鸣或极端战斗表现
5. **体系外例外**：常规角色需要调用核心、构筑术式或进入解放，主角却可以绕过两条常规路径直接产生结果；这种差距必须通过零核心激活、零以太调用、零术式构筑与旁人误判体现，不能把作者层真相写成“她的无吟唱更强”

## 十、高位战力展示限制

区域霸主级、世界影响级和月亮级的破坏尺度很大，但前中期不能频繁把战场写成毁城灭国，否则学院篇和公会新人期会失去紧张感。

1. **第一至二卷**：高位者主要通过封锁、压制、隔离、断外视、救场和争取时间表现强度，不要频繁展示毁城灭国级破坏
2. **第三至四卷**：可以让读者看见局部高位对撞，但重点仍是"把灾害压回可处理范围"，不是清地图
3. **第五至六卷**：公会和远征期可以展示会长级、边境级压场，但每次下场都要有代价、政治后果或封锁需求
4. **第七卷**：才允许完整展开世界影响级以上的终局破坏和三月归一的高层秩序改写
5. **写法口径**：高位强者越强，越要写他们为什么不能随便出手，而不是让他们变成随叫随到的战斗工具

高位战力的爽点不只来自破坏规模，也来自他们能把普通人已经理解不了的灾害压住、切开、封掉，给主角团留下继续行动的窗口。

---

## 十一、后续使用规则
1. **能力定位**：所有人物能力设计，必须先判断其属于哪个战斗岗位、哪种偏向、哪个境界
2. **怪异设计**：所有怪异设计，必须先判断其怪异等级、智慧程度、实体形态、战斗风格和弱点
3. **战斗规模**：所有大战设计，必须先判断战斗规模是房间级、楼层级、建筑级、街区级还是区域级
4. **大纲分工**：分卷大纲只负责安排「谁打谁、在哪打、打成什么样」，不再承担全局战力体系设定职责

## 十二、反面空间战斗补充

> **定稿源**：反面核心规则以《世界观设定》为准。本节为战斗层视角的补充说明，若与《世界观设定》冲突，以《世界观设定》为准。当前卷可见尺度与禁写项以《当前硬口径》为准。

### 1. 反面空间战斗规则

**第一卷补充：旧训练楼封印区局部危机。** 第一卷《银序一班》常规战斗只写旧训练楼传闻升级为局部异常夹层：门牌错位、走廊长度不对、巡廊骸、撤离线断掉和异核。敌人必须能看见、能打中、能破甲、有反馈；**入口硬约束（两层）**：① 入口在正面现场**凭空出现+肉眼可见**（无预兆/无过程，公众能看见/能拍照/能进入）；② 入口**关闭后不留下**白灰壳层、干燥裂纹、冷凝水痕、墙体温度、侵蚀残留、气味/气息/怪声等可被识别的反面物理痕迹；不要把第一卷主战斗写成抽象规则谜题，也不要提前写成大型空洞清剿。

**正面与反面边界**：
- 正面现场不表现反面异常，不写白灰、裂纹、冷凝、低温、气息或短时感知影响。
- **入口硬约束（两层）**：① 入口在正面现场**凭空出现+肉眼可见**（无预兆/无过程，公众能看见/能拍照/能进入）；② 入口**关闭后不留下**白灰壳层、干燥裂纹、冷凝水痕、墙体温度、侵蚀残留、气味/气息/怪声等可被识别的反面物理痕迹。
- 入口附近长期滞留才可能造成轻微异常侵蚀；短时靠近只按普通危险区域处理。
- 真正进入反面后，危险来自错位路径、隐藏空间裂缝、怪异本体与长期滞留后的侵蚀累积；防护消耗也随停留时间上升。反面不会在短时战斗中被动削弱核心能力或扭曲术式。

**进入反面后的战斗表现**：
- **路径错位**：危险来自反面空间本身的走廊偏移、门后位置变化和地图失效，不是角色感知被动变钝；林见星可以用星丝验证真实可走路径。
- **隐藏空间裂缝**：裂缝像无厚度陷阱，误入后会被传到不可预测的位置，也可能成为抄近路的机会；它们不密集到妨碍正常打斗。
- **核心与术式正常运作**：个人核心技和通用术式仍按原有触发、回路与适配规则生效。若出现落点偏移、护盾破损或核心失控，必须来自伤势、过载、敌方干扰、地形遮挡或角色自身问题，不能笼统归因于“反面让能力失真”。
- **特殊个体共鸣**：少数核心、血脉或怪异能力会与反面内富集的高浓度气态以太形成同向共鸣，例如白砚河的妖血升温；这是个人适配现象，不是全员增幅、全员削弱，也不等于污染进阶。
- **长期滞留负担**：只有停留时间拉长后，侵蚀累积、防护工具消耗、耳鸣、眩晕和动作迟滞才逐步出现；短时战斗不默认附加这些状态。

### 1.5 第一卷异核形态与击破反馈

异核是第一卷旧训练楼局部危机中可战斗的核心实体目标，也是巡廊骸的弱点节点。本节给战斗层一个具体可写的形态锚点，避免收口时只靠抽象"打掉核心"一笔带过。

**异核形态（视觉锚点）**：
- **位置**：巡廊骸躯干中后段偏下，外露不内嵌，视觉上直接可见，不藏在残骸堆里。
- **外观**：拳头大小的不规则晶体核，外壳灰白带冷青，中心有冷光透出；形态类似骨化不完全的冷凝矿石，边缘有细碎裂缝，光从裂缝里漏出来。
- **状态变化**：巡廊骸活跃时异核缓慢明灭（呼吸节奏），巡廊骸被压制或残骸重组时异核亮度上升、节奏紊乱；这是战斗层判断"它在动还是它被打停了"的视觉依据。
- **数量**：单个巡廊骸只对应一个异核，不写"多头巡廊骸有多个异核"；旧训练楼局部危机中同时在场的可战斗异核数量控制在三到五个，避免战斗变成割草。

**击破反馈**：
- **直接击破**：术式或近身打击直接命中异核外壳时，先看到外壳出现放射状裂纹，冷光外泄加剧；连续命中导致核心整体碎裂，巡廊骸残骸同步停止动作、整体崩散为灰白残片，残片在数秒内失去光泽沉入地面。
- **被压制后击破**：先由白砚河前锋承压拆解巡廊骸防护、林见星缠线校准锁住行动窗口，再由苏弥的封页调用或哥伦比娅主动压低后的冰霜外显命中异核；击破瞬间冷光骤灭，巡廊骸崩解节奏与直接击破一致。
- **未击破的退出**：如果战斗在异核未击破时主动撤离，异核会随巡廊骸一同进入休眠静默状态；数秒内冷光降到极暗、异核周围残骸回缩成静态团块；这种状态是"被打停但没消灭"，不是死亡，下次浮现可能重新激活。

**正文边界**：
1. 异核击破不产生"以太污染外泄""灰雾扩散""壳层散落"等绝区零式副作用；它就是核心碎裂、巡廊骸崩散、光灭即停，过程控制在数秒内。
2. 异核碎裂后的残片不作为正面空间物理残留；它们随巡廊骸崩散后一并退场，不留下白灰、裂纹、冷凝、低温、壳层。
3. 异核节点不写成可拾取道具、不写成"高纯度资源"——按 `术语口径统一表` 第三节口径，它在第一卷只是战斗收口锚点，不承担资源产业链功能。
4. 异核不被林见星断线瞳"看见空间裂缝线"功能直接定位；林见星只确认"路线是否还能撤""这一击打出去能不能收口"，不直接读出异核内部规则。具体收口由前锋/术式/主角主动降档完成。

### 2. 小队共振

**定义**：两个或多个队友在高压战斗中同时出手，如果他们的神秘核心频率和行动节奏对齐，就会产生短时联动效果。

**触发条件**：深度信任 + 同时承压战斗 + 神秘核心频率对齐

**等级**：
- **前兆**：神秘核心产生微弱共振，配合更顺但还没有质变
- **初激活**：共振产生可见效果（能力增幅、范围扩大、持续时间延长）
- **深层共振**：参与者可以无语言沟通，直接感知队友的意图和位置
- **完全同步**：极罕见，多名队友如同一人行动，协同效率与组合结果出现质变（第一卷不会达到）

**代价**：
- 共振结束后短暂脱力
- 共振越深，参与者之间的情感联结越强
- 如果参与者之间出现信任裂痕，共振会反向崩塌，所有人同时脱力

**多人共振递进规则**：
- **双人共振**：最基础的形式，触发条件相对宽松，只需要两人核心频率对齐。增幅效果稳定，代价可控
- **三人共振**：难度陡增，需要三人核心频率两两对齐，任何一对对不齐都会导致共振偏移。增幅效果远超双人，但脱力时间翻倍
- **多人完全同步**：后续卷如需使用，必须单独在对应卷大纲中安排，不作为第一卷默认机制。第一卷只写普通配合变顺或战术默契提升

**共振增幅边界**：

1. 小队共振首先提升的是信息共享、行动同步、命中窗口、能力衔接与失误容忍，不是无条件放大每个人的基础输出。
2. 共振可以让队伍稳定打出“各自单独做不到的组合结果”，但原则上不能让参与者稳定跨越一个完整战力大层级。
3. 跨层级结果必须依赖明确条件，例如相性克制、预设封印、敌人共振窗口、一次性装备或高位者压阵；共振只负责把窗口抓住。
4. 完全同步不能绕过规则权限差距。面对界临、秩序或神性位格压制时，仍需要对应层级的破局条件。
5. 共振结束后的脱力、核心偏移与信任风险必须实际影响后续战斗，不能只作为氛围描写。

**对哥伦比娅的风险**：
- 共振会让三月权柄本质短暂泄露
- 共振越深，队友越可能感到她的力量有异常位格感
- 这条风险从第二卷以后再逐步使用；第一卷不把身份压力放在共振机制上

### 3. 反面内部战斗环境

反面内部不主打复杂规则谜题，而主打持续侵蚀、路径不可信、实体敌人和撤离压力。战斗环境特点：

- **反面侵蚀**：持续影响神秘者的感知、身体和神秘输出，停留越久负担越重。**真正决定"能待多久/被侵蚀多快"的是抗性三要素（核心偏向契合度 + 规则认知深度 + 封印/净化工具辅助），不是单一装备**；抗性三要素完整机制见 `神秘设定` 第十三节第 6 段。
- **怪异亲和**：侵蚀越重的区域，怪异越容易成型、聚集和强化
- **防护工具消耗**：校服、防护贴片和神秘屏障属于抗性三要素中的"辅助"位——它们**不决定能否进入反面**，只决定"在抗性三要素确定的安全活动时间上能再延长多久"。工具耗尽后，角色仍可凭核心偏向契合度和规则认知深度继续撑，但撑的时间按抗性三要素的基线走，不再享受工具延长的部分。第一卷不写"工具打穿后反弹/反噬敌人"——被打穿就是被打穿。
- **侵蚀递进**：越接近反面核心，侵蚀越强，普通浮现区只是轻微不适，核心区才会出现认知错乱、神秘失控和怪异化风险
- **怪异化终点**：侵蚀不是单纯扣血，而是在把人推向怪异化。战斗写法上先表现为耳鸣、动作迟钝和输出发散，再升级到情绪失控、怪异特征；第一卷常规学生不写完整怪异化，唯一例外是被诱导链精确推至阈值、用于触发卷末局部封印危机的周砺，此案例不可复用
- **核心级异常**：空间畸变、重力偏移、时间错位只保留给反面核心、反面潮涌或终局级场景，不进入常规战斗

**反面战斗动线原则**：
1. 从边缘到核心：动线必须层层深入
2. 路线判断：队伍必须知道哪里能退、哪里不能走、谁负责确认出口
3. 险点设计：每场反面战斗至少一个"慢半拍就会被吃掉"的险点
4. 收口位置：每场反面战斗的收口必须落在明确位置
5. **防护工具状态追踪**：每场反面战斗在收口前必须显式呈现防护工具的当前状态（完整/半耗/耗尽）；不允许把"工具还有多少"留给读者自己脑补。工具耗尽后角色还能撑多久，按抗性三要素基线写，不让工具继续隐性起作用。
