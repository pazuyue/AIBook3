# 《三月女神今天也只是普通学生》八律中段戏剧化设定

---

## 一、定位

八律不是终局真相表格，而是中段必须能写出生活感、冲突感和人物记忆点的世界主体。所有八律成员本质上都是人族，差异来自旧时代三月法则留下的守律古脉、地域体质、家系教育和圣地传统，不写成八个生物种族。

八律共同承担三件事：
1. 展示异世界的日常奇幻感
2. 承接古脉秘史和三月创世分工
3. 在公会远征期制造现实利益冲突

## 二、八律总原则

1. 每一律都有可爱的日常面，也有难缠的现实利益
2. 每一律都保存一段不完整的古老禁忌
3. 每一律都不能一眼看出"我们的传承来自三月创世分工"
4. 第五、第六卷只揭使命和传承，第七卷才揭创世全貌

## 三、八律戏剧抓手表

| 古脉 | 生活特征 | 古老禁忌 | 中段冲突 |
|------|----------|----------|----------|
| 城邦律 | 最擅长制度、学院、公会和城市秩序，喜欢把危险写进规章 | 不能长时间直视三月同悬的旧图 | 城邦组织想用制度管理主角，却发现她不属于制度能定义的范围 |
| 魔女律 | 用口传、手抄本和仪式保存知识，喜欢把重要事写成看似闲聊的注脚 | 禁止完整复原某些旧月仪式 | 苏弥追查魔女断代时，发现古脉内部有人主动删过旧知识 |
| 圣林律 | 用藤蔓、风铃、树屋和长寿礼仪维持生活节奏 | 禁止砍伐某些"无根树" | 圣林对主角产生反应，但长老要求封锁消息 |
| 血契律 | 重视血契、礼仪和夜宴，外表优雅但极讲债务 | 禁止饮用某类月下旧血 | 血契上层认为主角的存在会让古老血契失效，试图试探她 |
| 亡名律 | 行事安静，记忆习俗强，重视"名字被叫住"和失联标记 | 禁止进入某些无声墓地 | 亡名律能感到林见星姐姐失联标记的残响，牵出"空洞失联者" |
| 妖血律 | 重肉身、战斗、血脉和本能，节庆常带竞技 | 禁止未获妖血认可者在霜月晦月期强开妖化 | 白砚河对完整妖化的追求与妖血律旧禁忌冲突 |
| 灵脉律 | 与灵脉、寒潮、地貌通感，生活方式更接近地脉守护者 | 禁止唤醒沉睡灵脉 | 月枝的灵脉守律使命把主角团带入八律圣物线 |
| 海脉律 | 以潮汐、航路、贝文和深海祭礼保存历史 | 禁止潜入无月海沟 | 海脉律掌握三月断联当夜的潮汐记录，但拒绝交给公会 |

## 四、八律现实利益压力表

八律冲突不能只靠"古老禁忌"撑着。每一律都要有现实利益、古脉压力和不愿相信主角团的理由。

| 古脉 | 最怕失去什么 | 最想保住什么 | 为什么不轻易相信主角团 |
|------|--------------|--------------|--------------------------|
| 城邦律 | 百城制度失去解释权，学院和公会体系被证明管不住真正高位异常 | 城市秩序、人才筛选权、灾害处置主导权 | 主角越强，越证明制度无法定义她，城邦高层会先想监测和规训 |
| 魔女律 | 旧档被复原后，证明古脉内部曾主动删史或误导后代 | 手抄本传承、仪式解释权、禁忌知识封存权 | 苏弥追查越深，越可能揭开魔女家系不愿承认的删史责任 |
| 圣林律 | 圣林对主角回应后，引来外部势力争夺古林和无根树 | 圣林自治、长寿礼仪、古地安宁 | 主角团的到来会打破圣林长期封锁消息的稳定状态 |
| 血契律 | 主角存在让古老血契失效，动摇血契上层的债务和契约秩序 | 血契权威、夜宴情报网、贵族体面 | 他们会先试探主角是否会成为旧契约的破坏者 |
| 亡名律 | 空洞失联者的失联标记重新回应，牵出亡名律也无法解释的旧亡名 | 名字档案、墓地边界、亡者记忆习俗、失联标记残响 | 林见星线会碰到他们最敏感的"谁有资格确认失联者已经死亡" |
| 妖血律 | 妖血禁令被证明与三月或旧时代有关，古脉内部强弱秩序被重排 | 血脉稳定、竞技传统、边境战力地位 | 白砚河的成长可能证明旧禁忌不是迷信，而是古脉隐患 |
| 灵脉律 | 沉睡灵脉被唤醒后，灵脉守律使命变成无法逃开的责任 | 灵脉安定、地貌节点、脉规传承 | 月枝一旦站到主角团这边，就等于把脉规和世界危机放上同一张桌 |
| 海脉律 | 潮汐记录公开后，证明某次大灾不是天灾，而是人为或高层失职 | 航路安全、深海祭礼、潮汐档案解释权 | 海脉律知道记录很值钱，也知道公开后会得罪不止一个势力 |

写八律时，每场阻力都要落到"他们在保什么"上。不要把八律写成单纯守旧，也不要让他们见到主角就立刻让路。

## 五、每律与三月的隐性联系

| 古脉 | 隐性联系写法 |
|------|--------------|
| 城邦律 | 负责把混乱世界写成制度，偏向恒月的"存在固定" |
| 魔女律 | 保存知识和禁忌，偏向虹月的"心智与未来分支" |
| 圣林律 | 守护生长和长期秩序，偏向恒月的创造侧 |
| 血契律 | 保存血契和代价，偏向虹月的欲望、心智和毁灭边缘 |
| 亡名律 | 记住亡者、名字和存在痕迹，连接恒月与霜月 |
| 妖血律 | 承接肉身、爆发和本能潮汐，偏向霜月的能力潮汐反面 |
| 灵脉律 | 维护灵脉和环境节点，偏向霜月的现在稳定 |
| 海脉律 | 记录潮汐、航路和三月变化，连接三月轮转本身 |

这些联系前中期只写成传统、禁忌和圣物反应，不要提前说成三月创世分工。

## 六、中段可用事件

1. 八律联合演习：各脉能力暴露出不属于常规神秘体系的古老部分
2. 魔女旧档缺页：苏弥发现缺失页不是遗失，而是被主动撕掉
3. 亡名无声墓地：林见星姐姐事件出现残留线索
4. 妖血霜月禁令：白砚河必须拒绝强开半吊子妖化，并最终通过获得妖血认可、完成稳定妖化，证明旧禁令把暴走结果误当成了妖血本身的罪
5. 灵脉沉睡灵脉：月枝必须在脉规和主角团之间做选择
6. 海脉潮汐记录：三月断联当夜的记录与学院旧巡夜记录互相印证
7. 血契旧宴：主角被邀请，却引发血契异常
8. 圣林无根树：旧时代圣林对主角低功率权柄产生回应

## 七、八律冲突不等于撕队

八律线的冲突应来自：
1. 古脉利益
2. 传承禁忌
3. 对公会和学院的不信任
4. 对三月权柄重归的恐惧
5. 对守护使命和自主意志的矛盾

不要把八律写成"主角来了大家立刻跪服"。
也不要把八律写成单纯阻挠主角的蠢反派。

## 八、第五、第六卷使用方式

第五卷：
1. 公会任务接触八律联络人
2. 圣物/遗迹出现异常反应，但归因为任务触发
3. 八律只表现出"你的神秘不一样"

第六卷：
1. 深入八律领地
2. 长老讲出更深传说
3. 确认八律守护使命与三月创世分工有关
4. 仍不完全揭开"八律传承来自三月创世分工"，这个留给第七卷
