# 核心技与解放归档

## 文件定位

本文是个人核心能力连续性索引，记录正文已经出现或已被跨卷设定正式定稿的：

- 核心能力与核心技；
- 核心引导语；
- 真名与解放语；
- 界临名、范围、锚点、默认效果与反制方式；
- 立序条件；
- 主角体系外能力的对外伪装记录。

通用术式的分类、编号、吟唱与标准回路统一记录在《术式吟唱归档》，不得混入本文。

---

## 一、归档边界

| 项目 | 是否归入本文 | 判断标准 |
|---|---|---|
| 核心能力 | 是 | 只有本人能够基于唯一核心法则完成 |
| 核心技 | 是 | 个人核心的具体战斗应用，可以命名或喊名 |
| 核心引导语 | 是 | 用于稳定认知、身体、媒介或核心同步，不参与公共回路 |
| 真名解放 | 是 | 核心动词、形态与代价同时发生规则性质变 |
| 界临 | 是 | 个人法则投射战场，具备范围、锚点、默认效果与反制方式 |
| 立序 | 是 | 唯一主序在条件成立时获得局部优先权 |
| 通用术式 | 否 | 可教学、可复现、使用标准回路与七十二式谱 |
| 主角三月权能 | 特殊记录 | 不纳入普通核心成长链，只记录对外伪装和体系外证据 |

核心技喊名、核心引导语、解放语和界临落名都不是术式吟唱。省略它们不会因为“节省吟唱”而降低能力完整度。

## 四、真名解放归档

### 归档字段

真名解放首次定稿时，必须一次写清：

| 字段 | 内容要求 |
|---|---|
| 核心表名 | 显相期使用的核名 |
| 完整真名 | 真名解放后的完整命题或称号 |
| 解放语 | 自我确认句，不是吟唱 |
| 动词质变 | 核心从“怎样做”跃迁到“真正作用于什么” |
| 形态变化 | 身体、武装、媒介、环境或核形的可见变化 |
| 可见代价 | 身体、精神、媒介、时间、范围或后续虚弱 |
| 首次出现 | 具体卷章 |
| 后续禁改点 | 不能随意抹去的限制与视觉锚点 |

## 六、立序归档

立序必须来自唯一核心本质，并登记：

1. 唯一主序；
2. 成立条件；
3. 作用边界；
4. 可被怎样抵抗或覆盖；
5. 施加者承担的代价。

当前已定稿的普通角色立序只有季悬灯的`灯界立序`。它是角色既有高位状态，但第一卷能力白名单仍禁止他下场接管旧训练楼危机。其他角色不得把“强力界临”“必中效果”或“威力极大的真名解放”提前写成立序。

### 季悬灯：灯界立序

| 字段 | 定稿内容 |
|---|---|
| **唯一主序** | 只要悬灯仍亮，其锚定范围就必须维持“可被识别、可被隔离、可被处理”的秩序状态 |
| **成立条件** | 季悬灯清醒并主动维持；至少一盏主悬灯完成锚定；副灯或灯线将目标区域连成连续灯界；必须先辨认灾情边界 |
| **作用边界** | 只覆盖主灯、副灯与灯线实际连成的连续区域；灯照不到、灯线未闭合或与主锚失联的位置不受完整立序支配 |
| **核心效果** | 把扩散、异化、精神躁动、规则外溢与外来窥视压回仍可辨认、隔离和处理的状态；保护已登记的己方对象与撤离线；不负责凭空抹除目标或替代终局裁定 |
| **反制方式** | 破坏、污染或夺取主灯；同时切断多段灯线；以同级以上界临或立序局部改写；制造多点并发过载；使用超出其当前认知与处理边界的世界终灾级异常 |
| **代价** | 消耗随范围、并发灾点、规则复杂度与断视强度上升；必须持续分配注意力；锚点断裂会引发秩序回冲并迫使重建；全力立序会暴露世界影响级灾情的战略信号 |
| **连续性来源** | 《季悬灯能力发展设定》“正式立序卡：灯界立序” |

## 七、主角体系外记录

哥伦比娅不拥有可按常规体系解释的唯一神秘核心。学院可以把她误判为稀有冰霜核心、静默术式特例、异常适配或遗物辅助，但作者层必须保留三项体系外证据：

```text
零核心激活
零以太调用
零术式构筑
```

结果仍然发生时，不得把它登记为：

- 无吟唱术式；
- 未知编号术式；
- 真名解放；
- 界临；
- 立序。

女神态、三月视界、创世态和三月权能另按《主角能力发展设定》记录，不占用普通核心成长链名额。

## 八、维护规则

每章定稿前，出现新的核心技名、核心引导语、完整真名、解放语、界临名或立序条件时，必须补充本文。

最小记录格式：

```text
角色：
神秘核心：
名称：
类型：
喊名/解放语：
首次出现：
能力结果：
可见代价：
后续禁改点：
```

判断顺序：

1. 先问“别人能否通过标准教学完整复现”。不能，则优先归入个人核心。
2. 再问“它是否依赖七十二式谱与标准回路”。依赖，才归入术式。
3. 招式有名字、角色喊出名字，都不能单独证明它是术式。
4. 核心技与术式可以组合，但必须分别描述和归档。
5. 真名解放、界临与立序必须经过对应成长节点，不得只靠换招名伪装升级。
