# 人物设定导航

> 本目录负责“谁是谁、如何成长、当前能做到什么”。当前卷可用能力仍受 `设定/当前硬口径.md` 和当前卷写作状态限制。

## 一、推荐读取顺序

1. [人物设定.md](人物设定.md)：身份、性格、关系、人物弧和长期人物口径。
2. [主要角色战力定位表.md](主要角色战力定位表.md)：公开定位、实际定位、阶段上限与团队岗位。
3. [主要角色能力发展表.md](主要角色能力发展表.md)：主要角色跨卷成长概览与终局担当。
4. 对应角色的专项能力文件：核心、核心技、等级进度、代价与解锁条件。
5. [角色能力发展设定模板.md](角色能力发展设定模板.md)：新建或重构角色能力文件时使用。

## 三、职责边界

- 人物总表回答“这个人是谁、与谁有什么关系”。
- 战力定位表回答“当前公开/实际处于什么位置、承担什么岗位”。
- 能力发展总表回答“跨卷怎么成长、终局做什么”。
- 角色专项文件回答“唯一核心是什么、阶段能力如何解锁、代价与反制是什么”。
- `设定/核心技与解放归档.md`只保存已经定稿的名称、解放语和连续性状态，不替代角色专项设计。
- `设定/术式吟唱归档.md`只保存角色实际使用过或已确定的通用术式。

冲突时，当前卷可用范围以当前硬口径和当前写作状态为准；跨卷能力设计以角色专项文件为准；总表只作为概览。
