# 《三月女神今天也只是普通学生》主角能力发展设定

***

## 说明

本设定为全书核心能力设定，严格服从《设定文件》《神秘设定》《战斗设定》的总框架。

本文件按《角色能力发展设定模板》的主角体系外例外执行。主角不使用单一正式神秘等级描述真实力量，改用“权柄接驳阶段 + 对外神秘等级/身份 + 实际战力层级”三轴记录；当前卷可用能力以当前硬口径和当前写作状态为准。

本书明确属于隐藏身份无敌流。主角不是"从弱到强的升级角色"，而是：
**开局就是天花板级存在，只是她平时把真正的天花板压着不用。**

因此，她的成长主骨架必须严格服从《设定文件》的固定五阶段：
**普通学生伪装期 -> 异常月相适应期 -> 三月权柄解锁期 -> 三月法则链接期 -> 完整月权执掌期。**

在这个五阶段主骨架之上，她的战斗表现层再采用：
**伪装态 + 女神态 + 创世态**\
也就是说，五阶段决定她接过了多少权柄，三种模式决定她在当下这场战斗里开到什么程度。

核心原则：

1. 三月是力量、法则与权柄，不拟人化
2. 主角大部分时候维持"普通学生伪装态"
3. 只有在重大危机、关键战役、剧情高点时，才切换到更高位的出手模式
4. 她的"成长"是接过三月女神死后留下的权柄，不是变强

用词口径：

- **权柄**用于宏观权限、位格和成长进度层面，如三月权柄、权柄回收、三月权柄解锁期。
- **权能**用于具体能力落地层面，如过去复写权能、改变未来权能、静止世界权能、生死操控权能。
- 阶段名统一用"权柄"，具体招式/能力项可用"权能"。

### 术式吟唱与高位展开口径

这里和《战斗设定》保持一致。

常规神秘体系里的术式发动，大体分三种吟唱层级：

- **全吟唱**：把术式完整放出来，最强、最稳，但也最慢
- **节省吟唱**：少走几步，速度更快，但威力和完整度会打折
- **无吟唱**：直接让术式成型，最快，但对大多数人来说最难

对普通术者来说，这主要是熟练度和风险的问题。\
但对主角来说，不是“她更会用术式”，而是**她本来就不用走普通术式那套流程**。

因此，主角后续所有能力写法都默认遵守下面这条特殊规则：

- 她平时随手用出的无吟唱，就已经能压过很多精英术者的全吟唱术式
- 她如果真把同类结构完整放出来，就不再属于普通术式能讨论的范围
- 正文里不要老直接说“百倍”，尽量用战场结果、旁人误判和强烈对比去写这种断层

执行边界：

- **吟唱**：只管普通术式发动流程，全吟唱/节省吟唱/无吟唱是术式层术语。
- **展开**：只管领域、半展开、女神态、视界或高位战场结构铺开，不再写全展开/节省展开/无展开。

## 一、五阶段主骨架与三种出手模式

### 五阶段主骨架

主角的成长不是普通意义上的升级，而是把三月女神死后留下的力量一层层接过来。固定阶段如下：

1. **普通学生伪装期**：伪装成普通学生，重点是藏身份、适应身体、维持日常
2. **异常月相适应期**：开始适应三月轮转带来的身体和战斗变化，三条月线慢慢冒头
3. **三月权柄解锁期**：恒月、虹月、霜月的代表招式基本成型，开始有卷末级收场能力
4. **三月法则链接期**：不只是会用能力，而是开始碰到更深一层的法则和战场规则
5. **完整月权执掌期**：三月真正归一，主角完整执掌月权，负责终局裁定

### 三种出手模式

三种模式不是拿来替代五阶段的，而是她在不同场面下会开到什么程度。

不是她不会打高位战，而是大多数时候根本没必要：

### 模式一：伪装态（日常战斗）

- **使用场景**：95%以上的战斗场景，日常巡夜、实战课、普通怪异事件
- **对外观感**：光明属性特殊神秘、罕见净化系天赋
- **核心逻辑**：所有效果都能被解释为常规光明系神秘，绝不暴露三月权柄本质
- **术式表现口径**：对外要尽量装成“她只是特别熟练”或者“她吟唱特别短”，不能让人一眼看出她根本不是走普通术式流程
- **表里关系**：表面上，她像个很夸张的光明系天才，会净化、会感知、会封印；实际上，她每一下都带着三月权柄最外层的判断惯性。伪装态只能把这种判断压成“更快、更准、更轻”的战斗结果，不能直接写成完整裁定
- **身份保密兜底**：虹月那部分精神和虚实能力，可以帮她在关键时刻制造短时间的错认和判断偏差，把高位表现重新包装回“还说得通”的范围。但这不能当成平时随便洗记忆的万能手段，而且对太高位的存在也不稳
- **视界表现**：平时只露出低配版的`月痕照界`，看起来像感知特别强，但本质上已经是三月视界最外面那层

#### 伪装态状态与技能总表（唯一口径）

伪装态不是一个固定强度档位，而是“隐藏身份时的外显状态”。它会随着五阶段推进而解锁不同能力，但外观、误判和压制上限必须始终受当前卷约束。

| 状态项      | 统一口径                                                                   |
| -------- | ---------------------------------------------------------------------- |
| **公开身份** | S级/天才级神秘者，偏光明、净化、感知、封锁方向                                               |
| **真实底层** | 三月权柄被压到最外层，只留下判断节点、截断动作、低频冷白反馈等边缘表现                                    |
| **战斗上限** | 上限固定为“天才神秘者的极限表现”，不能出现对手完全动不了、术式完全卡死、存在被钉住这类女神态压制效果                    |
| **爽点来源** | 比别人快半拍、准半寸、轻一招；她像提前知道结果，但正文仍要保留可解释空间                                   |
| **误判落点** | 高熟练度无吟唱、S级适配、复合神秘、异常共鸣、静默触发式术式                                         |
| **禁区**   | 不写完整领域、不写月轮显形压场、不写存在裁定、不写记忆清洗式兜底；不写伪装态有反噬、异物感、皮肤下涌动、压制失败或失控风险（任何阶段均适用） |

**瞳术（独立分类）**

`月痕照界`：三月视界的伪装态外显，属瞳术，不计入战斗技能池。**能力严格对照六眼 4 特征**（2026-06-12 校准）：

| 月痕照界能力                       | 战斗表现                            | 日常表现               |
| ---------------------------- | ------------------------------- | ------------------ |
| **看见节点**——攻击/术式的"核心节点"       | 看到对方术式的"能量流动"+"结构路径"            | 快速理解+动态视力（图书馆快速翻书） |
| **消耗最小化**——同样的冰能力消耗更少        | 同样的冰刃/冰墙/冰盾——主角用 1 份 = 对方用 10 份 | 长时间战斗不疲劳——S 级适配可解释 |
| **看破破绽**——看到对方术式的破绽/起手点      | 看到对方出招的"起手前 0.5 秒"              | —                  |
| **看下一步**——"提前 0.5 秒看到对方要做什么" | 提前 0.5 秒布置冰能力——对方撞上             | —                  |

**触发条件**：需主动将注意力投向目标才触发，**不是被动全域扫描**。

**第一卷低配上限**：

- 只能看**当前节点**——**无法预判多步连锁**
- **无法穿透高阶意识屏蔽或刻意误导**
- 切换目标需要**一拍重新聚焦**——**不能同时锁定多个目标**

**非战斗情境下主动投入注意力同样可用**（如快速翻书+快速理解）。

**对外观感**：

- "她像提前算好了每个落点"——**战斗**——**看见节点+看破破绽+看下一步**
- "她翻书的速度不像人"——**日常**——**看见节点+看下一步**——**快速翻书+快速理解**
- "她做的事比用的能量多"——**消耗最小化**——**同样的冰能力消耗更少**

**禁止写法**：

- 不写成"看见该冻住的点"（**这是旧设定的错描述**——**主角是"看见节点"+"看破破绽"+"看下一步"+"消耗最小化"——不是"看见该冻的点"**）
- 不写成被动全域扫描
- 不看完整时间线
- 不看完整生死线
- 不直接裁定结果

#### 冰能力常用表（**非技能**，2026-06-12 补）

按"战斗 = 技能（20%）+ 能力（80%）"分层——主角在伪装态下，**战斗 20% 用技能**（霜楔/霜华/霜印/第二阶段三技能），**80% 用冰能力**（即"冰的物理/规则延伸"——没有固定名，但日常战斗天天用）。

**这些不是正式技能**——**不计入战斗技能池**——**不喊名**——**不能作为伪装态"基础技"或临时短名引用**——**它们是"冰能力"的具体物理/规则延伸**——**配合技能一起用**——**不单独作为"伪装态基础技"出现在正文战斗描写中**。

|---|---|---|
| 冰刃 | 把冰凝成刃的形状——有形之刃 | 视觉冲击 |
| 冰墙 | 凝成墙——挡攻击/封退路 | 物理感 |
| 冰锥 | 凝成锥——远程攻击 | 视觉冲击 |
| 冰盾 | 凝成盾——防守 | 物理感 |
| 冰锁 | 凝成锁链——锁住对手/锁住自己 | 视觉冲击 |
| 冰盾·折光 | 凝成镜面盾——折射攻击 | 视觉冲击（与霜华联动） |
| 冰钉 | 凝成钉——从地面/墙/空中突刺 | 恐怖感 |
| 冰甲 | 凝成护甲——物理防御 | 物理感 |
| 冰步 | 凝成冰台阶——自由走位 | 视觉冲击 |
| 冰牢 | 凝成笼——困住对手 | 视觉冲击 |

| 冰爆发 | 集中所有冰能量一次爆发——大招 | 输出感 |
| 冰塑形 | 把冰塑成任意形状——战术塑形 | 战术感 |

**关键约束**（执行红线）：

1. **不喊名**——冰能力常用表里的能力**没有名字**——主角使用时**不喊"冰刃"/"冰墙"**——**只通过动作/效果描写**——读者通过画面"她凝成了什么形状"自然理解。
2. **不作为独立技能**——冰能力是"配合技能用"——单独使用冰能力而不配合技能时，**不写成"主角用 X 技能"**——**只写"主角凝成 X"**。
3. **与正式技能的区分**——冰能力常用表的能力**不能临时起短名**——不能叫"冰之刃"/"霜之盾"等——**伪装态战斗技能只能用伪装态状态与技能总表里已列出的 6 个技能**（霜楔/霜华/霜印/霜潮领域/霜镜折光/霜印·禁核）。
4. **正文用法**——冰能力是日常战斗 80% 的基础——**配合技能一起用**——**不单独作为"伪装态基础技"出现**——例如：Ch10 主角用"霜楔"+"冰锥"组合——霜楔是技能——冰锥是冰能力。
5. **不引入恒月/虹月**——冰能力都是冰系——**没有"光系"/"净化系"/"时间系"等伪装**——主角对外始终是"S 级冰系控场天才"。

**战斗 = 技能（20%）+ 能力（80%）**——**技能关键时刻用**——**冰能力日常战斗用**——**两者配合**——**完整伪装态**。

命名口径：本表是伪装态战斗技能命名的唯一来源；第一卷战斗喊名只用短名（`霜楔`/`霜华`/`霜印`，`月痕照界`属瞳术，不在此列），全名只用于学院档案/正式记录/归一之躯内部命名。喊术式名不是吟唱，也不参与发动；它只是战斗节奏和队友提示。真实仍是无吟唱发动，正文不得临时造新的伪装态技能名。

**补充口径·写法注**

- **三技能底层共同逻辑**：三个技能的核心都是冻结控场，共同效果是战斗拖得越久，对手越不利——`霜楔`主动播种蔓延，`霜华`把对手进攻反转为反标蔓延，`霜印`一点接触收束全部霜网。对外统一包装为高阶冰系控场天才的术式体系。
- **三技能联动写法**：`霜楔`植下霜线→`霜华`截击时对手节点继续被反标→`霜印`贴上去封固或引爆已积累的霜网，三步是一套连贯打法；不需要三技能同时出现，任意两技能衔接也能写出联动感。
- **实体适配（对非人类怪异单位）**：`霜楔`作用于目标悬停动能节点或能量集能点，效果等同；`霜华`截停实体冲击动能，绽出霜晶后反标攻击来源，同样可解释为S级感知卡点。
- **`霜印`双向选择时机**：封印方向适合对手已积累足够霜线、需要生擒或保留的场景；破坏方向适合彻底收口场景，霜线越多引爆范围越广；两种方向都从同一接触起手，外观在渗透方向上才分化，不写成两套完全不同的动作。

第一卷执行口径：战斗技只允许使用`霜楔`、`霜华`、`霜印`；瞳术只允许使用`月痕照界`。第一卷不得把伪装态写成完整光明系工具箱，也不得使用恒月、虹月的稳定外显能力来解释收口。

第二卷执行口径：后 3 个技能进入可用池，但只能作为三线分化边缘、训练误差、档案误判或短暂伪装态反馈出现；第二卷高光仍只给霜月女神态，恒月/虹月不做完整高光。

**爽点核心**：扮猪吃虎、离谱一招定胜负、别人苦战她随手收场。

**写法限制**：必须保留伪装空间，不能打出无法用常规神秘解释的效果；如果旁观者理应看出异常，也必须立刻给出“高熟练度无吟唱”“S级适配”“静默触发式术式”“异常共鸣”这类误判落点。

三条半展开线：

##### 恒月半展开·辉锁镇狱

- 金色辉光域铺开的瞬间，成片月钉、月链、月环会从上空和地面同时压下，把目标手脚、影子、核心一起钉死
- 域内所有目标的存在状态被强行锁死，无法虚化、无法逃遁、无法自爆、无法临时突破能力上限，挣扎越狠，锁得越重
- 它真正可怕的地方不是“锁住”，而是把目标此刻的一切成立条件一起裁掉：冲锋只能停在半路，再生只能停在将成未成，爆发只能卡在爆发前一拍
- 在极端高压下，可把区域霸主级怪异短时从半能量状态砸回实体，硬生生打断它最难处理的那一段变化；高光写法要像“不是她打中了它，而是它被这片月光判回了该被钉死的形态”
- 对外解释：特殊的「领域级存在固定术式」

##### 虹月半展开·断空红弧

- 抬手就是一道横跨数百米的猩红月弧，像把整片战场从中线硬劈开
- 这一斩切掉的不只是防御，而是目标接下来所有可能成立的动作链，招式、位移、爆发、反扑会在出手前就被斩断
- 真正可怕的地方不是威力，而是“先删结果再落刀”：她挥出去时，被斩中的目标往往还没意识到自己已经失去了翻盘、反击和继续往下打的资格
- 在最关键的一击里，可正面劈碎区域霸主级怪异的防御外壳，甚至把已经成型的大范围杀伤术式强行截断；高光写法要像“不是她砍得更狠，而是对手那条本该继续成立的未来被提前抹掉了”
- 对外解释：「规则级未来截断术式」的罕见高阶版本

##### 霜月半展开·零度封界

- 白域张开的瞬间，整片区域像被一只看不见的冰手猛地攥住，能量流、物理运动、能力运转速度直接砍掉九成以上
- 火焰会在翻卷到一半时冻成白晶，扑杀中的怪异会连同惯性一起僵在半空，重炮级冲击也会被压成一堵白色冻墙
- 它真正冻结的不是低温，而是“现在这一刻继续往后流动”的资格：爆炸只能停在将爆未爆，冲锋只能停在将至未至，伤势和崩塌也会被一并按在原地
- 可瞬间冻住区域霸主级怪异最关键的一轮动作与能力运转，把已经扩散出去的精神侵蚀波硬压成看得见、砸得碎的结晶；高光写法要像“整片战场被她按了暂停，只剩她自己还在往前走”
- 对外解释：罕见的「全域能量静止类神秘」的极限爆发
- **状态限制**：
  1. 单次展开最长不超过3分钟，时间再长会出现完整月轮虚影，身份暴露风险暴增
  2. 展开后高位法则残留与目击风险会持续至少12小时；这不是身体反噬，也不影响她切回伪装态
  3. 同一剧情卷内最多使用1次
- **爽点核心**：别人已经收不了场时，她一站出来就把战场接过去；重点不是她更强，而是她开始规定这场仗该怎么结束
- **写法限制**：不能频繁使用，每次使用都必须是剧情高点

## 二、真实核心能力与三月三线权柄

> **能力定义层**：本节只描述她拥有什么能力，不含卷次约束。卷次 × 阶段使用规则统一见第三节。

### 底层真实核心

她的力量最底下，可以先简单理解成四块：**规则、心智、虚实、生死边界**。

恒月、虹月、霜月，其实就是这四块东西在不同月相下的不同表现。

而她平时对外露出来的“光明系能力”，只是最表面、最方便藏身份的一层皮。

这部分是作者内部理解用的，不适合在正文里一次讲穿。

**三月权柄底层逻辑写法约束**：正文里三月权柄的底层逻辑（规则、心智、虚实、生死边界四块核心能力，恒月/虹月/霜月各自的权能对应，三月视界的分层展开）只能通过战斗结果和旁人误判间接呈现，绝不能由任何角色一次性讲穿。苏弥的归档能力可以让她逐步拼出碎片，但每次拼出的量不能超过当前卷的秘史推进表允许的揭露程度。

### 核心能力与三月权柄的对应关系

- **规则**：这是最底层的基础。恒月的稳住、霜月的按停、虹月的断未来，本质上都和“她能改局部规则”有关
- **心智**：主要落在虹月线上，负责压判断、带偏感知、撕意志、做更高位的精神干涉。正文里尽量少解释，多通过结果去写
- **虚实**：主要藏在恒月和虹月的一部分能力里，负责真假错位、表里错位和误导战场，也是她藏身份的重要暗手
- **生死边界**：主要由恒月和霜月碰到，负责“留住一口气”“把人按住”“把快断掉的命硬拖回来”这种高位收口

### 三月视界

`三月视界`是主角最核心的“神之眼”能力。

它不是额外外挂，而是三月权柄在“看见、判断、锁定、裁定”这四步上的直接表现。

- **伪装态**：仅显露为`月痕照界`
  - 看异常核心
  - 看术式结构
  - 看情绪破口
  - 看一场战斗里最该先被打爆的位置
- **女神态**：三月视界开始分线展开
  - **恒月视界**：看过去残痕、旧态落点、存在钉落点，像是能直接看见"哪一段过去接回来后整场就会重新站住"
  - **虹月视界**：看未来分支、过程断点、心神破绽，像是能直接看见"删掉哪一步之后，对手只能抵达败局"
  - **霜月视界**：看现在节点、能量停滞点、生死临界线，像是能直接看见"哪一拍一冻下去整场就会停"
- **创世态**：三线合一，进阶为`三月天目`
  - 不再只是看见目标的过去、现在、未来
  - 而是能在看见之后，直接裁定哪一条时间线、哪一种生死状态、哪一场战斗结果继续成立
- **外观与开眼特征**：
  - **月痕照界**：瞳孔浮出银白月纹，清亮到不像普通人
  - **恒月视界**：眼底泛开淡金色光，瞳孔边转出细小金色月轮，神圣安静、高位压迫
  - **虹月视界**：瞳孔拉开猩红细痕，艳而危险，美得不正常但让人后背发紧
  - **霜月视界**：眼底透出白银冷光，干净冷冽如雪夜月光，盯住人时让人本能放轻呼吸
  - **三月天目**：金、红、白三色同时在眼底亮起叠转，漂亮到近乎不真实；对上视线的人第一反应不是"她在看我"，而是"她已经决定我会怎么结束"
- **写法要求**：三月视界一定要写成“看到了就能打、看穿了就能压、看见破口就能定输赢”的战斗型神眼，别写成只会分析说明书的辅助眼

### 三月三线权柄

三月的能力必须始终围绕四大核心能力展开，不添加无关能力。

### 1. 恒月

| 项目       | 内容                                                      |
| -------- | ------------------------------------------------------- |
| **颜色**   | 黄色                                                      |
| **核心法则** | 过去时间线掌管、过去复写、存在创造、法则创作，核心权能是读取已发生痕迹，把某个已经成立过的过去状态重新接回现在 |
| **时间倾向** | 过去                                                      |
| **战斗风格** | 造物生成、稳固回护、封固钉锁、过去复写、旧态回接、临时规则创作                         |
| **观感**   | 金色月轮、月矛、锁链、屏障、月辉造物、淡金色时间波纹                              |

#### 恒月的实战方向

- **高阶能力（仅女神态以上可使用，伪装态绝对不显露）**：
  - **过去复写权能**：回溯目标区域的过去时间片段，读取已经发生的所有事件细节；可选定一个已经成立过的状态，把"曾经完整""曾经活着""曾经未崩塌""曾经被钉住"这类过去事实重新接回现在
  - **过去改写权能**：可修改某一个关键过去节点，直接改写当前事态走向；这不是普通时间倒流，而是让新的因果链从过去补上来
  - **法则创作权能**：临时创作局部生效的全新规则，让战场完全按自己的规则运转
  - **高阶造物权能**：不只是造出简单的武器屏障，还能根据需求创造出任何符合规则的实体造物，甚至能临时创造出一片适合己方作战的小型独立空间
  - **月狱裁线**：抬手放出数道淡金细线，像在战场上重新划出“可成立”和“不可成立”的边界。任何越线的动作链都会被当场裁断，适合做分区、截杀、逼位与读秒终结
  - **恒月裁钉**：恒月线的裁定级终结技。并非单纯把目标钉住，而是以一枚高位月钉把目标“继续存在并继续往下打”的资格一起裁掉。命中后可同时钉死目标的站位、动作链、核心回路、再生可能和最后那口执念，适合首领战最终收口与卷末级落幕一击
  - **旧刻回钟**：把一个小范围区域短暂拖回数秒前更稳定的状态，可用于把崩裂地形、断开的防线、濒死同伴和将爆未爆的术式硬拖回“还能接”的那一刻。不是大范围时间回溯，而是极高位的战场抢救与残局改写
- **恒月招牌读感**：恒月线不能只写成“会锁会护”，而要写成“她从过去调取一个已经成立过的结果，让本该崩掉的现在继续站住”。强项是复写过去、保局、钉局、补回旧态
- 核心本质：属于「神秘法则掌控」+「虚妄与生死边界操控」的高阶延伸形态

### 2. 猩红虹月

| 项目       | 内容                                                           |
| -------- | ------------------------------------------------------------ |

| **核心法则** | 未来时间线掌管、改变未来、败线剪定、最高精神/虚实掌控，核心权能是看见未来、删掉过程，让局面直接抵达对自己有利的既定结果 |
| **时间倾向** | 未来                                                           |
| **战斗风格** | 未来改定、过程删除、败线剪定、心智压制、虚实操控                                     |

#### 虹月的实战方向

- **高阶能力（仅女神态以上可使用，伪装态绝对不显露）**：
  - **改变未来权能**：可观测到目标所有可能的未来分支走向，主动否定不利未来，删掉中间过程，让战场直接抵达对自己有利的既定结果
  - **未来否定权能**：被她否定的未来不会稳定到达。敌人本该成功的反击、本该逃脱的路线、本该完成的连招，都会停在"可能发生"之前，永远无法真正落地
  - **根源毁灭权能**：毁灭是虹月高阶外显之一，斩击不再局限于肉身或防御，而是直接斩灭目标存在根源；使用时仍要服务"剪掉未来"的读感，不单独抢走虹月核心定位
  - **精神虚实权能**：心智虹吸是最高阶精神法则的具象化，可完全掌控目标的精神与灵魂；同时可随意操控虚实边界，构造的月相幻境可以做到「在幻境中死亡，现实也会直接灵魂崩解」的真实杀伤
  - **败线剪定**：先看到对手所有可能翻盘的未来枝线，再像剪烛芯一样一根根剪掉；必要时直接删掉通往翻盘结果的中间过程，逼得对手只能沿着她允许的败法往下走。适合打首领战、心理压迫战和高智斗法
  - **红妄换实**：瞬间错置虚实边界，让敌人以为自己斩中了真身、抓住了武器、冲出了包围，结果全是她提前换好的假结果。不是单纯幻术，而是让“假的先成立，真的后到”
- **虹月招牌读感**：虹月线不能只写成“精神压制+斩击”，而要写成“她看见未来，删掉过程，让你直接抵达她允许的败局”。强项是改未来、删过程、断局、剪未来
- 核心本质：属于「精神灵魂干涉」+「幻境造梦」+「神秘法则掌控」的高阶延伸形态

#### 霜月的实战方向

- **高阶能力（仅女神态以上可使用，伪装态绝对不显露）**：
  - **静止世界权能**：霜月掌管现在的上位权能，可以让当前时刻本身停住；极限状态下不是冻住能量，而是让世界停止继续往下一拍推进
  - **现在干涉权能**：可以定格当前时间节点，让局部区域的时间、动作、能力、伤势、崩塌、生命状态和规则推进同时静止；也可以修改目标的当前存在状态
  - **根源冻结权能**：冻结只是静止世界权能的战斗外显之一，可以冻结规则、灵魂、时间流动、空间崩塌、生命变化和抽象概念；被冻结的目标没有专属解限手段永远不会解冻
  - **生死操控权能**：可以随意操控生死边界，能把刚脱离肉身的灵魂直接拉回躯体完成复活，也能把活物的灵魂瞬间剥离打入死亡边界直接致死
  - **霜月静庭**：霜月女神态高光技。以吟唱【世界啊，静止吧】发动，把一片战场判成"当前状态不准继续推进"的白色静庭。重点不是冰封敌人，而是让整个战场进入霜月允许的静止状态
  - **霜潮寂灭**：接在霜月静庭之后使用的收束技。白色霜潮沿着被定格的供能链、反面裂口和核心回路铺开，切断并熄灭反面核心供能，让已经被按停的灾难失去继续燃烧和复起的来源。它不是单纯大范围冰爆，而是"静止之后的断供与熄灭"
  - **临界封生**：一指钉住生死线，把濒死同伴按在“还没死透”的边缘，也能把强敌卡进“最后那口气上不来、最后那一击落不下去”的临界状态
- **霜月招牌读感**：霜月线不能只写成“会冻会控”或“能量静止”，而要写成“她把现在这一刻连同世界一起按住，谁也别想往后走”。能量停滞只是结果之一，真正强项是静止世界、停局、锁命、定格现在
- 核心本质：属于「虚妄与生死边界操控」+「神秘法则掌控」的高阶延伸形态

### 4. 三月合流的核心本质与观感

- 基础配合逻辑：恒月负责"过去复写/旧态回接"，虹月负责"改变未来/删掉过程"，霜月负责"静止世界/定格现在"
- 完全合一本质：当三月权柄完全融合时，她将真正触及过去、现在、未来的完整联动，抵达接近创世裁定层的神性高度
- 合一状态观感：周身笼罩死寂+空灵的神性气场，三色月轮在背后悬空轮转，周围空间会自动浮现生死边界的灰蒙雾气

主角真正可怕的地方，不是她某一条线特别强，而是：
**她能在同一场战斗里，同时干预目标的过去残痕、现在节点与未来走向，从更高位阶决定这一战还能不能继续成立。**

## 三、五阶段权柄接驳进度（如何逐步接过三月权柄）

> **部署执行层**：本节集中管理所有卷次 × 阶段约束。调整剧情进度时只需更新本节，第二节能力定义不需随之改动。

她的"成长"不是变强，而是接过三月女神死后留下的权柄。
她不是从凡人爬上去，也不是旧神回到原位，而是外来继承者把空缺的神座一层层重新立住。
每接过一层权柄，三种出手模式内可用的能力都会变得更丰富，也会让她更接近完整的三月执掌者。

- 三条半展开线全部可用：辉锁镇狱、断空红弧、零度封界
- **第二卷执行口径**：可用不等于全部做高光。第二卷只实写霜月女神态高光；恒月/虹月半展开只作为理论可用、分化迹象、训练误差、档案误判或后续伏笔，不做完整女神态高光
- **三线切换限制**：女神态本身无切换代价，但第二卷仍处于月相旬完全锁定阶段。当前月相旬只能接通对应月线，其他月线即使理论上已经分化，也会被锁定机制拒绝，不能靠强行调用绕过；同一战里不存在主动切换第三条线或因切换导致失控的写法
- 三月视界可以短暂偏向某一条线：恒月视界看钉点，虹月视界看断点，霜月视界看停点

**对创世态的影响**：

- 仍不可用

**核心爽点**：同一套伪装技能开始露出三线分化的边缘，但旁人仍只能理解成极罕见神秘核心或S级适配异常

**写法要点**：三线开始分化但仍受月相锁定，限制来自权限尚未接通与身份暴露风险，不来自身体反噬；观感已明显脱离普通学生水准，但仍能勉强伪装成"极罕见神秘核心"。这一层要写出她开始像真正的怪物级新生

***

### 第三阶段：三月权柄解锁期（权能成型）

**权柄状态**：三条权柄各自拥有稳定的代表性战斗技，开始具备卷末级收场能力。到这一层，她已经能在关键战里打出"你们理解不了，但你们只能认输"的压制感。

**伪装态执行口径**：

- 伪装技能仍只引用“伪装态状态与技能总表”，不新增表外技能名。
- 第三阶段只是这些技能的精度、覆盖范围和误判难度提高；任何存在裁定、未来裁断、虚实边界操控、生死边界操控和三线合击，都不属于伪装态技能。

**对女神态的影响**：

- 三条半展开线更稳定，持续时间和精度明显提升
- **第三至第五卷高光分配**：第三卷只让恒月低功率明面化、虹月不稳定闪现；第四卷推进恒月女神态高光；第五卷推进虹月女神态高光。不要把恒月/虹月女神态高光提前挤到第二卷或第三卷
- 可以在女神态中打出"超出神秘体系理解上限"的效果
- 三条视界也开始稳定：恒月视界能直观看到结构死点，虹月视界能直观看到未来崩口，霜月视界能直观看到生死线和停滞线

**对创世态的影响**：

- 三线代表技成型、各线具备稳定的卷末收场能力后，创世态才达到最低触发条件；第三阶段起首次可用，但仅限于卷末终极大战
- 可以短时同时调用两条线的高位力量，接近三月合一的边缘
- 三月天目第一次短暂睁开，能同时看见两条时间线的碰撞结果，并直接挑掉其中一条

**核心爽点**：伪装态仍像“天才神秘者极限表现”，但读者能看出同一套技能背后已经有更深的三月权柄在压着

**写法要点**：三月轮转爆发越完整越接近暴露身份；她仍不能长时间高位镇压；但最强的地方已经不是"能不能赢"，而是"她决定这一战什么时候结束"

***

### 第四阶段：三月法则链接期（法则链接）

**权柄状态**：三条线开始形成法则联动，可以局部重写战场秩序，战力稳定逼近世界影响级门槛。她已经不只是打赢敌人，而是在重新规定敌人该怎么输。

**伪装态执行口径**：

- 伪装技能仍只引用“伪装态状态与技能总表”，不新增表外技能名。
- 第四阶段可以把同一套伪装技能压到极高熟练度，但大规模改写地形、剪定未来、全域压制和三线合技全部是高位展开，不属于伪装态技能。

**对女神态的影响**：

- 半展开线可以维持更久、精度更高
- 身份暴露风险仍然存在但更可控
- 恒月视界、虹月视界、霜月视界都能完整展开，但同一战里频繁切换会明显提高暴露风险

**对创世态的影响**：

- 可以同时调用三条线的高位力量
- 可以随意修改战场内的过去、现在、未来节点
- 操控区域内所有目标的生死状态
- 三月天目已经基本成型，能同时俯瞰过去、现在、未来三层战场，并直接裁掉其中一层

**核心爽点**：伪装态越来越像“所有常规术式都被她用到尽头”，但真正让战场听命的效果只能放在女神态和创世态

**写法要点**：每次法则深连都会显著提高身份暴露风险；若在错误时机过深展开，可能直接把后续剧情推到不该提前爆开的层面。这一层的高光要写出"整个战场都开始听她的"的压迫感

***

### 第五阶段：完整月权执掌期（完整归一）

**权柄状态**：三月归一，完整执掌三月，以月亮级位格裁定世界终灾。走到这里，她不再是藏在人群里的神，而是能为世界下判词的神。

**伪装态执行口径**：

- 伪装技能仍只引用“伪装态状态与技能总表”，不新增表外技能名。
- 第五阶段的变化不是新增伪装技能，而是她可以主动选择把同一套伪装态收得像普通学生，或放到接近神眼但仍不暴露终局位格的程度。

**对女神态的影响**：

- 女神态可以长时间稳定维持
- 每条线的半展开都可以精准收放
- 恒月视界、虹月视界、霜月视界都能独立长开，并且几乎不会再出现权限失真

**对创世态的影响**：

- 三月归一·月权裁定：进入三月完全合一状态，同时触及过去、现在、未来三层时间结构
- 可以深入生死边界，沟通亡魂、彼岸存在
- 直接裁定世界底层规则中的关键一环：裁定哪些存在还能继续维持、哪些规则需要被否定、哪场终极灾难必须在此刻停下
- 在终局条件凑齐时，拥有改写世界走向的资格，这就是完整月权应有的绝对权限
- 三月天目完全睁开后，她看到的不再只是"敌人怎么动"，而是"哪一条结果该被留下，哪一条结果该被裁掉"

恒月·大创生：以恒月权柄重立存在结构，重构战场、重立屏障、稳住世界裂口
虹月·终焉虹落：对高位存在降下猩红终焉斩，斩的不是表面防御，而是"继续存在并继续往下打"的资格
霜月·永寂白夜：让高层能量流和失控灾变进入近乎绝对静止

**核心爽点**：全世界都收不了场时，只有她还能站出来说"到此为止"；她不是在打怪，而是在决定高层灾难是否还允许继续发生

**写法要点**：终局之后已不再是"维持学生伪装"的叙事阶段；这一阶段不适合在前中期提前透支

***

## 四、成长内在逻辑

主角的能力成长逻辑，必须始终写成：

1. **普通学生伪装期**：压住位格，学会以普通学生身份稳定出手，并持续适应女性身体与少女身份
2. **异常月相适应期**：开始适应三月轮转，三线权柄边缘显形，身体、情绪和战斗节奏都开始受月相影响
3. **三月权柄解锁期**：三线代表技成型，开始真正拥有卷末级收场能力
4. **三月法则链接期**：从会用权能升级到会链接法则、重写秩序
5. **完整月权执掌期**：三月归一，以月亮级位格完成终局裁定

她的成长不是"更努力所以更强"。
**她是在五阶段中越来越能承受自己的本质，也越来越能把三月女神死后空缺的神权接过来。**

三种出手模式始终存在，但只作为五阶段之下的战斗表现层。
她开局就是天花板，只是前中期一直把真正的天花板压着不用。她不出手不是因为需要配合才能赢，而是因为隐藏身份和让队友成长。

***

## 五、写法原则

1. 主角的强一定要区分三种模式：伪装态日常兜底用、女神态濒崩用、创世态终局用，不能一上来就全程满功率
2. 主角的所有能力都必须和三月三线对应，不能额外乱加不相干能力
3. 恒月、虹月、霜月的观感必须明确不同，不能写成三种颜色不同但效果差不多
4. 主角的高光不是"普通战斗也乱杀"，而是队友扛不住时她站出来把战场直接定死——她不需要配合，她就是兜底
5. 她对外始终要保留可伪装空间，除非到了不需要再伪装的剧情阶段
6. 她再高位，也不能写成不分场合乱开权柄、前中期提前掀桌的纯平推机器；但到了创世态与终局篇章，她的无敌必须被完整兑现
7. 伪装态的爽点来自队友苦战她随手收场的反差；女神态的爽点来自全队倒下后她独战终结的震撼；创世态的爽点来自裁定一切的权能
8. 每次接过一层权柄，必须在对应的剧情高点爆发，带来明显推进，不能静悄悄解锁
9. 她可以压着打、留手打、装着打，但一旦认真打，就必须让读者立刻意识到这不是普通天才，而是神在下场
