# 《三月女神今天也只是普通学生》苏弥能力发展设定

---

## 说明
苏弥这条线，核心就一句话：**冷淡精密地追求正确答案，嘴硬毒舌地保护同伴；平时存术式，战时翻页打出去；前期测绘裂缝、封异常入口、拆异核频率，后期处理空洞与反面数据。**

苏弥不是"记录癖"。记录、归档和封页只是她追求真理的手段。她真正不能容忍的是：未经验证的结论被当成事实，真实发生过的事被错误记录覆盖。

她的神秘核心是「归档」，前期不能写成全知规则解析，也不要写成只站后排讲战术。她的战斗主轴是两条：

1. **资料归档与证据处理**：记录任务信息、稳定撤离线、保存权限缺口、判断核心频率、处理实体敌人和术式证据
2. **术式库存火力**：平时抄录、存储、校验低阶术式，战斗时通过封印书翻页激活，打出弱化版的连续术式覆盖

能力归属硬口径：苏弥唯一的神秘核心是「归档」。读取印记、封印书显化、抄录、存页、调页与组合页都是核心技/核心应用；书页中保存的封堵、净化、束缚、防护和攻击结构则是可教学通用术式。她不是拥有大量个人术式，而是用个人核心储存并调用公共技术。准备术式页时必须完成相应构筑与吟唱要求；战斗中翻页属于调用已完成构筑的库存，不等于她能让通用术式天然无吟唱。

## 零、归档核心内核定义

苏弥的神秘核心是「归档」。**凡是真实存在过的，都留有形式印记；凡是有印记的，都可以被读取、保存和调用。** 她所处理的，始终是事物的形式结构本身。

**感知对象**：术式有其构成印记，裂缝有其频率印记，侵蚀有其扩散印记，怪异有其核心能力印记，施术者有其习惯签名。归档感知任何具有形式结构的东西，不限于术式。

**两个方向**：
- **读印记**：她感知事物的形式签名——术式怎么构成、怪异能力如何运作、签名来自哪里。纯粹感知，输出信息。
- **存/调印记**：封印书是归档核心的实体化显化——她把读到的结构固化成页，存进书里，战时调出来以自身神秘力为燃料运行。封印书不是工具，是她的「存档体」。书页不是笔记，是被压入了形式印记的可激活结构。

两个方向是同一核心的两面，不是两套独立能力。

**成长本质**：能读的印记越来越复杂，书的容量越来越大，存档精度越来越高。**给她足够的时间和足够的样本，她能归档的东西没有上限。**

**法则锚点**：归档核心指向的法则是「痕」——任何真实发生过的事都留下形式痕迹。她读痕迹，保存痕迹，用痕迹对抗一切企图抹除痕迹的东西。终局意义：只要发生过就有印记，而她专门读印记。

**性格锚点**：她冷淡、精密、求知欲强，对人情反应慢，但不是冷酷实验者。她可以问出不合时宜的问题，因为她在确认变量；她不会把同伴当样本，因为她追求的真相最终必须服务于保护人。她有毒舌和傲娇，但毒舌只针对错误判断和莽撞行为，关心则藏在提前准备的封页、替人补漏和不提交错误结论里。

**台词锚点**：不要写成机器式"变量/样本"模板。她可以嘴硬，但要有人味。例如："多抄了一页，放着浪费。"、"空白可以以后再补，错了就会害人。"、"别动。你再晃一下，我这页就白写了。"

**怪异核心归档（中后期核心成长轴）**：归档能力最终指向怪异本身——读取怪异核心能力的形式结构，压入书页，以自身神秘力复现。遇到的怪异越多、书里存的越多、能打出的东西越杂越强。这是她无上限成长的根本机制。

**唯一核心边界**：她可以归档通用术式、怪异能力的形式印记，以及他人核心技留下的外显结构，但不能获得第二个神秘核心，也不能完整复制他人的真名解放、界临或立序。凡涉及个人核心的归档，都只能得到受理解度与书页容量限制的弱化外显版本。

**暗线·与幕后者的平行**：幕后者也在做怪异逆向工程——提取怪异结构为己用。苏弥做的是同一件事，目的相反。这条平行线在中后期产生强烈叙事张力：她是目前世界上最有资格读懂幕后者在做什么的人，因为她用同样的方式在读这个世界。

## 二、第一阶段：归档显相期

### 基础信息
| 项目 | 内容 |
|------|------|
| **神秘等级** | 神秘雏形·稳响/临显 → 神秘者·初显 |
| **核心成长节点** | 核心回响 → 核心显相 |
| **战力层级** | 神秘雏形境中后期 → 神秘者境初期 |
| **核心主题** | 存下来，翻出去 |
| **战斗定位** | 资料归档位、核心频率判断位、术式库存火力位 |

**临显—初显边界**：第一卷的封印书显化、存页、调页与连续翻页均可作为核心技雏形出现；临显阶段依赖封印书媒介、预构筑书页与有限库存局部显出核形，神秘者·初显后才可跨环境稳定维持完整归档显相。招式已有名字不代表她已提前晋升。

### 核心技与核心应用

| 能力 | 描述 | 效果 | 限制 |
|------|------|------|------|
| **封印书** | 将已学会、已抄录、已校验的低阶术式存成可激活的术式页 | 战斗时翻页激活，省去完整构筑时间 | 库存有限，用完就要重新准备 |
| **术式抄录** | 记录基础封堵术、净化术、束缚术、低阶攻击术的稳定结构 | 平时积累库存，战斗前按任务选择带哪些页 | 只能抄录低阶、结构完整、公开或允许学习的术式 |
| **撤离与防护** | 针对队伍撤离窗口、实体敌人压迫和防护缺口释放防护/净化术式页 | 稳住队伍节奏，减轻进入反面后的侵蚀压力 | 对核心区和高位规则无力 |
| **连续翻页** | 短时间连续激活数页术式 | 形成弱化版术式覆盖，打断、压制、清理低阶目标 | 单页威力低于原版，连续使用消耗大，容易空页 |
| **核心归档（萌芽·伏笔）** | 封印书对准怪异残留或能量痕迹，尝试读取其核心能力形式印记 | 第一卷：书页发光但归档失败，只留下一个空白页和一丝残余印记；此动作在第一卷出现一次作为伏笔，不展开，不可用 | 第一卷完全不可用；需神秘者境中上位以上才能稳定读取；第一卷只作暗线埋设 |

### 阶段限制
- 第一卷不能直接解析反面底层规则、主角位格或高位月相体系
- 封印书里的术式都是弱化版，强在快和连续，不强在单发杀伤
- 必须提前准备库存；遇到没准备过的场景，她不能凭空变出正确术式
- 主要处理裂缝、侵蚀、低阶怪异和术式残留，不负责正面硬杀高阶敌人

## 三、第二阶段：归档解放期

### 基础信息
| 项目 | 内容 |
|------|------|
| **神秘等级** | 神秘者·定相/深显 → 深律者·稳解 |
| **核心成长节点** | 核心深显 → 真名解放稳定 |
| **战力层级** | 神秘者境中上位 → 区域霸主级下段 |
| **核心主题** | 存得更多，打得更密 |
| **战斗定位** | 术式库存炮台、封堵控场位、残痕追踪位 |

### 核心技与核心应用

| 能力 | 描述 | 效果 | 限制 |
|------|------|------|------|
| **扩容封印书** | 术式页容量和稳定性提升 | 能带更多封堵、净化、束缚、攻击页进入战场 | 仍需提前准备，不能无限连发 |
| **术式连页** | 将数页同类型术式按顺序激活 | 形成火力线、封锁带、净化带或短时压制区 | 单页威力低于原术式的完整全吟唱版本 |
| **记录校验** | 读取任务记录、术式印、权限缺口和核心频率变化 | 判断是否有人为修改、重复签名、异常接入 | 只能得出“异常/疑似/同源痕迹”，不能直接确认底层真相 |
| **封堵强化** | 对同一空洞入口、裂口或核心节点连续覆盖多页封堵术 | 把低中阶空洞入口压回去，或短时稳定核心频率，给林见星和白砚河争取推进空间 | 高位空洞核心需要主角或高位者压场 |
| **核心归档（初步）** | 观察或接触中阶以下怪异，封印书读取其核心能力的形式印记，压入专属页 | 归档成功后可调用该能力弱化版，以自身神秘力为燃料运行；开始出现战场上「这能力哪来的」时刻；两种已归档怪异能力可叠页组合释放 | 归档版保真度打折，永远弱于原版；归档需要观察时间，不能战场即读；无法归档超出自身理解度的结构；书页容量消耗增加 |

### 阶段限制
- 术式库存越多，战前选择越重要；带错页会直接影响战斗表现
- 能追踪术式残痕，但不能把“残痕异常”写成“完整规则解析”
- 面对纯肉身、纯蛮力和高位位格压制，仍需要白砚河、林见星和主角接住

## 四、第三阶段：万象界临期

### 基础信息
| 项目 | 内容 |
|------|------|
| **神秘等级** | 深律者·深解 → 域主 |
| **核心成长节点** | 真名深解 → 界临 |
| **战力层级** | 区域霸主级中段 → 区域霸主级上段 |
| **核心主题** | 归档反面，覆盖战场 |
| **战斗定位** | 反面数据分析位、复合术式覆盖位、封堵中枢 |

### 核心技与核心应用

| 能力 | 描述 | 效果 | 限制 |
|------|------|------|------|
| **反面数据归档** | 记录反面浮现频率、裂缝位置、侵蚀扩散和术式签名 | 拼出幕后者长期动作，判断反面运行异常 | 依赖大量样本和时间，不能现场一眼看穿 |
| **复合术式覆盖** | 多类术式页组合释放 | 同时封堵、净化、束缚、轰击，形成战场级覆盖 | 消耗巨大，准备时间长 |
| **签名追索** | 从灰白术式印和裂缝残痕中追踪同一施术者习惯 | 帮队伍找到幕后线索和下一处危机 | 只能追踪痕迹，不能直接确认身份 |
| **大封页** | 将一片区域的裂缝处理结果固定下来 | 暂时阻止反面继续从该处渗出 | 对权柄级反面结构只能延缓，不能根治 |
| **核心归档（深化）** | 读取高阶怪异核心能力的主要结构特征，归档部分完整印记；开始尝试归档幕后者灰白术式印的形式签名 | 书里已有的怪异能力页可以自由组合打出复合效果；开始具备「让敌人用自己的能力来对付自己」的战术可能；幕后者签名积累到一定数量后可以逆向推算其行动规律 | 高阶结构仍只能部分归档，完整复现需更高等级；强行归档超出理解上限的结构会引发封印书局部崩坏；归档幕后者签名会触发反追踪风险 |
| **界临·归档境·万象入卷** | “归档”完成界临，封印书散开，所有书页以她为圆心悬浮于归档境，每一页面朝上、即时可见；不需要查找，不需要翻动——任何一页在意及的瞬间即可激活 | 这是她适应性的绝对上限：归档境内所有已归档能力和术式无准备时间，即时可调；任意两页可以即时组合释放，不需要战前规划；归档境内发生的任何新能力会被即时归档进新页，同一场战斗里读到的东西立刻可以调用；对于其中出现的任何新情况，她有档案，或者能用现有档案拼出一个答案——书越满，她能响应的情况越多，越没有她接不住的局面 | 第四卷解锁；界临强度=书的丰富程度，书里没有的东西归档境也变不出来；即时归档要求新能力在她的理解范围内，超出理解上限只能收到碎片；大量同时调用消耗神秘力极快；不能借此获得第二核心、完整真名解放、他人界临或立序；台词口径不是“全部展开”，而是“已收录”“我有记录”“这页，这页，还有这页”——胸有成竹，不是轰炸 |

### 阶段限制
- 反面数据归档依赖大量样本，不能靠一次现场观察下定论
- 面对完全不走通用术式体系、只靠纯高位肉身、核心技和绝对位格压人的敌人，需主角和白砚河承担主杀伤
- 强行归档超出承受上限的高位术式可能导致封印书崩坏
- 强行追索权柄级规则残痕会引发反噬，只能保留碎片，不能完整读取

## 五、成长内在逻辑
苏弥的成长，不是单纯"看得更远"或者"会的东西更多"，而是一步步把这件事做绝：
**先把术式存进书里，再开始归档怪异能力打成火力，最后用积累的所有印记读懂反面和幕后者的全貌。**

| 阶段 | 核心能力 | 特点 |
|------|----------|------|
| **第一阶段** | 存术式，翻页火力，封入口，核心归档萌芽失败（伏笔） | 前期战场价值明确；怪异核心归档轴尚未开启 |
| **第二阶段** | 扩库存，连页覆盖，核心归档初步成功 | 书里开始混入怪异能力页；「这能力哪来的」开始出现 |
| **第三阶段** | 读反面数据，追术式签名，核心归档深化，组合释放 | 书里怪异能力越来越杂；开始逼近幕后者行动规律 |
| **终局** | 封印书是整个故事里所有怪异和幕后者行动的完整档案 | 她不是越来越强的战士，是越来越像一本谁也不知道里面写了什么的书 |

## 六、适配怪异类型

| 阶段 | 擅长类型 |
|------|----------|
| **第一阶段** | 实体敌人、防护缺口、核心频率异常、需要净化或压制的目标 |
| **第二阶段** | 多裂缝战场、群体低中阶怪异、术式印干扰区、需要火力覆盖和封堵控场的敌人 |
| **第三阶段** | 反面运行异常、灰白术式印链条、规则化术式怪异、界临型侵蚀区 |

## 七、封页调用招式体系

苏弥的战斗体系统一称为 **封页调用体系**。写法重点不是“她分析出了什么”，而是“她翻到哪一页，哪一页落在战场上产生什么效果”。“封页”是归档核心的调用方式，页内承载的才是通用术式；封页招式名不等于另造一套个人术式。

视觉关键词：封印书、书签、银灰符文、页角燃烧、符页飞散、笔尖划线、归档完成、驳回、封存、复写。

### 基础封页

| 招式 | 画面 | 用途 | 限制 |
|------|------|------|------|
| **封页·止裂** | 银灰符页贴到裂缝边缘，像书页压住纸缝一样把灰白裂口压停 | 封小型裂缝，阻止低阶怪异继续渗出 | 对中高位裂缝只能延缓 |
| **封页·净尘** | 书页碎成细小光屑，沿地面和墙面铺开，把浅层侵蚀残留擦掉一层 | 清理浅层侵蚀残留，恢复队友落脚点 | 不能净化深层侵蚀 |
| **封页·缚章** | 由封印书调用预存的【禁式·十四，缚章。】，数枚书签状符页钉进怪异四肢、影子或外壳缝隙 | 束缚低阶怪异，给白砚河制造输出窗口 | 目标力量过强会撕页；束缚来自公共术式，翻页定位与连续调用来自“归档”核心 |
| **封页·短炮** | 攻击页亮起，数道低阶术式炮线从书页射出 | 远程火力，打断怪异靠近 | 单发威力低，靠连续性压制 |

### 连页与书阵

| 招式 | 画面 | 用途 | 限制 |
|------|------|------|------|
| **连页·三式并放** | 止裂、净尘、短炮三页连续翻开，裂缝被压住，侵蚀残留被清开，怪异被炮线逼退 | 小范围战场整理 | 消耗三页库存，不能频繁用 |
| **封页·落签阵** | 数十枚书签状符页飞出，钉在地面、墙壁和裂缝边缘，连成封锁区 | 区域封锁，限制怪异移动 | 需要提前准备足够符页 |
| **归档·复写火线** | 几道低阶攻击术式被复写到同一页，一口气打成密集火线 | 弱化版火力覆盖，压制群体低阶怪异 | 威力不如真正高阶炮击 |
| **归档·驳回再生** | 怪异准备恢复时，书页压住伤口或侵蚀核，恢复纹路被强行退回 | 限制低阶怪异或侵蚀体再生 | 第一卷只对低阶目标有效 |

### 高光展开

| 招式 | 画面 | 用途 | 限制 |
|------|------|------|------|
| **封页展开·百签锁缝** | 封印书抛起，数百枚符签像雨一样钉向空洞入口、裂口和核心节点，灰白裂口被一层层压回去 | 大范围封入口、稳撤离线、守楼、正式穿行战高光 | 消耗大，容易空书 |
| **归档书阵·万页齐鸣** | 大量提前准备的术式页同时亮起，净化、束缚、短炮、止裂一轮轮落下 | 中后期主力术式覆盖 | 需要长期准备，不能临场凭空开 |
| **终式·封存此页** | 她把整场异常的当前状态压进封印书一页，页角燃烧后合上 | 关键收束，封住短暂裂缝状态或低阶侵蚀 | 前期不能封高位怪异或反面底层规则 |

### 台词口径
- 苏弥的燃点是“我已经准备好了”，不是“我已经看懂了”。
- 常用短句可以是：“翻页。”“归档完成。”“驳回。”“这一页，到此为止。”
- 台词要短，像操作确认，不要写成大段讲解。

## 八、写法原则
1. **爽点放在三件事上**：翻页连发、封缝止潮、术式库存覆盖，别写成只会站后排讲战术
2. **前期不能过强**：第一卷只处理任务记录、权限缺口、低阶术式证据和公开基础术式，不能直接解析反面底层规则
3. **她必须能打**：每次参战都要有至少一次术式页落地的可视化效果，比如封堵、净化、束缚、短炮击
4. **强势感来自准备**：别人临场构筑，她已经把术式页带进战场；翻页快、落点准、连续性强
5. **她不抢主角的位置**：她负责把反面渗出压住、把小裂缝封掉、把幕后痕迹留住；真正高位压制仍由主角承担
6. **边界必须一直在**：库存有限、威力打折、需要提前准备，带错页或用空页都会让她陷入危险
