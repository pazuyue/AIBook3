# 《三月女神今天也只是普通学生》白砚河能力发展设定

---

## 说明
白砚河这条线没那么绕，说白了就是：**先撕开一个口子，再把这个口子一路撕到整片战场。**

他的神秘核心是「裂锋」，本质就是近战破防型。前期靠贴脸硬拆，中期开始能一边砍一边封路，后期再把整片场子都变成“你动一下就要挨切”的范围。

> ⛔ **正文执行红线**：裂锋的本质是“找到本来就会断的地方然后断它”，不是蛮力砸开。正文写他出手时，写“他知道往哪切”，不写“他用力砍”。以下机制描述仅供作者理解，禁止逐字转入正文。

本文件按《角色能力发展设定模板》执行。等级按跨卷阶段双轴记录，当前卷可用能力以当前硬口径和当前写作状态为准。

---

## 零、裂锋内核定义

白砚河的神秘核心是「裂锋」。**万物皆有其固有断裂线——不是被砸出来的，是本来就在那里的。裂锋所做的，是感知那条线，然后顺着它走。** 他从来不是蛮力砸开，是精准找到本来就会断的地方，然后断它。

**妖血与裂锋的关系**：妖血不是独立附加，也不是越强越容易失控的危险源，而是另一套尚未被他完整接通的力量回路。他血脉里的妖血本就带裂属性；当妖血、裂锋、肉身和本人意志方向一致时，力量会更热、更快、更猛，也更稳定。第三阶段觉醒是四者彻底同步，不是突然出现的新能力。

**妖血本能意志**：妖血中存在由血脉长期战斗与生存沉淀出的本能意志。它不是邪恶诅咒，但拥有强烈的战斗判断和生存倾向。白砚河前期只能强行借用它的力量；一旦强开过深，本人意识会被反压，妖血本能夺取行动主导权，进入类似野兽接管身体的暴走状态。中期他开始在内在冲突中听懂并回应这股意志；后期双方对“该撕开什么、该守住什么”达成一致，完成真正协同。

**妖化稳定性硬口径**：
- 完整妖化不会因为力量增强而失控；越成熟，控制越稳定
- 半吊子妖化会真正暴走：白砚河强行唤醒妖血深层力量，却尚未得到妖血本能意志认可，妖血本能会压过本人意识并接管行动
- 强行提升半吊子妖化程度会让妖血本能取得更多主导权，但问题来自双方关系与妖化阶段未完成，不是妖血力量越强越邪恶
- 白砚河的成长方向不是压制或驯服妖血，而是理解、承认并与妖血完成协同

**跨卷妖血关系阶梯**：
- 第一卷·争夺：白砚河把妖血当作必须听令的力量，只能在安全阈值内借力；强开半吊子妖化会与妖血本能争夺身体主导权并暴走
- 第二至三卷·理解：他开始在内在冲突中识别妖血的本能意图，逐步获得认可，并用裂锋给共同力量明确方向
- 第四卷以后·协同：妖血响应他的选择，他也接受妖血的战斗本能；双方共同完成稳定完整妖化

**三层成长本质**：
- **找裂**：感知固有断裂线，顺着它激活。每一刀落在裂线上，不是蛮力，是效率。（第一阶段）
- **激活裂**：主动扩大断裂，铺设切线，让战场开始变成他的刀。（第二阶段）
- **造裂**：没有裂线，给它造一条。妖血与裂锋咬合后，他不再依赖结构弱点——他就是断裂线本身。（第三阶段觉醒）

**领域逻辑**：裂锋领域的核心不是"他在主动出击"，而是"把裂线铺满空间，你动就撞上去"。被动收割，不是主动砍人。你站哪都有裂线，动一步都在帮他切自己。

**法则锚点**：裂锋核心指向的法则是「断」——万物皆有固有断裂线，凡有结构，必有可断之处。终局意义：他能找到任何层级结构的断裂点，包括规则、封印，以及他人为之死守的东西。

**叙事暗线·断与不断**：裂锋能找到任何结构的断裂线——包括团队、包括信任、包括他自己。他用整本书学的是：找到裂线后，选择撕开哪一条、保住哪一条。第一卷先落地为"真正的完胜"：敌人要打倒，人也要全部带回来；只顾追击而让身后出现伤亡，根本不算赢。

**与林见星的隐性对应**：
- 林见星读织——她看见结构怎么连在一起
- 白砚河找裂——他找到结构哪里最容易断

两人操作同一套世界结构的相反方向，她缝，他断。战场上是天然搭档：他破开，她补住；她读到弱点位置，他去砍。

## 二、第一阶段：强攻基础期

### 基础信息
| 项目 | 内容 |
|------|------|
| **神秘等级** | 神秘雏形·待定型 → 神秘者·初阶 |
| **战力层级** | 神秘雏形境上位 → 神秘者境初期 |
| **核心主题** | 破甲，切割 |
| **战斗定位** | 单体破防主攻手 |

### 核心能力

| 能力 | 描述 | 效果 | 限制 |
|------|------|------|------|
| **裂锋附着** | 将裂解力附着在拳、脚、臂刃或临时武器表面 | 命中后先撕开极细裂口，后续攻击更容易切入 | 需贴身攻击 |
| **破甲拳** | 基础近战重击 | 第一拳裂口→第二拳扩大→第三拳崩开整块防御 | 仅对实体目标有效 |
| **切锋步** | 依靠妖血强化爆发力进行短距离突进 | 快速贴近远程敌人，适合追击、换位、补击 | 非瞬移，需短距离助跑 |
| **裂痕追斩** | 对已留裂口的目标追加高速斜斩 | 沿裂纹切入更深，保证单体硬目标持续杀伤 | 需先造成裂口 |

### 阶段限制
- 近战依赖极高，必须贴身
- 无成型范围技，清群能力弱
- 对高速位移、重度幻术干扰、缺少明确受力点的敌人容易吃亏
- 妖血爆发时间短，连续爆发后体力掉得快

## 三、第二阶段：强攻拓展期

### 基础信息
| 项目 | 内容 |
|------|------|
| **神秘等级** | 神秘者·中阶/深阶 → 深律者·中阶 |
| **战力层级** | 神秘者境中上位 → 区域霸主级下段 |
| **核心主题** | 解锁范围切割与攻击型控场 |
| **战斗定位** | 核心打手、强杀位、清场位、前线控场位 |

### 核心能力

| 能力 | 描述 | 效果 | 限制 |
|------|------|------|------|
| **裂锋斩轮** | 压缩裂锋成高速旋转的圆弧斩盘甩出 | 极薄极快的远程切割，切穿目标直至更高强度防御 | 消耗高于贴身攻击 |
| **裂界横扫** | 挥臂/挥刃斩出横向大范围裂解波 | 同时切开多个目标、障碍物和低阶护盾 | 需蓄力 |
| **裂锋封步** | 将裂解力打入地面、墙体或空气特定位置 | 敌人踏入时被切线逼停、逼退或转向 | 非定身，靠杀伤威慑封路 |
| **妖血连爆** | 妖血与裂锋短时高速同步，进入高热爆发状态 | 短时间内连续高强度切割与突进 | 同步尚不成熟，结束后肌肉、关节和精神负荷明显 |
| **断首线** | 提前在敌人四周铺设多道裂锋轨迹 | 破绽出现时切线同时收束，形成斩首切割 | 需预判敌人行动 |

### 阶段限制
- 远程切割消耗高于第一阶段贴身攻击
- 攻击型控场本质是「用杀伤封路」，面对完全无视切割的高规则目标会吃亏
- 妖血连爆不能长开；当前阶段持续过久会逼近半吊子妖化阈值，引发妖血本能反压与暴走风险

## 四、第三阶段：形态觉醒期

### 基础信息
| 项目 | 内容 |
|------|------|
| **神秘等级** | 深律者·深阶 → 域主 |
| **战力层级** | 区域霸主级中段 → 区域霸主级上段 |
| **核心主题** | 觉醒领域级碾压与终极斩击 |
| **战斗定位** | 领域主宰、正面碾压位、单体斩首位、范围屠场位 |

### 核心能力

| 能力 | 描述 | 效果 | 限制 |
|------|------|------|------|
| **裂锋形态觉醒** | 妖血、裂锋、肉身与本人意志彻底同步，进入完整妖化战斗形态 | 头发、瞳色、血纹、气息异变；力量、速度、反应、抗打性、斩击力道全面跃升，控制精度同步提升 | 形态持续消耗精神力 |
| **裂锋领域·无间断界** | 展开专属高阶领域，将裂线铺满整片空间 | 被动收割——不是他主动出击，是裂线铺满空间，敌人移动就撞上切线；你站哪都有裂线，动一步都在帮他切自己；压迫感来自被动，不来自主动 | 领域范围有限；对能感知并规避切线位置的高位目标效果减弱 |
| **断界天斩·造裂** | 妖血与裂锋完全咬合后的质变斩击——不再需要目标有弱点，强行在任何结构上造出断裂线，然后沿那条线一切到底 | 切开防御、肉身、护盾、结界；面对没有固有裂线的高规则目标也能出手 | 消耗极高，再出间隔长；是「找裂」到「造裂」质变后才解锁的形态 |
| **千锋覆杀** | 领域范围内同时引爆多段交错切线 | 整片空间无死角切割，适合清群、破军、斩断援军 | 非单点最强，侧重范围 |
| **裂锋镇场** | 持续释放高压裂解威势 | 压缩敌人动作幅度、位移路线和出手窗口 | 对高等级敌人效果减弱 |

### 阶段限制
- 领域与终极斩击消耗极高，不能连续长开
- 对高规则型、行动模式极难锁定的目标，需苏弥与主角先拆条件、破规则
- 完整妖化本身稳定可控；高强度持续作战的限制来自消耗、伤势和精神负荷，不来自力量越强越失控
- 连续高强度开领域会导致身体崩坏

## 五、成长内在逻辑
白砚河的成长一直只做一件事：**把"裂"这件事越做越深。**

从找到裂线，到激活裂线，再到自己就是裂线。

| 阶段 | 核心概念 | 能力质感 |
|------|----------|----------|
| **第一阶段** | 找裂·顺线断 | 精准感——每一刀落在裂线上，不是蛮力，是效率 |
| **第二阶段** | 激活裂·铺线封场 | 压迫感——切线铺开，你动你挨；战场开始变成他的刀 |
| **第三阶段** | 造裂·我即断裂线 | 碾压感——没有弱点他就给你造一条；领域被动收割，动一步切一步 |

## 七、写法原则
1. **爽点就四个字**：硬、快、狠、正。别把他写成玩花活的人
2. **招式一定要直给**：他出手就是撕、砍、撞、压，不需要拐弯，越直接越有压迫感
3. **中期以后要扛主攻**：他得越来越像队里最能硬吃正面的那把刀
4. **领域要有“到处都在切”的感觉**：不是单纯范围大，而是你站哪都不安全，动一步都可能被收掉
5. **他不替代主角**：他负责把最硬的正面撕开，主角负责把最后的上限和收口定死
6. **精准感比蛮力感更恐怖**：不写"他用力砍"，写"他知道往哪切"；每一刀落点都是裂线，读者要感受到他不是在乱砸，是在选位置
7. **领域是被动的**：无间断界不写他主动斩击，写"你动你死"；压迫感来自裂线铺满空间，不来自他发多少次招
8. **暗线·断与不断**：第一卷写他停手比出手更难；他会的是找断裂线，他在学的是找到了不一定要撕开

## 八、第一卷变强理念与影子训练场心理动机

白砚河是 **A+ 适配度**（4 份文件确认：世界观设定 / 当前写作状态 / 章节目录版节奏表 / 钩子统计）——顶尖梯队，**仅次于 S**。

Ch8 受控反面实训时他亲眼看到：她一掌拍碎别人三分钟的怪异。

她**没**用全力——这是他能看出来的。更重要的是，她出手之后敌人倒下、队友安全、现场没有留下需要别人替她收拾的缺口。

白砚河认定最强必须完胜。前期的他把完胜理解成：正面必须由自己打穿，第一必须由自己拿下，任何人替他补位都说明他赢得不够完整。

他的现实瓶颈不是妖血越强越容易失控，而是尚未获得妖血本能意志认可。他只能在安全阈值内借力；一旦强开半吊子妖化，妖血本能就会反压意识并夺取身体主导权。他想要完整妖化，因为那意味着他终于能在最强状态下保持自我，并完成自己认定的完胜。

陆沉对他的精准话术不是泛泛的"让你变强"，而是：

> "你离完整妖化只差最后一步。学院让你慢慢等，是怕你出事；但等下去，你永远只能让别人替你收尾。"

影子训练场第一次训练确实让白砚河获得了接近完整妖化的错觉：妖血响应更快，裂锋输出更高，而且强开后没有立刻被妖血本能夺走主导权。因此他主动选择相信。

**第一卷角色弧**：
- Ch8：见识哥伦比娅的完胜，认定自己必须追上并正面超越她
- Ch9-10：陆沉先给出合法有效的同步建议，再用"提前完成妖化"击中他的瓶颈；他主动进入影子训练场
- Ch11：短期假峰值压住暴走征兆，让他误以为自己已经接近完整妖化
- Ch12-13：看见许临与自身后果，承认影子训练场给的不是妖血认可，只是用侵蚀暂时压住本人意识与暴走征兆
