# 曜月七星（学院顶级序列）人物设定

---

## 说明

本文件定义曜月七星七名序列学生的核心设定。每人具备独立的神秘核心方向、个人标签、校园存在感和与主角的潜在关系线。

设计原则：
1. 对标《某科学的超电磁炮》学园都市七名Level 5——每人有明确编号、独特能力、鲜明性格和独立叙事价值
2. 七人能力路线互不重叠，覆盖不同战斗维度
3. 主角最终路径是成为新的No.1，七星是她必须逐步面对的阶梯
4. 第一卷不展开七星个人剧情，仅作为背景压力；第二卷起逐步登场
5. 每名七星只拥有一个神秘核心，个人招式与个人战场结构均属于核心技/核心应用；术式只指可教学的通用能力
6. 七星的个人成长同样服从“核心回响 → 核心显相 → 真名解放 → 界临 → 立序”。核心技喊名与解放语都不是吟唱

---

## No.1 ——裴泽渊

| 项目 | 内容 |
|------|------|
| **年级** | 四年级 |
| **神秘核心** | `绝域` |
| **战力层级** | 神秘者境巅峰，接近深律者门槛 |
| **个人标签** | 冷峻、寡言、绝对理性 |
| **校园存在感** | 极低调——几乎不参加公开活动，但所有人都知道他是No.1 |

### 能力特征
- 能力方向：高阶干涉核心场。绝域投射是“绝域”的核心应用，不是域主级界临，也不是通用术式；激活时地面和空气中会浮现淡银色几何线框，他可以对少量、明确、已理解的物理过程或术式过程施加限制，例如让某片火焰难以继续燃烧、让一段动能传导被削弱、让特定术式结构难以成型
- 战斗风格：不主动进攻。开场构筑绝域投射后，以一至两条预设限制打乱对手最依赖的能力流程，再以最简洁的一击收场
- 标志性表现：对手会看到自己的术式进入那片几何线框后迅速失稳——火焰熄弱、冲击偏散、术式结构崩开。整个战场变得异常“安静”——所有混乱都被他提前理解并压进了有限条件
- 视觉印象：淡银色的半透明线框覆盖战场，像一张平静的几何网格落在现实上方；被消解的术式会化为银色微粒散落
- 能力边界：每次只能稳定维持一至两条预设限制；限制必须基于他已经理解并完成构筑的过程；对陌生核心技、高于自身理解层级的规则、怪异核心能力和三月权柄只能干扰，不能强制生效；绝域投射结构被破坏后需要重新构筑

### 个人动机
不追求变强——他已经是No.1，没有人能给他压力。他在意的是“我的绝域有一天能否成为真实世界的小型副本？”他对神秘体系的底层规则有近乎执念的探究欲——是什么决定了“火必须燃烧”这种基础规则？他现在只能以绝域投射限制已经理解的过程，却不知道它们为什么存在。他隐约感觉到现有神秘体系之上还有更高层的东西——某种不需要铺设核心场就能直接改变规则的力量。

### 与主角关系线
- **定位**：最终Boss式的标杆——主角要取代的No.1
- **关系走向**：第二卷远距离观察主角，第三卷发现“这个新生的冰霜核心技进入我的绝域后没有被消解”的困惑，第四-五卷正面交锋
- **核心冲突**：他的“绝域”是学生层对规则干涉的极致，而主角的三月权柄是规则本身的源头——他只能限制已经理解的过程，主角却能“定义规则是什么”。他是第一个真正感觉到“有人站在我头顶”的序列成员

## No.2 ——沈映霜

| 项目 | 内容 |
|------|------|
| **年级** | 四年级 |
| **神秘核心** | `镜渊` |
| **战力层级** | 神秘者境上位 |
| **个人标签** | 优雅、冷淡、完美主义、隐含危险感 |
| **校园存在感** | 高——学院公认的"最美的序列成员"，但所有靠近她的人都觉得隔着一面镜子 |

### 能力特征
- 能力方向：镜像复制与反射。她可以完美复制已解析的通用术式并以更精准的方式反弹回去；面对他人的核心技，只能复现可观察到的外形、能量路径和物理结果，无法复制其唯一核心法则
- 战斗风格：永远后手，等对手出招后用更完美的版本打回去
- 标志性表现：对手会觉得在和一个"更好的自己"作战

### 个人动机
追求"完美的复现"——她认为世界上没有不可理解、不可拆解和不可再现的能力。她对通用术式的复制近乎完美，也因此错误地相信核心技迟早同样能被完整复现。当她遇到只能复制外形、无法复制核心本质的东西时，会产生存在性的动摇。

### 与主角关系线
- **定位**：镜像对照——能力上与主角形成"复制 vs 原本"的对立
- **关系走向**：第三卷试图复制主角对外展示的冰霜核心技，发现连“外形可复现、底层不可成立”的常规边界都被打破，产生强烈兴趣
- **核心冲突**：三月权柄既不是通用术式，也不是常规神秘核心，无法被镜渊完整复制——她的能力边界在主角面前第一次被打破

## No.3 ——岑远行

| 项目 | 内容 |
|------|------|
| **年级** | 三年级 |
| **神秘核心** | `烈行` |
| **战力层级** | 神秘者境上位 |
| **个人标签** | 热血、直率、好战、重义气 |
| **校园存在感** | 最高——七星中最活跃的成员，经常在训练场和对抗赛中出现，学生论坛的常驻话题 |

### 能力特征
- 能力方向：速度与冲击力的极致，擅长高速突进和连续打击
- 战斗风格：纯正的正面强攻型，不用计谋，靠速度和爆发力碾压一切
- 标志性表现：他的移动会在空气中留下灼热的残像，全力冲刺时地面会被冲击波碾出裂纹

### 个人动机
纯粹追求更强。他不关心排名本身，只关心"下一个能让我全力出手的对手在哪里"。他是七星中最纯粹的战斗型人格。

### 与主角关系线
- **定位**：友好型对手——白砚河的高年级映射
- **关系走向**：第二卷注意到主角团中白砚河的战斗风格，产生兴趣；第三卷可能成为主角团的意外盟友
- **核心互动**：他和白砚河之间的"前辈后辈"关系是重要副线——白砚河想超越的不只是七星，更具体地说是想超越No.3

## No.4 ——柳笙寒

| 项目 | 内容 |
|------|------|
| **年级** | 三年级 |
| **神秘核心** | `寂声` |
| **战力层级** | 神秘者境中上位 |
| **个人标签** | 安静、阴郁、独来独往、不近人情 |
| **校园存在感** | 极低——大部分学生不知道她长什么样，只知道No.4存在 |

### 能力特征
- 能力方向：声波与精神干扰，可以用无声的频率波动影响对手的感知、判断和情绪
- 战斗风格：从不正面交锋，在对手意识到之前就已经被干扰了精神状态
- 标志性表现：和她对战的人事后只记得"突然什么都听不见了"

### 个人动机
她不喜欢人群，不喜欢被关注，进入序列是因为实力够了而不是她想要这个位置。她在意的是"安静"——但她的能力恰恰是通过声波影响别人，这个矛盾让她对自己的力量有复杂情感。

### 与主角关系线
- **定位**：暗线型角色——与主角的"三无脸"形成微妙共鸣
- **关系走向**：第三卷可能因为某个事件与主角产生交集，发现主角是少数不被她的精神干扰影响的人
- **核心互动**：两个都不太合群的人之间的无言默契

## No.5 ——谢峥

| 项目 | 内容 |
|------|------|
| **年级** | 四年级 |
| **神秘核心** | `铸锁` |
| **战力层级** | 神秘者境中上位 |
| **个人标签** | 稳重、务实、制度拥护者、有领导气质 |
| **校园存在感** | 中高——穿行管理委员会核心成员，经常出现在学院公务场合 |

### 能力特征
- 能力方向：封锁与锁定，可以制造物理性和规则性的封锁结构，锁定区域、锁定目标、锁定能力
- 战斗风格：控场型，不追求击倒对手，而是把对手完全锁住
- 标志性表现：他的封锁一旦成型，除非从更高规则层面打破，否则无法从内部解开

### 个人动机
信奉秩序和制度。他认为学院的评级体系、序列制度和穿行管理规则是保护所有人的基石。他的恐惧是"秩序崩溃"——如果规则失效，力量没有约束，就会有人受伤。

### 与主角关系线
- **定位**：制度派对手——与叶澜的立场互补
- **关系走向**：第二卷因主角团的"越级行动"与他们产生摩擦，第三卷逐步认可主角的实力和判断
- **核心冲突**：主角的三月权柄本质上是比任何制度都更高层的"锁"——他的铸锁在主角面前如同纸糊

## No.6 ——姜吟渡

| 项目 | 内容 |
|------|------|
| **年级** | 三年级 |
| **神秘核心** | `渡引` |
| **战力层级** | 神秘者境中位 |
| **个人标签** | 温和、随性、老好人外表下藏着深沉的洞察力 |
| **校园存在感** | 中——最平易近人的序列成员，食堂里碰到会和低年级学生聊天 |

### 能力特征
- 能力方向：能量引导与转化，可以将一种形态的神秘力转化为另一种形态，也可以引导他人的能力流向
- 战斗风格：辅助和转化型，擅长在团战中改变能量流向，把对手的攻击转化为己方的补给
- 标志性表现：队友和他组队时会觉得自己的通用术式更顺、核心输出更稳了，但说不清为什么

### 个人动机
他真正关心的不是战斗力排名，而是"以太的本质是什么"。他是七星中最接近研究型的人格，对以太体系的底层机制有自己的理论。他的温和不是伪装——他真的对人没有恶意，但他的洞察力让他总能看到别人不想被看到的东西。

### 与主角关系线
- **定位**：潜在盟友与信息源——最可能提前感知到主角"不对劲"的序列成员
- **关系走向**：第二卷后段可能因为感知到主角身上的能量特征异常而产生好奇，但不会追查
- **核心互动**：他的"渡引"能力接触到主角时可能会产生异常反馈，成为主角身份危机的潜在触发点

## No.7 ——秦落微

| 项目 | 内容 |
|------|------|
| **年级** | 二年级 |
| **神秘核心** | `蚀刻` |
| **战力层级** | 神秘者境初期（但成长速度极快） |
| **个人标签** | 天才、孤傲、不稳定、对规则有本能的排斥感 |
| **校园存在感** | 争议性极高——二年级就进入序列，是历史上最年轻的序列成员之一 |

### 能力特征
- 能力方向：物质与规则的侵蚀与刻印，可以在接触目标后蚀刻自己的能力印记，改变目标的物理特性或弱化目标的防御
- 战斗风格：近距离接触型，需要碰到才能发动，但一旦碰到就极度危险
- 标志性表现：被她蚀刻过的东西表面会留下类似冰裂纹的银灰色纹路

### 个人动机
她进入序列不是因为想证明什么，而是因为实力到了这个位置。她对序列制度不感兴趣，对排名不在意，唯一让她认真对待的事情是"我的能力边界在哪里"。她和主角有类似的"不太在同一频道上"的特质，但原因不同——主角是高位存在的频道错位，她是天才的社交隔离。

### 与主角关系线
- **定位**：镜像与后辈——主角"隐藏实力"的反面；她不藏，但也不炫耀
- **关系走向**：第二卷可能是最先和主角产生直接互动的序列成员（距离最近的年级），第三卷可能成为主角正式进入序列视野的催化剂
- **核心互动**：两个"不太合群的强者"之间的微妙认同——但秦落微不知道主角到底强到什么程度

## 写作使用指南

### 第一卷
- 不展开任何序列成员的个人剧情
- 可使用的元素：论坛传闻、训练场预约占用（系统显示"高阶序列占用"）、学生议论中提及编号或"七星"
- 最多出现一个名字（在论坛帖子或议论中一闪而过）
- 不让序列成员正式登场或介入旧训练楼事件

### 第二卷起
- 逐步让序列成员登场，优先从No.7秦落微（年级最近）和No.3岑远行（性格最活跃）开始
- 序列成员的登场必须带有独立的人物弧线和剧情功能，不能只是"出来打一架"
- 主角与序列的关系从"被比较"过渡到"被注意"，再到"被挑战/拉拢"

### 跨卷注意
- 七人的设定是框架，具体细节可在对应卷的写作中调整
- 能力层级会随剧情推进变化（毕业、升级、受伤等）
- No.1裴泽渊的“绝域”能力与主角三月权柄的对照关系是长线伏笔，需要从第二卷开始铺垫

## 序列之战（全校年度排位赛）

### 制度概要

序列席位不是终身制。学院每年举办一次**序列之战**——为期两周的全校排位赛，是曜月学院最大规模的年度赛事。所有在册学生均有资格报名参加，最终排名前七者获得下一年度的序列席位。

### 赛制结构

| 阶段 | 内容 | 时间 |
|------|------|------|
| **预选轮** | 全校报名者分组淘汰赛，按当前评级分档抽签，取前三十二人晋级 | 第一周前三天 |
| **积分轮** | 三十二人循环积分赛（非全打，按种子位抽取对阵表），取前十六人 | 第一周后四天 |
| **淘汰轮** | 十六强单败淘汰，决出最终排名前七 | 第二周 |
| **序列确认** | 教务处根据最终排名 + 裁判组综合评议，正式公布下一年度七星序列 | 赛后三天 |

### 核心规则

| 项目 | 规则 |
|------|------|
| **参赛资格** | 全校在册学生，无评级限制（但低评级者需通过预选轮证明实力） |
| **现任序列特权** | 现任七星成员自动跳过预选轮，直接进入积分轮（种子选手） |
| **裁判组** | 教务处派五名以上高阶导师担任裁判，院长级别担任终裁 |
| **判定标准** | 以胜负为主、综合评议为辅——边缘席位（第6-8名附近）由裁判组投票裁决 |
| **安全规则** | 全程在高阶结界训练场进行，裁判有权随时中断比赛，禁止致命手段 |
| **退出机制** | 毕业自动退出席位；重伤休学超过一学期自动由上届排名第八者递补 |

### 非赛季规则

- 年度排位赛外不设正式挑战通道
- 序列成员之间的排名互换不需要等排位赛——两人达成协议后报教务处批准即可对调
- 特殊情况：若学院判定某学生因外部事件（反面浮现、校外任务等）表现出远超当前排名的实力，可由院长级别直接提名"特批晋升"，不需等待年度赛

### 序列之战的叙事功能

1. **年度校园高潮**：序列之战是全校最大的事件，两周内论坛爆炸、赌注飞起、训练场爆满、全校围观——天然的"校园祭"级别叙事容器
2. **主角登顶阶梯**：主角团通过每年一次的排位赛逐步攀升，每次参赛都是一个完整的"tournament arc"
3. **时间压力**：一年只有一次机会，错过就再等一年——给备战阶段制造紧迫感
4. **全校参与**：不只是七星和挑战者的事，所有学生都在参与，制造群像和围观视角
5. **角色关系催化剂**：赛前备战（结盟、训练、情报战）、赛中碰撞、赛后格局变化都是高密度叙事

## 主角团替换路线

### 最终格局

主角团四人最终全部进入七星序列，形成新一代序列核心：

### 替换节奏原则

1. **不集中爆发**：四人的替换分散在第三到第四卷，不在同一届排位赛内同时完成
2. **多种入场方式**：白砚河和苏弥通过排位赛正面打入；林见星因姜吟渡毕业空缺+排位赛表现递补；哥伦比娅在第四卷准降临危机中暴露实力后，由院长在提前毕业手续生效前特批晋升No.1
3. **原序列成员不弱化**：他们不是"输了就消失"的工具人——毕业的继续在外界活跃，被击败的可能成为校外盟友或对立面
4. **主角最后登顶**：哥伦比娅的No.1是全书序列线的最终高光，必须压到最后
5. **学院序列线在毕业前收尾**：第二卷末尾主角团首次参赛（试水），第三卷排位赛白砚河突破，第四卷排位赛林见星和苏弥入围，准降临危机后哥伦比娅特批晋升；第五卷只承接学院公布、历史榜单和公会评价余波
