## 说明
1. 这张表看的不是“谁升了几级”，而是**每个人到底怎么长、长完能干什么**
2. 所有成长都得服从《神秘设定》的神秘核心逻辑，以及《战斗设定》的实战口径
3. 每个角色统一按「第一阶段」「第二阶段」「第三阶段」「终局担当」四段来拆
4. 本表只做角色成长概览，不作为当前卷能力解锁或等级裁决来源
5. 各角色的跨卷等级进度、神秘等级与战力层级拆分，按《角色能力发展设定模板》和对应角色能力发展设定执行

## 一、哥伦比娅·希珀塞莱尼娅

| 项目 | 内容 |
|------|------|
| **神秘核心/本质** | 三月女神承载者，外来继承者，非旧神复生者；真实力量来自三月权柄，不属于常规神秘体系 |
| **明面战斗岗位** | 兜底位、封印处理位、救场收口位 |

## 八点五、主角团不可替代价值矩阵

主角可以在纯战斗结果上兜底，但不能独自完成所有胜利条件。团队价值不以“谁能造成最高伤害”衡量，而以“没有谁就会失去哪一种胜利”衡量。

| 角色 | 不可替代维度 | 缺席时的直接后果 |
|---|---|---|
| **哥伦比娅** | 高位兜底、最终裁定、超出常规体系的收口 | 队伍可以完成局部任务，但无法处理真正越过常规上限的终局问题 |
| **林见星** | 路径确认、撤离线、救援、团队连接与生命续航 | 队伍可能赢下敌人，却无法保证所有人回来，也无法在崩盘时维持阵型 |
| **白砚河** | 正面承压、强行破口、斩首、打穿高耐久目标 | 队伍会缺少把情报与控制转化为实际突破口的执行者 |
| **苏弥** | 证据、术式库存、核心频率、签名追踪与战后归档 | 队伍可能解决眼前危机，却无法确认真相、阻止复发或形成后续追索链 |
| **月枝** | 环境塑形、范围控场、复杂地形与异域主场 | 队伍会被迫在敌方环境中作战，难以稳定处理大范围与长期战场 |
| **季悬灯** | 高位封锁、断外视、灾情收束与政治护航 | 主角团即使能打赢，也可能暴露身份、引来更高层敌意或让灾情扩散失控 |
| **方导师** | 学生阶段的流程接管、职业判断、带队保护与学院制度接口 | 学生会被迫承担不该承担的职业风险，学院秩序也无法合理接住后果 |

使用原则：

1. 战斗胜利、人员存活、证据保全、身份保密、灾情不扩散和社会后果可控是不同胜利条件。
2. 主角出手可以解决最强敌人，但不能自动补回撤离线、证据链、政治授权和战后善后。
3. 配角高光应优先落在“只有他能完成的胜利条件”，而不是临时把伤害数值抬到主角附近。

## 九、发展原则
1. **成长别只看伤害**：更重要的是职责变重了、能打的场合变多了、有没有资格碰更高危的事
2. **每个人都得有自己的位子**：不能写到最后变成只剩主角一个人在解决全部问题
3. **团队分工要一直清楚**：白砚河最燃，林见星最稳，苏弥最巧，月枝最能铺场，季悬灯压高位秩序，主角专门兜底收场
4. **主角不是升级流**：她不是从弱变强，而是从压着神性打，慢慢回到真正能执掌神权的状态

### 10.1 哥伦比娅——抗性与认知跨卷变化

| 阶段 | 抗性机制 | 认知层级 |
|------|----------|----------|
| 第一卷 | 归一之躯被动排斥（不进抗性三要素链）；Ch1 被动冷白反馈一次性；Ch8 脚底微凉呼应；Ch11-13 接触侵蚀残留时手凉→结霜→冻停 | 主角独有认知（模糊）：不理解被动反应机制，只知道"身体在自动反应" |
| 第二卷 | 被动排斥继续；开始出现"低频冷白反馈"的稳定表现 | 主角独有认知（深化）：隐约感到"这跟我的身体有某种反应"——不点破 |
| 第三卷 | 日常反面环境暴露增多，被动排斥频次微增 | 主角独有认知（具体化）：开始反向追问"这跟我变强有关系吗"——仍不确认 |
| 第四卷 | 学院准降临危机中被动态势压力提升；被动排斥在准降临中可能短暂显形 | 主角独有认知（接近质变）：开始触碰"我可能不只是继承者" |
| 第五卷 | 接受"反面与三月权柄同源"——但仍不跨过"我自己参与了制造"的心理边界 | 主角独有认知（确认同源）：仍不跨过"我自己参与"边界 |
| 第六卷 | 三线法则联动时被动排斥可能从"接触残留触发"升级为"全境反推" | 主角独有认知（触碰旧时代）：三线法则联动时触碰"封印松动导致的上古黑暗侵蚀世界" |
| 第七卷 | 三月归一时不再走抗性三要素基线，转为"位格确认"——归一之躯与反面底层规则的关系由"排斥"转为"裁定" | 归一时由权柄/旧神意志给出"真实底层"——主角接受真实底层，但不原谅也不惩罚 |

**关键约束**（跨卷不变）：
