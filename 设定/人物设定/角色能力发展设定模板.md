# 《三月女神今天也只是普通学生》角色能力发展设定模板

---

## 一、用途与适用范围

本模板是角色能力发展文件的结构唯一来源，用于解决“当前卷等级不能代表后续等级”“神秘等级与实战战力混写”“后续能力被误当成当前已解锁能力”等问题。

角色能力发展文件描述的是角色跨卷成长路径，不给角色设置一个永久不变的等级标签。所有等级、能力与战斗表现都必须落到具体阶段和具体卷次中。

普通角色按“正式神秘等级 + 实战战力层级”双轴记录。主角属于常规神秘体系外例外，按“权柄接驳阶段 + 对外神秘等级/身份 + 实际战力层级”记录。

---

## 二、角色能力归属与成长链

角色层能力必须先分清两条互不替代的路线：

1. **神秘核心**：每个人唯一且不可复制的个人能力本质。它决定角色能做什么、为什么只有本人能做到、能力如何成长，以及代价与反制边界。
2. **核心技/核心应用**：角色对自身神秘核心的具体用法。个人招式、个人形态、专属感知、专属兵装和专属战场结构都归入这一层，不再归入术式分类。
3. **术式**：学院、公会和职业体系可以教学、记录、抄录与复现的通用能力。吟唱只属于术式；核心技可以有喊名、引导语或解放语，但这些都不是吟唱。

普通角色的核心成长统一使用：

> **核心回响 → 核心显相 → 真名解放 → 界临 → 立序**

与正式神秘等级的对应关系如下：

| 神秘等级 | 内部阶段 | 核心成长含义 |
|---|---|---|
| **神秘雏形** | 初响 → 稳响 → 临显 | 从第一次得到核心回应，到能够稳定回应，再到逼近个人能力显相 |
| **神秘者** | 初显 → 定相 → 深显 | 核心能力首次稳定显现，逐步定型，并把同一能力本质推到更深层 |
| **深律者** | 初解 → 稳解 → 深解 | 触及并逐步稳定真名解放，让核心本相进入完整战斗形态 |
| **域主** | 不细分 | 完成界临，个人核心能够接管或持续改造局部主战场 |
| **秩序者** | 不细分 | 完成立序，个人核心能够稳定制定一部分局部规则 |

使用边界：

- **临显不等于尚无个人能力**：神秘雏形·临显可以在固定媒介、动作、目标、环境或短时爆发条件下，局部显出核形并使用核心技雏形；招式已经命名不等于角色已经晋升神秘者。神秘者·初显的判据，是离开单一触发条件后仍能跨环境、可重复地维持完整核心显相。
- 真名解放不是统一变身模板，可以表现为身体、兵装、召唤物、书阵、环境或规则结构的质变。
- 界临不强制写成封闭结界；只要个人核心真正铺到战场结构层并持续改变作战条件，即可构成界临。
- 核心技的喊名只是战斗锚点，不自动构成发动条件。
- 解放语用于确认、稳定或开启真名解放，不得写成术式吟唱。
- 同一角色可以学习多种通用术式，但只能拥有一个神秘核心；通用术式不能替代个人核心的辨识度。

## 三、固定信息与阶段信息的边界

### 固定信息

以下内容可以放在文件顶部，作为跨卷稳定设定：

- 神秘核心与底层法则
- 核心技从底层法则派生的统一逻辑
- 核心执念与成长方向
- 长期战斗岗位
- 永久能力边界
- 视觉特征与写法原则

### 阶段信息

以下内容不得作为角色永久标签，必须写入跨卷等级进度表或阶段能力卡：

- 神秘等级
- 核心成长节点
- 战力层级
- 常态状态
- 爆发上限
- 当前可用核心技、通用术式与不可用能力
- 能力范围、消耗、代价、反制方式
- 阶段结束标志与下一阶段解锁条件

---

## 四、等级双轴与状态定义

| 项目 | 定义 | 使用要求 |
|---|---|---|
| **神秘等级** | 角色在常规神秘体系中的正式等级与阶段 | 使用《神秘设定》的正式等级名称，不使用战力层级名称代替 |
| **核心成长节点** | 角色的神秘核心处于回响、显相、真名解放、界临或立序中的哪一步 | 必须与正式神秘等级匹配，不把核心技熟练度误写成等级晋升 |
| **战力层级** | 角色在实战中的综合威胁、压制与处理能力 | 使用《战斗设定》的战力层级名称，可与神秘等级不完全一致 |
| **常态状态** | 角色能够稳定、重复使用的日常战斗输出 | 当前卷正文默认按此状态写 |
| **爆发上限** | 角色在明确条件与代价下能够短时达到的最高表现 | 必须同时写明触发条件、持续时间、代价与恢复要求 |
| **阶段终点** | 角色完成该阶段时能够稳定维持的状态 | 用于衔接下一阶段，不等同于阶段全程水平 |

神秘等级与战力层级必须分栏记录。角色可以因能力克制、装备、经验、界临、团队配合或特殊代价出现越级战斗表现，但不得因此直接改写其正式神秘等级。

## 五、跨卷等级进度表

每份角色能力发展文件都应在神秘核心之后、详细阶段之前设置跨卷等级进度表。

| 阶段 | 对应卷 | 神秘等级 | 战力层级 | 常态状态 | 爆发上限 | 阶段结束标志 |
|---|---|---|---|---|---|---|
| 第一阶段 | 待填写 | 待填写 | 待填写 | 待填写 | 待填写 | 待填写 |
| 第二阶段 | 待填写 | 待填写 | 待填写 | 待填写 | 待填写 | 待填写 |
| 第三阶段 | 待填写 | 待填写 | 待填写 | 待填写 | 待填写 | 待填写 |

表内范围表示该阶段从起点到终点的变化，不表示角色在阶段开始时已经拥有终点能力。

## 六、阶段能力卡

每个详细阶段至少应包含以下字段：

| 字段 | 内容要求 |
|---|---|
| **对应卷** | 明确该阶段主要覆盖的卷次或剧情区间 |
| **神秘等级** | 正式神秘等级及阶段范围 |
| **核心成长节点** | 本阶段从哪一层核心状态推进到哪一层 |
| **战力层级** | 实战战力层级及阶段范围 |
| **常态状态** | 当前阶段可稳定复用的输出与处理范围 |
| **爆发上限** | 短时上限、触发条件、持续限制与代价 |
| **核心主题** | 该阶段能力成长的核心命题 |
| **战斗岗位** | 该阶段在队伍或战场中的职责 |
| **可用核心技** | 本阶段已经解锁并可在正文中使用的个人能力与个人招式 |
| **可用通用术式** | 本阶段已经学会或准备完成的可教学术式；没有则明确写无 |
| **不可用能力** | 后续阶段能力、禁用表现与越界写法 |
| **发动与反制** | 核心技写触发、媒介、作用范围、代价与反制；通用术式另写吟唱层级与构筑条件；真名解放另写解放语及开启条件 |
| **阶段结束标志** | 进入下一阶段前必须完成的稳定化结果 |

已有角色文件可以分步补齐字段，但“神秘等级”与“战力层级”必须先完成拆分。

---

## 七、当前卷引用规则

角色能力发展文件负责记录跨卷成长，不直接决定当前章节可以使用什么能力。

当前卷写作时，能力可用性按以下优先级裁决：

1. `设定/当前硬口径.md`
2. 当前卷的当前写作状态、事件链与口径卡片
3. 对应角色能力发展设定中的当前阶段
4. 若使用个人能力、真名解放、界临或立序，再读取 `设定/核心技与解放归档.md`
5. 若使用通用术式，再读取 `设定/术式吟唱归档.md`

后续阶段在能力发展文件中出现，不代表当前卷已经解锁。两份归档负责连续性，不负责越过当前卷白名单提前解锁能力。

---

## 八、主角体系外例外

主角不是普通意义上的升级角色，不使用单一正式神秘等级描述真实力量。主角能力发展文件应使用以下三轴：

| 轴 | 含义 |
|---|---|
| **权柄接驳阶段** | 她接过三月权柄的进度，是主角真实成长主轴 |
| **对外神秘等级/身份** | 学院、队友和敌人能够观察并合理解释的公开口径 |
| **实际战力层级** | 她在当前阶段与当前出手模式下能够真实达到的战斗表现 |

主角的伪装态、女神态与创世态是战斗表现层，不替代权柄接驳阶段。任何卷次的可用模式与实际上限仍以当前硬口径和当前写作状态为准。

主角真实力量属于三月权柄，不拥有常规意义上的神秘核心，也不进入“核心回响—核心显相—真名解放—界临—立序”成长链。她对外伪装成罕见冰霜类核心持有者；霜楔、霜华、霜印等只作为伪装核心技被旁观者理解，不登记为个人术式。
