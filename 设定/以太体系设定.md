## 二、以太的本质

### 2.1 对外口径（学院/普通神秘者认知）

以太是这个世界最常见的超凡能量载体。学院只教它是"常见超凡能量载体"，不讲起源。绝大多数神秘者把以太当做理所当然的修炼材料，不会追问来历。

### 2.2 秘史真相（写作者知道，第三卷后按梯度揭露）

三位月之女神降临压下上古黑暗时，月光照过世界，在物理层沉淀出可被术式调用的超凡载体，后世称之为以太。

这解释了以下现象的底层逻辑：
- 以太沉积点高度集中于旧时代遗迹、灵脉节点和三月权柄共鸣区域
- 三月断联事件后，世界以太总量逐渐出现缓慢下降趋势（学院有内部监测数据，未公开）
- 反面内部的以太流向与正面相反（逆流），是因为反面是三月女神以死亡完成的上古封印因松动，上古黑暗侵蚀世界所形成的异常规则空间——同源力量在封印松动过程中的反作用导致以太逆流

**揭露节奏**：第三卷苏弥可发现旧档案中"以太沉积峰值与三月断联时间点"的关联，作为秘史证据之一。正文不提前点破。

## 三、以太的物理性质与分类

以太不是单一物质，而是超凡能量在物理层的稳定化形态，按状态分为三类：

| 形态 | 描述 | 用途 |
|------|------|------|
| **气态以太** | 弥散于特定区域空气中，肉眼不可见，可用专业仪器检测浓度 | 修炼效率基础；高浓度区域修炼加速 |
| **晶化以太**（以太结晶） | 自然沉积或人工聚集形成的固态载体，稳定可存放 | 储能、交易、设备供能 |
| **以太场** | 高浓度以太形成的稳定能量场，需要阵列维持 | 学院训练设施区的核心供能基础 |

### 3.1 以太浓度的影响

- 高浓度以太环境下，术式成型速度和稳定性明显提升
- 浓度不足会缩短高阶术式的持续时间和范围
- 反面内部以太逆流，导致术式输出方向偏移——这是反面战斗高危的来源之一，不是神秘核心本身变弱

### 3.2 以太调用与神秘输出闭环

常规神秘者的能量流程固定为：

```text
环境以太 / 以太结晶 / 设备供能
→ 神秘核心感知并调用
→ 核心按个人法则与能力结构完成转化
→ 形成神秘输出与术式效果
→ 通过休息、低负荷环境和以太补给恢复调用能力
```

1. **个人不大量储存以太**：人体与神秘核心只能维持少量周转余量，不能把自身当成长期以太仓库。
2. **神秘核心不是燃料罐**：核心负责转化、定向与稳定，以太负责提供常规术式所需的能量载体。
3. **神秘力量消耗**：指核心转化负荷、个人精神与身体负担、以及可调用以太余量的综合下降，不是另一种与以太无关的独立燃料。
4. **神秘输出**：指经过核心转化后实际进入术式、武器或战场的力量表现。
5. **无以太环境**：常规神秘者仍可依靠体内少量周转余量发动少数基础能力，但持续时间、范围与稳定性快速下降；高阶术式和长期战斗无法正常维持。
6. **主角例外**：三月权柄线不经过常规以太调用与神秘核心转化，因此可以出现零以太波动、零术式构筑痕迹的体系外结果；其明面神秘核心伪装仍按常规流程表现。

## 四、以太的采集与流通

### 4.1 来源渠道

| 来源 | 说明 | 产量 | 管控 |
|------|------|------|------|
| **自然沉积** | 旧时代遗迹、灵脉节点周边的长期沉积 | 低但质量高 | 采集权归学院/公会管理 |
| **反面采集** | 经正规流程处理后的反面资源衍生产物 | 中，需专业团队 | 严格；学院内部为主 |
| **阵列量产** | 以太收集阵在普通环境下的工业性聚集 | 高但品质偏低 | 公开，百城市场主要来源 |
| **遗迹发现** | 旧时代封存的高阶以太结晶 | 极低，不可预期 | 须上交检验，古遗级 |

### 4.2 流通层级

| 等级 | 形态 | 流通范围 | 管控 |
|------|------|----------|------|
| **标准以太晶** | 量产低阶结晶 | 百城公开市场、学院积分兑换 | 低 |
| **精炼以太晶** | 中阶处理结晶 | 学院高阶兑换区、公会奖励 | 中 |
| **高阶以太核** | 高密度稀有结晶 | 秘藏级市场、高位拍卖行 | 高 |
| **反面衍生以太** | 反面资源处理产物 | 学院内部配给、公会专项 | 严格管控 |

### 4.3 战略意义

- 以太是百城经济的重要战略资源，灵脉节点和遗迹的采集权是各城政治博弈的核心筹码之一
- 控制高阶以太来源等于控制一定程度的地区话语权
- 公会任务中大量"远征遗迹""封印点维护"的委托，本质上有相当部分是高阶以太采集任务
- 学院以太配给制度是一种权力分配机制，不只是后勤安排

## 五、学院以太体系

### 5.1 以太配给制度

学院以太设施区按实测评分发放月配额（详见《曜月学院学生评级与待遇体系》"以太配额"节，本文件不重复）。

核心写法要点：
- S级的"独立调配额度"不是无限免费，每次超额申请都会留下审批记录
- 这是优待，也是监视
- 普通学生会因以太配额不足产生现实压力，这是制度性的训练门槛，不是恶意为难

### 5.2 以太设施功能

学院以太设施区（以太中枢）的四类功能：

1. **教学与训练**：为银阶训练场、深层训练室提供稳定以太供能；高阶训练室的高压环境依赖精炼以太维持
2. **防护**：学院边界以太防护层，低阶怪异难以穿越
3. **监测**：院内以太浓度和异常波动的实时检测
4. **后勤**：照明、门禁、宿舍热水等基础设施的稳定供能

### 5.3 反面资源作为以太补充

正规流程：**采集 → 封装 → 净化 → 鉴定 → 分级 → 配给 → 导师/公会监管使用**

- 正规高阶训练场使用经完整流程处理的反面资源作为补充供能或压力模拟材料
- 经过完整处理的反面资源是以太体系的合法补充来源，学院有完整的管控链
- 影子训练场的骗局核心：把"正规处理后使用"偷换成"直接接触封印区源头"——后者可以造成真实的即时输出增益，但不会转化为神秘核心的长期成长；仪器记录到的上扬属于不可持续的假峰值，并伴随身体侵蚀

**写法区分**：
- 正规渠道：以太结晶 → 设备充能 → 稳定输出
- 骗局渠道：直接接近反面源头 → 即时输出短暂增益 + 测量数据假峰值 + 侵蚀积累 + 核心不稳

## 六、以太与神秘体系的关系

以太是神秘者修炼、术式发动和武器充能的通用能量载体，但二者不能等同：

| 对比项 | 以太 | 神秘 |
|--------|------|------|
| **本质** | 超凡能量的物理载体 | 个人认知与规则咬合的独特投影 |
| **可量化** | 可测量浓度、可存储、可交易 | 不可直接量化，因人而异 |
| **可替代** | 高浓度以太可以提升修炼效率 | 以太无法替代规则理解 |
| **成长作用** | 加速"修炼时间积累" | 决定"能力方向和上限" |

**核心原则**：高以太浓度环境只加快熟练度积累，不能压缩规则理解过程。一个以太资源充裕的天才，和一个以太资源匮乏但实战经验丰富的神秘者，最终成长上限由神秘核心决定，不由以太决定。

## 七、秘史关联（内部设计，不进正文）

以太的真实起源确认了以下秘史钩子：

1. **以太沉积峰值地图 = 三月权柄共鸣强点地图**：旧时代遗迹往往是权柄残片集中处，不只是古人留下的建筑
2. **三月断联后以太总量缓慢下降**：这是一条尚未公开的内部监测数据，揭露后会解释为什么近几百年来高阶神秘者的诞生越来越困难
3. **反面以太逆流与三月权柄碎片**：反面内部的以太逆流现象，本质是被用于编织反面的三月权柄碎片在持续"吸收"正面以太

**揭露梯度建议**：
- 第三卷：苏弥发现档案里以太峰值分布图与旧时代遗迹完全重合，但无法解释原因
- 第五卷：公会远征时，发现某处遗迹以太枯竭与三月断联时间点完全匹配
- 第六卷：长老讲出创世传说时，以太起源的真相作为其中一个组成部分完整呈现

## 八、写法口径

1. **以太是辅助轮，不是成长核心**：以太资源稀缺可以制造现实压力，但不要让它成为成长的唯一卡点
2. **不把以太写成神秘力量本身**：它是载体，不是本体；神秘者不是"充以太"就变强，而是"在高以太环境下练得更有效率"
3. **消耗感要写出来**：使用以太结晶的设备、符文、武器都有明显的"用了多少"的消耗感，不是无限供能
4. **学院配给描写落到场景**：写以太配额时，落到"排队领取""配额用完提前""S级走优先通道"等具体动作，不挂嘴边
5. **不把以太和三月权柄写成同一种东西**：同源，但以太是"月光沉淀后的物理残留"，三月权柄是"月之女神的法则权限"，二者有根本性质的差异
