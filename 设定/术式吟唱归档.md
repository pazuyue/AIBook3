# 术式吟唱归档

## 文件定位

本文是“七十二标准式谱”与正文术式连续性的定稿索引，只记录可教学、可抄录、可由不同施术者复现的通用术式。

个人核心能力、核心技、核心引导语、解放语和界临名统一记录在《核心技与解放归档》，不得再放入本文。

使用优先级：

1. 术式分类、编号区间与公共轴规则，以《神秘设定》《战斗设定》为准。
2. 某名角色能否使用某个术式，以对应角色能力发展设定和当前卷白名单为准。
3. 已注册术式的编号、名称、标准功能与吟唱，以本文为连续性定稿。
4. 正文首次出现、节省吟唱变化和预先构筑记录，以本文“正文连续性”表为准。

---

## 一、编号与命名规则

标准格式：

```text
【分类·序号，术式名。】
```

七类术式为：攻式、禁式、护式、行式、察式、净式、阵式。

| 序号区间 | 常见权限 | 写作口径 |
|---|---|---|
| 一至十八 | 新生与基础职业课程 | 结构清楚、功能单一，靠时机与熟练度发挥价值 |
| 十九至三十六 | 高年级、成熟职业者常用进阶式 | 可以出现复合效果、远距构筑或更高穿透 |
| 三十七至五十四 | 高阶职业式与复杂复合式 | 通常需要专门训练、媒介或明确适配 |
| 五十五至六十六 | 高风险或多人协同式 | 常见于职业小队、封控任务与大型战斗 |
| 六十七至七十二 | 战略级、限制公开或大型阵列式 | 不作为学生随手使用的常规技能 |

编号是全谱注册号，不在不同分类中重复占用。它大致反映复杂度、负荷与权限，但不是绝对威力排行；低编号术式用得准确，仍可打断高编号构筑。

## 三、标准吟唱

吟唱用于补全公共回路。核心技喊名、核心引导语、解放语与界临名不是吟唱。

### 察式·四，定标

- 节省吟唱：`【一点照见，一息留痕。察式·四，定标。】`
- 全吟唱：`【以所见为点，以所知为标；留痕一息，为后来者指路。察式·四，定标。】`

### 行式·七，折步

- 节省吟唱：`【借力折向，一步换轨。行式·七，折步。】`
- 全吟唱：`【足下借力，身前改向；旧势归零，新路成步。行式·七，折步。】`

### 攻式·八，针芒

- 节省吟唱：`【定标，贯核。攻式·八，针芒。】`
- 全吟唱：`【一点定标，一线贯心；锋芒循路，破壳见核。攻式·八，针芒。】`

### 护式·九，偏甲

- 节省吟唱：`【斜面承击，余势偏流。护式·九，偏甲。】`
- 全吟唱：`【以斜面承击，以侧隙泄力；正锋不受，余势偏流。护式·九，偏甲。】`

### 攻式·十一，短炮

- 节省吟唱：`【聚流，离掌。攻式·十一，短炮。】`
- 全吟唱：`【聚流于掌，束压成膛；前路既定，冲击离弦。攻式·十一，短炮。】`

### 禁式·十二，锁步

- 节省吟唱：`【三点定足，越线止步。禁式·十二，锁步。】`
- 全吟唱：`【以三点定足，以四隅封路；越界者止，受缚者留。禁式·十二，锁步。】`

### 禁式·十四，缚章

- 节省吟唱：`【以签为钉，以影为页。禁式·十四，缚章。】`
- 全吟唱：`【以签为钉，以影为页；四肢各归其位，越动者自缚其章。禁式·十四，缚章。】`

### 净式·十六，净尘

- 节省吟唱：`【剥异，退浊。净式·十六，净尘。】`
- 全吟唱：`【剥离异质，归还本色；浅痕止步，浊流退散。净式·十六，净尘。】`

### 阵式·十八，止裂

- 节省吟唱：`【四角归位，裂隙暂止。阵式·十八，止裂。】`
- 全吟唱：`【四角归位，八线相持；破处不进，裂隙暂止。阵式·十八，止裂。】`

### 攻式·二十七，贯星

- 节省吟唱：`【聚芒成轨，贯星而去。攻式·二十七，贯星。】`
- 全吟唱：`【收四野微光，定一线长轨；所指之处，贯星而去。攻式·二十七，贯星。】`

## 四、发动层级

| 层级 | 连续性定义 | 正文要求 |
|---|---|---|
| 全吟唱 | 完整补全标准回路 | 威力、范围、稳定性和附加效果最完整；必须给出施术窗口 |
| 节省吟唱 | 省略次要节点，只保留主要功能 | 更快，但至少一项效果缩水 |
| 无吟唱 | 不经声音完成完整回路 | 仅限顶尖熟练者、特殊媒介或明确代价；必须仍有术式构筑轨迹 |
| 预先构筑 | 把完整回路存入书页、符签、阵盘或设备 | 临场近似瞬发，但库存有限且需要战前准备 |
| 多人构筑 | 多人分别承担节点或回路段 | 允许使用更高编号术式；任一关键节点被破坏都可能崩式 |

“只报标准名”不自动等于无吟唱。若回路已预存，报出术式名可以只是选择、释放与队友提示；若没有预存，则必须说明熟练压缩、媒介辅助或相应代价。

## 六、归档规则

每章定稿前，出现下列任一情况就补充本文：

1. 新的标准术式编号或名称；
2. 已有术式首次出现全吟唱、节省吟唱、无吟唱或预先构筑版本；
3. 标准功能出现经学院批准的正式改式；
4. 某术式首次形成重要连续性限制。

最小记录格式：

```text
使用者：
分类·编号·名称：
发动层级：
吟唱原文：
首次出现：
正文效果：
后续禁改点：
```

禁止事项：

1. 不把裂锋、星丝、断线瞳、归档、寒潮、压线、悬灯等个人能力登记成术式。
2. 不给核心技补全吟唱、节省吟唱或术式回路。
3. 不因角色喊出招名，就把个人能力改判成术式。
4. 不随意占用已注册编号；确需改号时，必须同步全文、角色设定和本文。
5. 主角出现“零核心激活、零以太调用、零术式构筑却产生结果”时，不得归档为无吟唱术式；那是体系外异常。
