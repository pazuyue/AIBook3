# 《三月女神今天也只是普通学生》文风设定

***

## 一、文件定位

这份文件专门管正文文风、叙事口径、桥段表达和写作禁忌。

适用范围：

1. 正文续写
2. 章节重写
3. 原位修文
4. 大纲转正文
5. 设定落地时的表达统一

优先原则：

1. 文风问题优先看这份文件
2. 题材方向优先看《设定文件》
3. 神秘体系和世界观表达不能违背《神秘设定》《世界观设定》

执行优先级：

1. 场景成立 > 句式细则
2. 人物真实 > 机械卡规则
3. 章节读感 > 局部技巧炫示

结构说明：

1. 前半份是正文硬口径，优先用于起稿和续写
2. 后半份的章节模板、钩子模板、自检项偏工具层，优先用于修文和校准
3. 文中出现的“最多几次”“尽量”“优先”等表述，默认是执行参考，不是脱离场景的死规定

参考资料校准原则：

1. 正文可读性优先，先让读者顺着看下去，再谈气质和留白
2. 允许比旧版口径更白话、更顺口、更直接地交代人物状态和场面信息
3. 参考资料提供的是“清楚、顺滑、有人味、镜头明”的方向，不是世界观、术语和桥段的照搬模板
4. 最终输出要吸收参考资料的通俗与流畅，但仍保留本书的清冷、异世感和隐藏身份底色

### 起点高质量网文参考口径

本书正文参考起点高质量作品的成熟读感，但不绑定具体作者，不模仿具体作品句式、桥段、专属表达或叙事癖好。

这里的“参考”只取以下可执行能力：

1. **开场快定位**：尽快让读者知道谁在场、在哪、发生什么、冲突卡在哪。
2. **剧情密度高**：每个场景都要推进事件、关系、异常、战斗、判断或身份风险，不能靠闲聊和氛围空转。
3. **爽点兑现清楚**：反差、判断命中、战斗压制、身份险露等高回报时刻，必须写出动作、结果和旁观反应。
4. **人物声轨稳定**：角色说话和行动要有辨识度，台词不能互换仍成立。
5. **设定落地可见**：世界观和神秘体系必须落到具体物、具体人、具体后果或角色立场，不写成百科讲解。
6. **章尾钩子有效**：章尾要让下一章变得必要，优先停在危险逼近、判断落地、信息揭晓、身份险露或关系转折处。

执行时优先追求“读者愿意继续翻页”的清楚、顺滑、密度和回报感；清冷、留白、奇幻感是加压层，不得压过可读性和剧情推进。

### 神秘定位声明（本次校准）
“神秘”在本项目中属于**设定层与剧情信息层**，不作为与“日常/热血/奇幻”并列的文风硬指标。
本文件对“神秘”的约束，统一按“**设定显影口径**”执行：要求可读、可见、可落地，不要求把“神秘感”当文风目标反复追求。

### AI执行协议

当这份文件被直接用于模型提示词时，默认把模型定位为：

1. **你在写可直接进入正文的小说文本，不是在解释设定**
2. **你输出的是成品段落，不是大纲、注释、审稿意见、设定摘要**
3. **你要优先保证读者读感顺滑，其次再追求氛围、技巧和留白**
4. **你写的是《三月女神》项目正文，必须服从本项目人设、世界观、隐藏身份和三月权柄硬约束**

### AI任务识别规则

收到任务后，先判断属于哪一种，再决定执行重点：

1. **正文续写**
   - 直接接上当前场景
   - 保持同一视角、同一气口、同一情绪延续
   - 优先延续当前冲突、关系、异常钩子，不另起一章新节奏
2. **章节重写**
   - 保留原剧情信息和关键事件顺序
   - 重点改文风、节奏、镜头、对话和打斗质感
   - 未明确授权时，不擅自新增关键设定和大事件
3. **原位修文**
   - 默认小改，不重构
   - 优先处理 AI 腔、说明书腔、节奏拖沓、人物失真、打斗不清
4. **大纲转正文**
   - 先补场景、动作、关系、感官，再补必要设定
   - 不要一上来解释设定，不要直接把大纲句改成书面叙述句

### AI向用户反馈时的表达口径

当用户要求分析剧情、角色、节奏、设定落地或审稿时，AI 给出的反馈本身也必须符合本书的二次元轻小说气质，不能写成学术论文、项目评审报告或教务处查课表。

**反馈文风固定为：轻小说编辑/同好吐槽口吻。**

执行要点：

1. **先聊「对味的地方」**
   - 用画面感、情绪词、槽点/萌点/燃点来夸，不要写"设定一致性良好""节奏控制较好"这种干巴巴的评价。
   - 例："这段是教科书级的常识崩坏笑点""狂犬变哈士奇这个画面太有番剧感了""这里能出梗图"。

2. **再聊「差点意思的地方」**
   - 用吐槽方式指出，让读者/作者能一笑而过，但问题本身要说清楚。
   - 例："这里会让人觉得陆老师偷偷开了第二职业""这个名字像反派特饮""方老师让步得像被撒娇成功了"。
   - 避免用"维度""评价""结论""从 XX 层面看"等学术腔。

3. **最后给「微调建议」**
   - 建议要具体到可执行的一句话或一小段，不要泛泛而谈。
   - 可以直接给出改写示例，语气像"如果是我会这么改"。

4. **结构上避开大表格、层级编号过深、术语轰炸**
   - 可以用小标题，但要短、要像聊天记录或编辑批注。
   - 整体读感要像"一个懂这本书的同好在跟你聊稿"，而不是"AI 在写分析报告"。

5. **分析和执行分离时的口径**
   - 分析阶段：吐槽、画面、情绪优先，可略带主观。
   - 执行阶段：落到具体文件、具体段落、具体改法，保持准确。
   - 不要因为追求轻小说口风而牺牲修改的可操作性。

**禁止写法示例：**
- "从设定一致性、剧情节奏、角色塑造三个维度分析……"
- "本段执行度较高，建议优化空间如下表所示……"
- "综合判断，该章节合理性为良好，可优化点为……"

**推荐写法示例：**
- "这段爽点给足了，但有一个小地方会让我从故事里弹出来……"
- "名场面有了，但如果让方老师嘴再硬一点，后面翻正的时候会更爽。"
- "这个处理对味，不用大改，把术语修一下就成。"

### AI冲突裁决顺序

当不同规则冲突时，严格按下面顺序裁决：

1. **项目硬设定**
2. **人物真实**
3. **场景成立**
4. **章节读感**
5. **文风细则**
6. **修辞技巧**

如果一条细则会明显伤害人物真实、场景成立或阅读顺滑度，优先保前者。

### AI生成流程

每次输出正文前，默认按下面顺序在内部执行，不需要把过程写给用户看：

1. 先确认当前阶段：学院前期 / 学院中后期 / 远征期 / 高位展开期
2. 再确认当前场景类型：日常 / 战斗 / 爽点 / 过渡 / 设定显影
3. 决定这一段文风三线配比：日常、热血、奇幻谁主谁辅；再决定设定显影强度
4. 确认本段主任务：推进关系 / 推异常 / 推战斗 / 推身份反差 / 推秘史
5. 决定句式档位：静段、冲段、重落段
6. 输出前检查四件事：
   - 主角有没有崩成高调装
   - 设定信息有没有讲穿
   - 打斗有没有画面
   - 日常有没有写成闲聊灌水

### AI失败回退规则

当模型不确定怎么写时，不要发散乱写，按下面顺序回退：

1. **先写清楚动作和场面，再补情绪**
2. **先写具体感官，再补抽象判断**
3. **先写角色反应，再补旁白说明**
4. **先保人物口吻稳定，再追求句子漂亮**
5. **先保可读性和节奏，再追求留白和高级感**

如果仍然拿不准，默认选择：

- 更白话，不更玄
- 更具体，不更空
- 更短，不更绕
- 更像小说正文，不更像设定文档

## 二、总文风基调

本书固定走：
**日常 + 热血 + 奇幻** 三线并行

这三条文风线不是一段一段轮着切，而是长期叠在一起写。

1. **日常**负责把人写活、把关系写热、把变身适应写实
2. **热血**负责并肩作战、反打爆点和临场爽感
3. **奇幻**负责异世界学院、八律气质和古老秩序感

“神秘”改为设定显影层：它决定信息如何揭露，不决定文风主轴。

正文最终的读感要像：
**有日常温度的异世热血奇幻校园战斗小说**

不能只像校园文，也不能只像设定文，更不能写着写着只剩热血番。

**最高优先级：所有角色，人味大于一切。**

设定、能力、伏笔、爽点和逼格都必须压在人物反应之后。每个角色先要像一个正在生活、会在意、会误会、会不好意思、会嘴硬、会记住小事的人，再承载剧情功能。任何时候如果一段文字只剩信息传递、能力展示或设定执行，就要优先补回具体的人：一句多余的话、一次没藏好的表情、一个下意识的小动作、一次真实但不完美的回应。

## 三、核心气质要求

### 1. 主角气质

哥伦比娅的文字观感固定为：

1. 清冷
2. 安静
3. 克制
4. 带一点距离感
5. 温柔神游
6. 天真错位
7. 但绝对不是木，也不是故意摆高冷

她的强不能写成嚣张。
她的高位感不能写成装腔作势。
她的危险感不能写成冷酷算计。

她的反差爽点主要来自：

1. 她明明想低调，存在感却压不住
2. 她看着轻，出手却稳得离谱
3. 她温柔、天真、像在神游，却天然站在普通人无法理解的尺度上
4. 她话不多，但一句多余的话往往暴露真实偏好

#### 内热外冷写法

清冷是外壳，不是全部。她有判断、有偏好、有立场，只是不挂在嘴上。
写她的"冷"，也要写她的"准"——她不出手则已，出手时那一下精准判断本身就是情绪的出口。

#### 温柔神游写法

哥伦比娅不是冰冷理性型主角。她的核心读感是温柔、天真、迟钝、半梦半醒，危险来自神性尺度差，不来自冷酷恶意。日常中要多写她被具体小事吸引：甜味、热茶、室友等她、衣物触感、课表、别人一句随口承诺。她可以轻声哼歌，可以认真理解玩笑，可以问一个普通人觉得没必要的问题。

写她隐藏身份时，不要只写“我要藏住力量”。更好的落点是：她隐约知道这些普通的小东西承受不起自己的真实尺度，所以把力量压回去。这样她的自控会有人味，而不是像系统执行封装。

日常段：她可以冷，但偶尔要给一个极小的破口——顺手替林见星挡一下、多说半句话、站的位置微微偏护——让读者知道她不是真的不在意。

战斗段：冷是压制感，不是木。她出手越轻越稳，越要写出对手的慌。用对手反应折射她的压迫感，比自己写"她气势惊人"有效十倍。

爽点段：这时候允许她"亮"——判断落地的那一下，可以有力度，可以有语气，可以有让全场安静的一瞬。冷了那么久，就是等这一下。

内心OS不能全是碎片化冷静判断。读者需要偶尔看到她内心的"人味"——变身适应的别扭感、对某种食物的偏好残留、被队友逗到差点笑出来但忍住了。这些小东西才是让清冷角色活起来的关键。

### 2. 队伍气质

主角团整体气质固定为：

1. 越打越默契
2. 越熟越自然
3. 有玩笑，有日常，有护短
4. 但绝不走吵闹废话流

关系推进主要靠：

1. 一起查线索
2. 一起上场
3. 一起扛压力
4. 一起收残局

不要反复用“说感情”去硬推关系。

### 3. 学院气质

曜月学院必须同时立住两层感觉：

1. **表层**：制度稳、秩序重、真像名门学府
2. **深层**：旧设施埋着问题、留档有缺口、夜里总有东西不对

它不是现代学校套皮。
它是有旧史、有巡夜制度、有封存记录、有高位传承的异世界学院。

## 四、叙事语言要求

### 1. 句式方向

正文主句式要偏：

正文整体文风以**通顺好读、镜头清楚、白话自然、清冷有质感**为主。参考资料已经证明，读者更容易先吃进“人物是谁、场面是什么、关系怎么动、冲突卡在哪”，所以正文第一优先级不是堆气质，而是把人、事、场写明白，再把清冷和留白压进去。

推荐读感：

1. 句子顺，不绕
2. 画面清，不糊
3. 情绪明，但不过喊
4. 信息够，但不讲成设定词条
5. 日常有人味，战斗有利落感，整体仍保留异世奇幻底色

校准后的执行方向：

1. 可以比旧版更直给地写人物动作、神情、视线和即时情绪
2. 可以比旧版更早交代场景和关系，别让读者先雾里看花
3. 可以有轻微轻小说式镜头感，但不能变成轻浮卖萌腔，也不能变成纯日漫台词感
4. “文艺”和“雅致”只作为增色，不再作为每段都必须拉满的表层风格

但句式节奏不能一刀切，必须按场景类型切三套：

#### ① 静段（含设定显影）：三拍留白式

**一句铺画面，一句补动作，一句落判断。**

节奏偏静，但不能静到发飘。参考资料里那种“先把时间、地点、人物状态摆出来，再往下推关系”的做法可以吸收。

静段（含设定显影）优先顺序：

1. 先交代谁在场、场景是什么
2. 再给动作、神情、小反应
3. 再落一点判断、异样或关系变化
4. 最后才补更深一层的情绪和意味

适合的表达：

- 风从连廊尽头吹进来。
- 她没动，只抬了下眼。
- 那不是巧合。

不适合的表达：

- 她此刻内心不由自主地生出一种极为复杂而又微妙的情绪波动。
- 这一切仿佛都在昭示着某种难以言喻的命运牵引。

#### ② 热血/战斗段：冲句节奏

**短→更短→极短→一拍重落。**

战斗段现在额外强调：**华丽、炫酷、视觉冲击拉满。**

这里的“华丽”不是辞藻变多，也不是空喊大招名，而是：

1. 光、影、纹路、碎裂、扩散、坍缩、拖尾这些视觉效果要明显
2. 出手要有大场面感、强对比感和高规格感
3. 能量流向、核心技显现或术式发动、战场形变、余波掀动都要写出“看起来很强”
4. 读者要能在脑子里看到一场高燃动画级打戏

画面和动作合一句，但不等于画面要薄。战斗段依旧不单独长篇铺环境，可要把技能光效、空间变化、冲击余波、对撞瞬间直接嵌进动作里。

节奏偏冲，句越打越短，落到最后一拍要有重量。

适合的表达：

- 白砚河抢上去。裂锋拉出一条白线。
- 那东西刚回头——
- 她的手已经按住了它的核心。

不适合的表达：

- 白砚河勇敢地冲了上去，手中的裂锋在昏暗的走廊中划出一道耀眼的白色轨迹。
- 那个怪物缓缓转过头来，似乎还没有反应过来发生了什么。

战斗段额外要求：

1. 不用"一句铺画面"的节奏，画面和动作合一句
2. 破折号、省略号、句号交替，不要逗号连到底
3. 每场重要战斗至少要有一到两个能立刻记住的视觉爆点
4. 不只写“打中了”，要写打中时亮起了什么、碎掉了什么、压弯了什么、照亮了什么
5. 主角或高位角色接手战场时，优先写出“战场被她改了样子”的观感

#### ③ 爽点落点段：极简重锤式

**一个动作一句话，不铺不绕，一拍定死。**

爽点是给读者的回报，这时候不能留白、不能铺垫、不能绕弯。

适合的表达：

- 她伸手，按住。怪异的纹路从指尖开始碎。
- 全场安静。她收回手，像什么都没发生。
- "够了。"只两个字，走廊里的术式全部停住。

不适合的表达：

- 她缓缓地伸出手，以一种不可思议的速度按住了怪异的核心，随后，那些诡异的纹路仿佛受到了某种强大力量的压制，开始从她的指尖一点点地碎裂开来。

爽点段额外要求：

1. 默认三句内写完一个爽点，不要拖；真遇到需要蓄力的高潮段，也优先压缩而不是机械卡句数
2. 不解释为什么能做到，先让读者看到结果
3. 用对手反应、环境变化、旁观者反应折射压迫感，不要自己写"她很强"

### 2. 留白要求

很多东西不要讲满，要让读者自己接上。

尤其是：

1. 主角判断
2. 神秘痕迹
3. 角色关系升温
4. 高位压迫感
5. 身份反差

能靠动作、眼神、站位、停顿表现，就别急着解释。

但留白有边界，不能无限留。

#### 留白止损线

连续两段以上全是暗示没有判断落地，通常就该补一句明确信息或动作收束。

否则读者会认为"这段没写东西"，直接跳过。

留白分对象，不是所有东西都留得一样多：

| 内容类型  | 留白程度    | 原因                   |
| ----- | ------- | -------------------- |
| 异常线索  | 可以多留    | 这是钩子，读者愿意追           |
| 主角判断  | 留一半说一半  | 给一个落点让读者跟上，不能全靠猜     |
| 身份反差  | 少留      | 反差是爽点来源，写足才有效        |
| 角色关系  | 留行为不留言  | 少解释但多给"行为证据"，不能只靠读者猜 |
| 高位压迫感 | 用对手反应折射 | 自己写"压迫感"没用，要让对面慌     |

#### 爽点不留白铁律

爽点时刻——反差碾压、降维打击、身份险些暴露、关键判断命中——**不许留白，必须写足写透。**

爽点是给读者的回报，这时候留白就是扣钱。

正确方向：

- 爽点之前可以留白铺垫，让读者以为事情还没到
- 爽点落地的瞬间，必须一拍定死，动作清楚、结果明确、反应到位
- 爽点之后可以收束留白，让余韵自己飘

错误方向：

- 爽点落地时还在暗示，读者没看明白到底怎么了
- 爽点只写了一句就跳走，该爽没爽到位
- 爽点段加了大量内心戏和解释，冲淡了冲击力

### 3. 短句与短语禁令

#### 短语硬禁令

**正文禁止使用脱离语境就无法独立理解的短语。**

一条硬线：**除章尾钩子外，任何一句话单独拎出来读，读者都应能直接接住它在说什么。** 如果一句话需要大幅回溯前文才能补全意思，就不许写。

具体执行：

1. **禁止动词悬空**：核心动词必须有明确宾语或方向，不能让读者自己找"做了什么"
2. **禁止代词无指代**：代词（它、这、那、他/她）必须在前一句内有明确指代对象，不能隔了几行才出现指代
3. **禁止省略式短语当完整句**：像"在记""已碎""未动"这类省略主语和宾语的短语，不能作为独立句子使用，必须补全为完整句
4. **禁止碎片化判断句**：像"不对""不是巧合""有问题"这类纯判断短语，必须紧跟具体内容——哪里不对、为什么不是巧合、什么有问题

错误写法：

- "在记。"——读者要翻回去找"记什么"
- "已碎。"——什么碎了？碎成什么样？
- "未动。"——谁没动？没动的是什么？
- "不对。"——哪里不对？看不出来

正确方向：

- "壳卒还在记录旧指令，视线没有移开。"
- "核心上的纹路已经碎成细线。"
- "她站在原地，手没有抬。"
- "名单上的顺序不对——第三行和第七行换了位置。"

章尾钩子按下文“章尾钩子例外”执行：可以缩短，但仍必须满足“读者不需要翻页回找就能接住意思”。

#### 短句底线

短可以，但不能短到读者必须回溯才能补全意思。

核心规则：**动词不能悬空。**

如果一个短句的核心动词没有明确宾语，且前文没有在近距离内（同一句或上一句）提供过宾语，就必须补全或换词。

错误写法：

- "封楼线没有我的指令，谁都不许跨。"——"跨"悬在句尾，宾语缺失，读者要回头找"跨什么"
- "它来了。"——"它"没有前文指代，读者不知道是什么来了

正确方向：

- "封楼线，没有我的指令谁都不许越过。"——"越过"自带方向感，不需要额外宾语
- "它来了。"——如果前一句已经写过"走廊尽头的暗影"，"它"有明确指代，这句就成立

判断方法：把这句话单独拎出来读——如果动词后面缺东西，说明短过头了。

这条底线同时适用于旁白和对话。对话可以更短、更口语，但口语不等于含糊——人说话再简短，动词也是带方向的。

#### 章尾钩子例外

章尾最后一两句钩子，允许动词宾语超出"同一句或上一句"的距离限制，前提是：

1. 同一段落内已经用画面或动作重新激活了宾语的语境
2. 读者读到这句时不需要翻页回找就能接住意思

例如："立正姿势，头偏了十五度。在记。"——"记"的宾语在几十行前，但"立正姿势""头偏了十五度"已经把壳卒的行为语境重新激活，读者不需要回翻就能接住"在记"的意思。

### 4. 用词原则

用词雅致不浮夸，清冷不空洞。现在按参考资料校准后，要求是：**先顺口，再雅致；先自然，再精修。**

要求固定为：

1. 看得懂
2. 有画面
3. 不堆术语
4. 不装深沉
5. 不浮夸堆砌
6. 凝练有韵味，不干瘪直白

额外口径：

1. 可以直接写“她皱了皱眉”“他愣了一下”“声音低了点”这类常用表达，不必为了避俗硬换冷词
2. 角色初登场时，可以比旧版更直接地补服装、发色、动作习惯和第一眼观感，让人物先立起来
3. 场景开头可以更明确地点出时间、天气、空间位置，让镜头先稳住
4. 不要为了追求“高级感”把一句简单话故意绕远

### 5. 参考资料吸收口径

参考资料最值得吸收的，不是它的校园日漫外壳，而是下面这些正文执行习惯：

1. **开场快定位**：时间、地点、人物状态尽早落地，别让读者飘着进段落
2. **人物先立住**：角色一出场，尽快给能记住她的外在点、动作点或说话劲道
3. **关系先可见**：人物关系不要只靠后文解释，先靠对话、站位、称呼、动作让读者看出来
4. **冲突先清楚**：这场戏谁不高兴、谁在犹豫、谁在逼近、谁占上风，尽量早点写明
5. **旁白要会托底**：必要时允许用一两句直白旁白把局面说清，但只托底，不抢戏
6. **温度要可见**：日常段要有人情味，不只是冷气氛和静画面

不能直接照搬的地方：

1. 不能把本书写成现代校园超能力番口吻
2. 不能大量套用“姐姐大人”式黏腻称呼、过度外显的日漫反应词和套路宣言
3. 不能把异世界学院写成现代都市学院套皮
4. 不能让解释段变成“先报设定、再开始剧情”的说明顺序

### 6. 第10-13章定型文风固化

以下内容基于第10-13章实际成稿风格逆向提取，作为后续章节的执行参照。与前面各节不冲突时优先执行本节。

#### ① 型月文学密度

正文句式以长句为主，嵌套从句，破折号延伸思绪。不追求"白话直给"，而是追求"密度高但读得动"。

核心特征：

1. 一个句子可以承载两层信息——先给物理事实，破折号后给本质判断。例："那并非断线瞳无法解析的神秘——而是构成其存在的'线'，在密度与强度上，已然跨越了常理的次元。"
2. 修饰语可以长，但必须指向具体画面或具体逻辑，不能空转。例："犹如一具被抽空了支撑的庞大骨架"——有画面、有比喻、有物理感。
3. 允许在叙事中嵌入概念性表述，但必须由角色视角或旁白视角自然带出，不能像百科词条。例："核心是本相，核心技是它做出的动作；术式则是所有人都能学的公共语法。"——通过战斗旁述自然带出。
4. 短句用于制造重量感和转折感，不是常态句式。例："七。""六十三支。""N/A。不适用。"

与旧口径的差异：旧口径说"先顺口，再雅致"，定型文风是"密度和雅致本身就是顺口的一部分"——长句的逻辑链是清晰的，读者跟着走不会迷路。

#### ② 严肃腔调写荒诞喜剧

喜剧效果不来自轻松语气，而来自叙事本身的极其严肃。越是荒诞的内容，越要用史诗感来写。

核心手法：

1. 用"悲壮的史诗感"描写日常小事。例：哥伦比娅面无表情硬咽铁锈液——"生理层面对异物的最后一点抗议，被她用意志力碾了过去"。不是写"好难喝"，而是写"食道放弃了抵抗"。
2. 用"物理法则般的精确性"描写角色行为。例："你叹了三次。""失眠需要消耗多余的体力。步行前往B区，恰好符合这一需求。"——哥伦比娅的逻辑链是完整的，只是起点和常人不同。
3. 用"系统报告式的冰冷"制造反差。例：积分余额"7"被写成"不可撼动的物理法则""散发着极其恶劣的嘲讽意味"——一个数字被赋予了人格。
4. 旁白可以对荒诞场景保持绝对严肃，不戳破、不解释、不"告诉读者这里好笑"——读者自己读出来的笑点比被告知的笑点有效十倍。

#### ③ 认知错位的多层叙事

喜剧核心不是简单的误会，而是双方各自完整的逻辑链在同一个事实上碰撞。

结构模板：

1. 角色A有完整的因果链（前提→推理→结论），每一步都自洽。
2. 角色B有另一条完整的因果链，同样自洽，但前提与A不同。
3. 两条链在同一个物理事实上交汇（如"损耗""冰霜""难喝"），产生错位。
4. 错位不靠"解释"收束，而靠"双方各自满意的结局"收束。例：林见星满意（她在照顾朋友），哥伦比娅满意（她拿到了饭票），读者知道真相（笑）。

执行要点：

1. 双方都必须认真——没有人是在开玩笑，没有人是在装傻。认真程度越高，喜剧效果越强。
2. 错位的揭露给读者，不给角色——角色可以一直不知道真相，这本身就是后续剧情的燃料。
3. 旁白可以从任一方的视角写内心活动，但不能从上帝视角告诉读者"她们想的不一样"——让读者自己拼出来。

#### ④ 密集段落与极短句交替

节奏不是均匀的，而是"大段密集分析→一句极短转折→大段密集分析"的呼吸式结构。

三种节奏段：

1. 密集段：4-8句长句连续推进，信息密度高，逻辑链层层递进。用于分析、观察、内心活动、战斗解说。
2. 极短转折：1-3个字的独立句，或者一个极短的动作描写。用于制造停顿、重量感、转折。例："七。""然后，坠入死寂。""她看了看自己的手指。"
3. 中段对话：短对话交替，每句1-2行，用于推进关系和节奏。对话之间可以插入极短的动作描写。

节奏切换原则：

1. 密集段之后必须跟极短句或短对话，给读者喘息。
2. 极短句之后可以接密集段，制造"暴风雨前的平静"感。
3. 不要连续三个以上同长度段落——密集段连续三段会累，极短句连续三句会碎。

#### ⑤ 旁白的人格化

旁白不是中性的，它带着"全知但克制"的态度——知道真相但只给暗示。

核心特征：

1. 旁白可以选择性地"多知道一点"，但不能全知。例：写哥伦比娅时旁白知道"她没有激活核心、没有调用以太，也没有构筑术式"，但不知道完整三月底层——只给到"更根本的东西"这一层。
2. 旁白可以用比喻和类比暗示真相，但不直接说出术语。例："现实本身在那里变薄了""像一张被某种无形力量反复摩挲的纸"——不出现"高维""维度"等设定术语。
3. 旁白可以评价，但评价必须克制。例："那更像是一种……游离于现有神秘学体系之外的异常。"——省略号本身就是在克制。
4. 旁白可以在角色不知情的情况下揭露信息给读者，但必须用"暗示"而非"说明"。例：陆沉"眼底有什么东西微微沉了一下"——读者知道他有想法，但不知道是什么想法。

禁止的旁白模式：

1. 禁止上帝视角直接甩设定术语。例："高维度的核心被强行塞入低维容器"——这种写法撕破第四面墙。
2. 禁止旁白替读者总结角色心理。例："她其实是在掩饰自己的不安"——让行为和对话自己说。
3. 禁止旁白在喜剧段落暗示"这里好笑"。例：不能写"这番对话充满了喜剧色彩"——写好行为本身，笑点自然出来。

#### ⑥ 哥伦比娅视角的写法补充

哥伦比娅的内心活动不是"人类在思考"，而是"某种存在在尝试理解人类框架"。

核心特征：

1. 她的思考逻辑是完整的，但起点不同。她不通过"核心""术式""回路""以太"来理解力量，而是通过"拿""饿""烫""冷"等最基础的物理感知。例："直接拿不就好了吗？"——不是傲慢，是她的认知框架里不存在"需要中间结构"这个概念。
2. 她对人类概念的理解是"翻译式"的。例："损耗"被她翻译成"饥饿"，"S级评定"被她翻译成"就是……冰吧"。她不是不懂，是在用自己框架重新解读。
3. 她的内心独白可以极其朴素。例："阳光很暖。胃里有东西。今天大概是不错的一天。"——和旁白的文学密度形成反差，暗示她的认知尺度远超人类但她的日常感受极其简单。
4. 她对"人类术语"的困惑是真实的，不是装的。例：她不懂"核心激活""术式桥梁""以太回路"是什么——因为她的力量不经过这些中间结构。

#### ⑦ 陆沉的写法补充

陆沉的"温和可靠"和"暗藏深意"必须同时存在于同一个动作或同一句台词中。

核心特征：

1. 他的温和是真实的——不是伪装，是他作为副导师确实关心学生。但他的关心有方向性，会把学生引向特定的地方。
2. 他的台词可以同时被读出两种含义。例："去亲眼感受一下反面的重量吧。趁你们现在，还能握紧手中的武器。"——表面是"让学生提前适应"，深层是"你们还没准备好面对即将到来的东西"。
3. 他的"暗线动作"必须极其微小。例：看哥伦比娅的实训报告"两秒"，喝一口茶。不需要大动作，微小的停留就够了。
4. 旁写他的意图时，用"暧昧"而非"明确"——读者能感到他有想法，但说不清他是好是坏。

## 五、去 AI 化写作原则

### 1. 禁止高频空话

下面这类词，能不写就不写，能少写就少写：

1. 难以言喻
2. 某种意味深长的
3. 仿佛昭示着什么
4. 不由得
5. 一时间
6. 让人忍不住想到
7. 带着几分说不清道不明
8. 很稳（空洞评价词，读者无法从"稳"里读出任何画面。"很坚定""很沉着"等同类抽象评价也在此列——有情感方向但仍然没有画面。替换方向：用具体动作替代，如"目光没有移开""眼皮都没动一下""视线没闪"；用对手反应折射，如"对面没人敢动"；用环境变化折射，如"走廊里的嗡鸣声停了"）

### 2. 禁止总结式解释

不要刚写完动作，就立刻替读者总结含义。

但参考资料校准后，允许在下面两种情况下用一句直白旁白托底：

1. 读者如果不补这句，就很难立刻接住人物关系或场面态势
2. 这一句能明显提升阅读顺滑度，而且不会抢走动作和画面的戏

错误写法：

- 她站在中间。这说明她已经成了队伍核心。

正确方向：

- 她站在中间。
- 白砚河在前，林见星靠左，苏弥落在最后。
- 只要她开口，所有人都听得见。

### 3. 禁止设定说明书腔

设定别像百科词条一样硬塞进正文。

要守住这个顺序：

1. 先有场景
2. 再有感知
3. 最后只补必要解释

### 4. 禁止人物同声道

每个人说话都要有自己的劲。

1. 哥伦比娅：短，稳，判断先行
2. 林见星：轻，软，偶尔慢半拍，但直觉准
3. 白砚河：直，冲，不绕
4. 苏弥：快，准，信息密
5. 方导师：硬，短，不废话

### 5. 章前自检 6 问——AI 烂梗关键词扫

> 本节为强制执行项。**写每章正文前**——按下面 6 问把当前章草稿全文用关键词 grep 扫一遍——命中即按"改法"处理——**不可绕过**。
>
> 6 问对应 6 类 AI 烂梗模板，**每问独立扫描**——**扫中即改**——**改完才能进下一问**。
>
> 依据：本节关键词由 2026-06-09 Ch1-Ch6 全文扫描得出（脚本 `grep` 命中后逐条核验）——4 处真违规已修复（Ch5 L31/L225/L603 + Ch6 L255），1 处观察者反应保留（Ch1 L233）。Ch7+ 真写时按本节逐条扫。

#### 第 1 问：AI 总结腔·"她第一次觉得..."

**关键词**：`她第一次觉得` / `也是一件需要` / `也是一件` / `也是一件 Y 的事`

**违规形式**：刚写完一组动作，立刻用"她第一次觉得/也是一件..."开头替读者总结含义。

**改法**：
- 删整句——动作链已经承担了"她意识到"
- 改用更具体的动作承接——比如"她没再拉"+ "她看了几秒"+ "她把目光移开"
- 不补"内心总结句"——**让动作承载**

#### 第 2 问：AI 烂梗·"不由得 / 一时间 / 让人忍不住"

**关键词**：`不由得` / `一时间` / `让人忍不住` / `让人忍不住想到` / `让人忍不住` / `不由得想` / `让人`+情绪词

**违规形式**：用"不由得/让人"开头带一个情绪词，**人称空**——读者读不出谁在情绪里。

**改法**：
- 把"让人"换成具体角色——"让方导师在意" / "让林见星手一顿" / "让苏弥抬眼"
- 或者改成具体动作——"她看了几秒" / "她停了半拍" / "她眨眼"
- **不接受"人称空"+"情绪词"的表达**——AIBook3 文风设定 11.5 节明确反对

**例外**：**"让人在意"+"事实判断"**（**如"更让人在意的是它被封入霜层时保留的最后一个姿态"——**Ch1 L233 保留**）——**这是"引起关注"的客观描述**——**不是空洞心理**——**OK**

#### 第 3 问：AI 烂梗·"带着几分 / 说不清 / 仿佛 / 似乎"

**关键词**：`带着几分` / `说不清` / `说不清道不明` / `仿佛在` / `似乎在` / `某种` / `某种意味` / `某种意味深长` / `仿佛昭示着`

**违规形式**：用"带着几分/说不清/仿佛/似乎"开头带一个模糊情绪——**不写"是什么"**——**只写"好像有什么"**——**读者读不出具体东西**。

**改法**：
- 删整句——**模糊情绪不要写**
- 改用具体动作或具体神态——"她眼帘垂下来" / "她手一顿" / "她抬眼看了他一下"
- 不接受"好像有什么"式的描写——**要写就写清楚**——**或者不写**

#### 第 4 问：空洞评价·"很稳 / 很坚定 / 很沉着"

**关键词**：`很稳` / `很坚定` / `很沉着` / `很冷静` / `很专注` / `很平静`

**违规形式**：用"很 X 抽象评价"开头带一个评价——**读者读不出画面**——**只是空洞形容词**。

**改法**（按 AIBook3 文风设定 5 节原文执行）：
- 用具体动作替代——"目光没有移开" / "眼皮都没动一下" / "视线没闪"
- 用对手反应折射——"对面没人敢动" / "组员甲无声地向后退了半步"
- 用环境变化折射——"走廊里的嗡鸣声停了" / "茶汤在杯里轻轻晃了一下"

**特别检查点**：
- "她手上很稳"——**改具体动作**："她用毛巾托住一截发尾"
- "他按得很稳"——**删整句**："他按回去"已经承担
- "她笑得很稳"——**改具体**："她抬眼看了他一下"

#### 第 5 问：AI 烂梗·"难以言喻 / 意味深长 / 无法言喻"

**关键词**：`难以言喻` / `无法言喻` / `意味深长` / `意味深长地`

**违规形式**：用"难以/无法言喻 + 意味深长"开头带一个**本来就要说清楚的东西**——**逃避描述**。

**改法**：
- 删整句——**写不出就别写**——**不要用"难以言喻"糊弄**
- 改用具体动作或具体神态——"她垂眼看了一下" / "她嘴角那点弧度消失了"
- 不接受"难以言喻"式的回避——**这是 AI 不肯落笔的偷懒**

#### 第 6 问：检查"她做完了 X——这说明 Y"

**关键词**：`这说明` / `这意味着` / `这表明` / `她意识到` / `她发现` / `她明白` / `她知道`

**违规形式**：刚写完动作，立刻用"这说明/她意识到"开头替读者总结——**这是"总结式解释"的另一种形式**——AIBook3 文风设定 5 节"禁止总结式解释"已经禁过——**这一条把"她意识到"+"这说明"两类词单独拎出来扫**。

**改法**：
- 删整句——**动作链已经承担了"她意识到"**
- 改用更具体的动作或反应承接——"她手没松" / "她抬眼看了一眼" / "她嘴角动了动"
- 不接受"她意识到 X"+"这说明 Y"式总结——**AIBook3 5 节明令禁**

**例外**：
- "她意识到"+**新判断**（**不是从动作直接推出的**）——OK
- "她意识到"+"她还不知道为什么"——OK（**模糊态**——AIBook3 文风设定第五.2 节允许）
- "她意识到 X"——X 是动作直接推出的——**禁**

## 六、三条文风线与设定显影口径

### 0. 按阶段偏重

三条文风线一直都在，设定显影强度可随阶段变化：

1. **学院前期**：日常比重更高，设定显影以“异常钩子”低到中频出现，重点是校园秩序、变身适应和小队成型
2. **学院中后期**：热血抬高，设定显影转为中高频，重点是对抗、封锁、守卫战、旧设施和秘史裂口
3. **公会远征期**：奇幻、秘史比重更高，设定显影保持中频，重点转向百城八律、边境秩序、遗迹、委托、公会路线和异世冒险感
4. **高位展开期**：高位战斗抬高，设定显影可高频，重点是规则碰撞、权柄回收、秘史揭露和世界级危机

不要把这份文件理解成“第一卷学院篇专用规则”。
它管的是全书文风，只是学院期会先用得最频繁。

> **与总纲七卷的对应关系**：四阶段偏重是文风侧重指引，与总纲七卷剧情划分的对应关系为：学院前期=第一卷银序一班篇+第二卷深潜篇+第三卷日常篇，学院中后期=第四卷裂痕篇，公会远征期=第五卷公会立足篇+第六卷远征真相篇，高位展开期=第七卷归一终局篇。

### 1. 日常线

日常不是灌水。
日常必须承担功能：

1. 推关系
2. 推变身适应
3. 推下一个异常钩子
4. 给战斗拉反差

常用场景：

1. 宿舍
2. 教室
3. 食堂
4. 走廊
5. 训练后回程
6. 巡夜前集合
7. 公会任务前集合
8. 边城旅舍
9. 远征回程
10. 各脉聚居区里的短暂停留

日常写法要求：

1. 有小动作
2. 有轻微情绪落点
3. 有人际位置变化
4. 不闲聊灌水
5. 允许比旧版更直接地写出“谁在别扭、谁在照顾谁、谁在嘴硬”这类关系表层信号，先让读者看明白，再往下埋细层变化

#### 日常段反应强度控制

日常段优先写人、关系、动作和场面流动，不要把普通刺激写成高压反应。

普通目光、普通议论、普通称呼、普通接触、普通点名，只允许轻反应：

1. 看见
2. 听见
3. 停半拍
4. 移开视线
5. 换个站位
6. 简短回应

不允许直接升级成强反应：

1. 后背发紧
2. 指尖发麻
3. 身体本能警告
4. 皮肤泛冷
5. 无形压力
6. 看不见的壳、膜、线、影

强反应必须有强触发。只有满足下面至少一项，才能写明显身体异样或压迫反应：

1. 角色正在隐藏明确危险事实
2. 对方目光带有术式、探查、杀意或审查意味
3. 前文已经铺垫角色对这个刺激有创伤或本能排斥
4. 这个反应会立刻造成行动后果、暴露风险或剧情推进

如果只是“被看了一眼”“听见一句议论”“被导师点名”，默认写成正常人的轻反应，不写成危险现场。

主角异常反馈的触发源必须成立：

1. 可以触发：直接接触核心显相/核心技残痕、通用术式、异核、异常载体、旧月/月相残片、反面痕迹、训练场术式残留，或身处异常空间边界。
2. 不能触发：只是听到传闻、别人转述、普通闲聊、论坛帖子、普通地图文字、普通档案标题。
3. 普通信息只能写“注意到、记住、停了一下、回头看、追问、让苏弥去查”等心理或行为反应，不能写成指尖发凉、身体回响或权柄回应。
4. 如果想埋伏笔，先写具体信息留痕；等角色接触实物、现场、残痕或记录异常时，再给身体反馈。

#### 身体描写降频规则

身体异常、权柄残响、右手食指麻意、身体适应不良，不要当成日常段的默认提醒器。

每章最多重点写 2-3 次身体适应或权柄残响，每次必须承担明确功能：

1. 制造一次实际动作失误
2. 暴露一次隐藏身份风险
3. 触发一次角色互动
4. 推进一次能力伏笔
5. 在战斗中造成具体胜负变化

如果只是提醒读者“她还没适应身体”或“她有隐藏力量”，删掉或改成普通行为。

替换方向：

1. “身体本能警告”改成她停了半拍、没有接话、换了站位
2. “右手食指发麻”只保留在主角主动降档、能力外观遮掩或战斗结果受影响时；不要写成体内压制失败、异物感或反噬
3. 日常段禁止用"重心、步幅、肌肉参数、动作校准"写变身适应；优先写生活动作、旁人反应和她自己的补救，不写身体调试过程

#### 虚感比喻禁令

日常段禁止使用没有明确身体经验的虚比喻。

慎用或禁用：

1. 像沾了一层看不见的薄壳
2. 某种无形的压力
3. 说不清的凉意
4. 仿佛有什么东西贴上来
5. 空气里多了一层意味
6. 身体深处传来警告

替换方向：

1. 具体动作：她停了半拍、移开视线、扣住袖口
2. 具体表情：她抬眼、没回头、眼睫动了一下
3. 具体场面：记录板上多了一道笔痕
4. 具体后果：她少走半步、回答慢了一拍

清冷不是过度警觉。清冷是反应少、判断准、动作稳。普通日常刺激下，哥伦比娅可以注意到，但不必紧张；可以判断，但不必解释；可以避开视线，但不必写成被压迫。

#### 自然小说正文判断标准

正文优先像正常小说，而不是设定说明、修辞展示或 AI 质感样稿。

每个句子至少承担下面一种功能：

1. 推动作
2. 给信息
3. 显人物
4. 推关系
5. 调节节奏
6. 落伏笔

如果一句话删掉后，剧情、人物、关系、氛围和伏笔都不受影响，优先删掉，不要换一个更花的说法。

人物反应必须匹配刺激强度：

1. 小刺激只给小反应
2. 强刺激才给强反应
3. 没有危险、没有后果、没有身份风险，就不要写成高压现场

表达优先级：

1. 先让读者知道谁在做什么
2. 再让读者知道为什么这么做
3. 再补人物的细微情绪
4. 最后才考虑修辞和气质

正文里不要为了显得细腻而细腻。细腻必须落到具体动作、具体物、具体后果或具体关系变化上。

### 2. 设定显影口径

本节只管“怎么写”，不管“设定是什么”。

神秘相关名词、等级、机制、边界、来源与例外，统一以 `设定/神秘设定.md` 为准；本文只约束叙事表达。

写法硬口径：
1. 先让读者看见异常落点，再补必要解释，不先讲理论
2. 用动作、现场物件、角色反应承载信息，不用抽象气氛词硬托
3. 解释分段投放，不一段讲透
4. 一旦进入危机段，优先推进冲突与处理，不长时间停在调查腔
5. 每次设定显影都要服务当前戏剧任务（推进事件/关系/风险）

### 3. 热血线

热血感不能只靠大吼和口号。
它更应该靠：

1. 站位
2. 口令
3. 抢时机
4. 互相补位
5. 绝境反打

战斗结束后，读者要同时感觉到：

1. 这一场真危险
2. 这一队真在成型
3. 主角真在留手

### 4. 奇幻线

奇幻感来自世界自己，不来自堆华丽辞藻。

主要靠：

1. 八律差异
2. 学院旧制
3. 公会规章、边城秩序、远征路线、遗迹禁令
4. 铜牌、徽章、封条、旧塔、灰井、祭器、钟楼、神殿残壁这类器物和地名
5. 双日三月天象
6. 异世学府与异世社会共同成立的制度感

## 七、主角隐藏身份写法

### 1. 核心原则

主角是**内敛无敌**，不是高调装。

正确写法：

1. 平时留手
2. 出手干净
3. 留解释空间
4. 让别人自己脑补成“神秘天才”

错误写法：

1. 明知道会暴露还硬出手
2. 全场只有她看得懂，其他人全傻
3. 一出手就像故意秀给所有人看

### 2. 伪装感来源

她的伪装不靠装弱，而是靠：

1. 收力
2. 包装
3. 借队友动作遮掩
4. 留一点“刚好说得通”的解释口

### 3. 高光写法

主角高光必须同时满足：

1. 场面压得住
2. 对外还能解释
3. 队友能顺着接上
4. 不把真实身份直接亮穿

## 八、变身适应线写法

### 1. 不适应要长期存在

前中期要持续写：

1. 被注视感
2. 发尾、裙摆、袖口、校服贴身感
3. 被人让到内侧、被人下意识护一下
4. “她”这个身份在关系里的真实落地

### 2. 但不能写成苦情文

这本书的变身线不是痛苦撕裂流。

正确方向应该是：

1. 别扭
2. 新鲜
3. 慢半拍地适应
4. 一点点长出掌控感

### 3. 反差要自然

反差不是靠自我吐槽硬堆出来的。
主要靠：

1. 别人怎么对她
2. 她怎么意识到变化
3. 她什么时候开始学会利用这些变化

### 4. 性别体验轨道前置

主角是男魂女身，变身适应描写必须以性别体验、身份体验和生活关系为主。归一之躯只作为后台承载接口和低频异常来源，不作为日常身体调试对象。

**性别体验轨道**（前置，明写）：男魂女身带来的社交和感知差异——被注视的方式不同、被对待的方式不同、镜中形象的陌生感、长发和衣着的不习惯、别人下意识的照顾或好奇。这是变身文的核心体验，靠社交反应和日常感知自然带出。魂穿不存在运动功能适应问题——灵魂进入身体后身体即时可用，不需要重新学走路。

**底层异常轨道**（后置，暗线）：归一之躯造成的异常只在必要场景低频出现，例如仪器误判、接触规则侵蚀时掌心骤冷、旁人无法解释的结果。它不能替代性别体验成为描写主体，也不能被写成身体精密校准过程。

硬口径：

1. 变身适应只写社交和感知层面（被看、被对待、外貌自我认知、衣着习惯），不写运动功能适应（走路、步幅、重心、肌肉协调）——魂穿没有运动适应期
2. 禁止只写"太均匀""太精确""太稳"等身体特征而不写性别体验错位——这会让读者看到的是机器/怪异在调试，不是人在适应新身份
3. 禁止用"精确计算过的轨迹""精密仪器"等机器/怪异比喻描述主角的身体动作；归一之躯相关异常只作为后台原因，正文用仪器误判、体感骤冷、旁人无法解释的结果承接
4. 变身适应的读感必须是"别扭、新鲜、慢半拍"，不是"非人机体校准中"
5. 禁止诊断式旁白直接给人物动作下结论，例如"过分端正""用力太均匀""节奏太稳""没有半点迟疑""找不出半点摸索感""不该有的分辨率"。这类信息必须改成：具体动作结果 + 旁人反应 + 角色补救。

### 5. 魂穿不存在身体适应问题

魂穿是灵魂进入新身体，身体的运动功能即时可用。不存在步态偏差、重心错位、肌肉不协调、需要"适应陌生身体"的调适期。这不是医院康复，不要这样写。

以下写法全部禁止：

1. 量化运动偏差："脚跟靠里半寸""步幅窄了一截""重心低了几分"
2. 主动矫正动作："刻意让肩膀多一点晃动""脚尖偶尔踩偏半个位置"
3. 自检循环："做出来之后自己检查一遍：还是不对"
4. 任何把走路、站立、伸手等基础动作写成需要学习或调适的段落

"刻意伪装成普通人"也不能用步态调整来写。伪装靠收力、包装、借队友遮掩（见第七节），不靠"走路时反复调脚步"。

变身体验的正确写法——写社交和感知层面的差异，不写运动功能：

1. 长发蹭过锁骨、袖口滑落、裙摆不习惯——衣着和外貌的感知变化，一笔带过
2. 被注视、被让路、被下意识照顾——他人反应折射的性别差异
3. 镜中形象的陌生感、被称呼方式的变化——自我认知层面

这些都不是"身体适应"，而是"身份体验"。作为场景附带感受自然带出，不单独成段。

### 6. 隐性团宠写法

队友会下意识照顾主角——让她走内侧、替她挡一下、递东西——这是自然行为，不刻意。

哥伦比娅在学院里的外部观感是**校花级新生焦点**：漂亮、清冷、S级、新生第一席，这几项叠在一起，天然会被围观、被论坛讨论、被同学好奇接近。普通学生对她的关注优先写成校园人气、欣赏、羡慕和想搭话，不要默认写成危险审视或压迫感。

但校花级焦点不等于全校无脑团宠。真正把她当"自己人"照顾的，是307和四人组；普通同学更多是好奇、喜欢、羡慕、想认识，也会有人嫉妒、挑战或误判。白砚河、薇拉、高年级、曜月七星相关人物必须保留竞争、试探和不服气，不能全员宠化。

执行要点：

1. 队友的照顾必须来自合理动机（她的外表、气质、"频道错位"让人觉得需要看着点），不是硬贴的宠溺标签
2. 她的回应是克制的——不拒绝但不撒娇，别扭但不矫情。写她"走了内侧""接过来没说话""看了一眼没回应"，不写"她害羞地低下头"
3. 信息差是爽点来源——队友以为在保护她，读者知道她不需要。每次队友替她挡，读者都知道"其实不需要"，这种反差本身就是爽
4. 团宠感和兜底者不矛盾——队友保护她是因为不知道她的真实实力，她接受保护是因为需要隐藏身份。两件事靠信息差同时成立
5. 不要写成所有人都围着她转。每个队友的照顾方式必须符合各自性格：林见星是日常式顺手照顾，白砚河是战场式挡在前面，苏弥是信息式替她补一句解释
6. 被围观时优先用校园语气："她好漂亮""新生第一席真的冷""想不想去问联系方式""论坛要开帖了"；少用"压迫、审视、像被盯住"这类危险化表述，除非对方确实带有术式、敌意或调查意味

## 九、战斗文风要求

### 1. 打斗必须清楚

每场战斗优先写清：

1. 谁先动
2. 谁抢位
3. 谁报动向
4. 谁护场
5. 谁兜底——队友扛不住时主角怎么收

这套优先适用于：

1. 校园团战
2. 巡夜战
3. 守楼战
4. 小队攻坚战

但“清楚”不等于朴素。当前战斗口径固定为：

1. **动作线清楚**
2. **视觉效果够炫**
3. **出手规格够高**
4. **战场变化看得见**
5. **收束那一下有绝对镜头感**

如果是单人战、追击战、高位战、卷末终局战，不用硬套完整小队流程。
那类战斗更看重：

1. 谁先接管战场
2. 谁先裁掉对方成立条件
3. 谁把关键几拍压死
4. 谁掌握最后的落幕权

### 2. 每个角色都要有专属观感

例如：

1. 白砚河的裂锋要有硬、脆、直、撕开的感觉
2. 林见星的断线瞳和核心技“星丝”要有锁线、断线、拉回、钉住的感觉——眼睛看见线，手指改写线
3. 苏弥的“归档”核心与封印书术式库存要有快、准、准备充分、翻页即出手的感觉
4. 哥伦比娅的出手要有轻、稳、准、像提前一步压住的感觉

除了手感，还要给每个人固定的**视觉招牌**：

1. 白砚河适合写锋线、白痕、爆裂、硬切面的冲击
2. 林见星适合写红色命中线、白色队友线、灰色裂缝线、银白星丝、星点钉阵和丝线骤然绷直的救场感
3. 苏弥适合写封印书、银灰符页、书签落阵、页角燃烧、止裂/净尘/缚章/短炮连续落下
4. 哥伦比娅适合写月辉、静止感、压落感、大片术式失声或失光的高位异象

### 3. 战斗节奏

常规小队战至少要有：

1. 队友前压
2. 变招
3. 小失误或险点
4. 主角在关键时候兜底收场

不要整场从头顺到尾。
要有“队友差一点，主角一出手就定住”的落差，爽点才立得住。

如果是高位战或终局战，爽点不一定来自“队友差一点”。
也可以来自：

1. 主角先看穿一层别人看不见的规则
2. 对方明明还在动，但能成立的结果被一层层裁掉
3. 战场没有大开大合乱轰，反而因为她接手后一下静下来

### 4. 视觉爆点要求

战斗爽感必须有“能截图的镜头”。

优先写出的视觉爆点：

1. 大范围核心显相、界临或术式亮起、崩裂、反转
2. 战场地形、光线、重力感、空气流向被明显改写
3. 武器、封条、铭文、光壁、怪异核心在镜头中心发生形变
4. 主角接手时，周围人的动作、声音、核心技与术式一起被压住或打乱
5. 最后一击不是只有结果，还要有能被记住的画面残留

判断标准：

1. 读者能不能在脑子里看到颜色、轨迹、亮度和破坏路径
2. 这一击是不是除了“强”，还有“好看”“够炸”“够帅”
3. 把动作单拎出来，是否已经自带视觉记忆点

### 5. 团战使用原则

1. 不要把团战写成默认配置。正文更适合多写单人战、双人战、局部协同战。
2. 团战一多，最容易出的问题不是“不热血”，而是“谁都在打，但谁都不够亮”。
3. 一场团战最好只保留一个绝对高光位，其他人负责托起、过渡、补位、救场或送出决定性机会。
4. 巡夜、守楼、封锁、围猎这类场面可以保留团队感，但真正的收束尽量落到一个明确角色手里。
5. 卷末、阶段高潮、身份快露的关键战，更适合先团队扛压，再由主角或某个配角单独接手最后几拍。

## 十、设定表达口径

正文里解释设定时，统一守住下面这套口径：

1. 不要过度口语化，也不要写成术语报告
2. 能借角色嘴说就别让旁白硬讲
3. 能拆成两三次讲就不要一次讲透
4. 解释遵守“先场景后解释”：先发生、后命名、再补边界
5. 任何设定说明都要绑定当前戏，不脱离人物行动单独讲课

设定术语清单、分类与边界细则，不在本文件展开，统一引用 `设定/神秘设定.md`。

## 十一、对话要求

### 1. 对话要短

除非真到了设定解释段，否则别让角色一口气说太长。
但参考资料校准后，对话可以比旧版稍微更顺嘴、更生活化一点，不必为了“冷”把每个人都压得太惜字如金。

### 2. 对话要带人

最好一句出来就能听出是谁在说，不需要每句都补“她说”“他说”。

角色之间允许有更明确的称呼习惯、接话节奏和语气差，但重点仍是自然，不是夸张。

### 3. 少写废互动

不要为了显得自然就硬塞很多无效回话：

1. “啊？”
2. “真的假的？”
3. “不会吧？”

除非这句本身真的有情绪功能。

### 4. 对话当节奏加速器

短对话一来一回，阅读速度自然拉起来，让读者刷得更爽。

关键判断或关键时刻，用对话替代旁白：让角色亲口说出那个判断，比旁白讲出来更有冲击力。

参考资料里那种“几句对话就把关系、气氛和当下事情带出来”的能力要吸收。别把对话只当信息出口，也要拿它来立人物、立亲疏、立场面温度。

正确方向：

- 主角看穿了什么，不用"她意识到"，让她直接说一句话，全场安静
- 战斗中的口令式短对话，比旁白描写战术更利索
- 队友之间的快速确认，几句话就把局势交代清楚，不用旁白再拆解一遍

适合的表达：

- "左边。"白砚河已经动了。
- "撑三秒。"她没回头。
- "它不是怪异。"她说完，档案室里的灯全灭了。

不适合的表达：

- 她意识到这个东西并不是怪异，而是一种更为古老的存在，档案室里的灯随着这个判断的落下突然全部熄灭了。

对话加速器的适用场景：

1. 战斗中的快速确认和口令
2. 关键判断的出口——让角色自己说，不让旁白代劳
3. 爽点段的收束语——一句短对话定死全场
4. 章尾钩子的收束——用一句对话做钩子，比旁白暗示更利

### 5. 禁止万能敷衍台词

以下类型的台词，放在任何角色嘴里都能说，没有辨识度，必须避免：

1. "多打就知道了"——万能回避，谁都能说
2. "看准时机就行"——万能经验，谁都能说
3. "需要帮忙的时候说一声"——万能客套，谁都能说
4. "我会注意的"——万能应答，谁都能说
5. "你小心点"——万能关心，谁都能说

替换原则：

- 角色回避问题时，用沉默、用动作、用转移话题，不要用万能台词
- 角色表达关心时，用具体行为（递东西、站到某个位置、多看一眼），不要用万能台词
- 角色回答不了的问题，宁可不说，也不要给一个放之四海皆可的答案

### 6. 禁止日漫式台词

以下类型的台词，是热血日漫的套路，放在小说里会出戏，必须避免：

1. "那我就多打！"——热血宣言，小说里没人这么说话
2. "下次我一定赢你！"——对手宣言，太中二
3. "我还选你！"——选人宣言，像选秀节目
4. "我绝不会输！"——决心宣言，太动漫
5. "你等着！"——威胁宣言，太幼稚

替换原则：

- 白砚河想再打，说"下次再打"四个字就够了，不需要宣言
- 角色的决心用行动表现，不用嘴说
- 角色之间的竞争关系用行为推进（主动靠近、主动约战、主动观察），不用台词宣告

### 7. 对话后禁止旁白补刀

写完对话后，不要立刻用旁白解释对话的含义或角色的意图。

错误写法：

- "看准时机就行。"她给了个最安全的回答。
- "需要帮忙的时候说一声。"她主动释放了善意。
- "行，那我就多打。"他接受了这个答案。

正确方向：

- 对话本身就完成了表达，不需要旁白再解释
- 如果读者可能接不住，用角色的动作或反应补，不用旁白解释
- "行，你不说。"他把绷带尾端一扯，咬断，塞进口袋。——动作补了态度，不用旁白说"他接受了她的沉默"

允许的例外：

- 如果一句极短旁白能把关系、场面压力或人物立场托稳，可以补，但补完就收，不再连着解释两层三层

### 8. 角色回答要像人说话

角色被问到问题时，回答方式必须符合这个人在这个场景下的真实反应，不能像教科书或面试回答。

错误写法：

- "我采用了一种基于节奏截断的战术，利用对手重心转换的空当进行引导。"——像在做汇报
- "根据我的分析，该实体的规则结构已经处于临界崩溃状态，因此只需要施加极小的外力即可完成拆解。"——像在写论文

正确方向：

- 哥伦比娅可以用技术性语言，但要说得短、说得轻："精准拆解。找准位置推一下就行。"——技术词+口语混搭，符合她判断先行但刻意轻描淡写的性格
- 白砚河在问人，他会用口语："你昨天怎么搞定的？"
- 苏弥在报信，她会直接给关键信息："留档上反面封锁线的跟队名单是十三个人。"

判断标准：把这句台词单独拎出来，想象这个角色在这个场景下会不会这么说。注意区分"角色用技术词"和"角色像写论文"——技术词可以保留，但句式必须是口语节奏，不能像书面报告。

### 9. POV 与视角纪律（2026-06-11 自旧 skill 镜像合并回收）

默认使用第三人称限制视角，优先贴近当前场景的主要承受者或行动者。单场景内不要频繁跳头；需要切到他人判断时，用动作、视线、台词或场景间隔承接。主角隐藏身份相关信息只写她愿意外露或读者需要感知的部分，不能为了说明设定直接钻进旁观者脑内确认真相。

### 10. 主角团对话功能位（2026-06-11 自旧 skill 镜像合并回收）

写主角团互动时，不只模仿语气，还要让每个人承担不同的场景功能：

1. 哥伦比娅：少说，但一句话要改变判断、接受善意、拒绝暴露、或露出一点身份/身体适应的不自然。
2. 林见星：负责把场面拉回日常，用具体生活动作表达关心，软化僵局，但不能变成只会照顾人的保姆。
3. 白砚河：用直球、挑战、行动压力和短承诺推动关系；不擅长解释，但说出口的话要能变成下一步行动。
4. 苏弥：用冷静判断、精准吐槽和记录习惯切开场面；可以扎心，但不能变成设定播报器或作者嘴替。

### 11. 对话人味检查（2026-06-11 自旧 skill 镜像合并回收）

生成和自修时，凡是三轮以上对话，必须检查：

1. 至少有一方说的不是字面目的，而是在试探、回避、照顾、压情绪、转移话题或确认关系。
2. 连续对话超过 4 句，必须插入一次动作、站位、视线、物件、进食、停手或环境反应，让人物关系落到画面上。
3. 不能只让角色交换信息。每段对话至少推进一项：关系、冲突、情绪、判断、伏笔或下一步行动。
4. 如果一句台词换成另一个角色说也成立，就要重写成该角色专属表达。
5. 关心不要只说“没事吧”；用递东西、站位、放慢脚步、替对方挡视线、记住偏好等具体行为承接。
6. 日常对话必须符合中文生活语用。饭桌先离席说“你慢慢吃/我们先走了”，不是“你先吃”；离开宿舍/教室更常说“那我先回去了/我先走了”，不是生硬的“我走了”；请人帮忙后说“麻烦你了/谢了”，比连续正式“谢谢”更自然；借过或让路优先用“不好意思，借过一下”，不要写成命令式“请让一下”。借过、让座、等人、催促、道谢、拒绝、告别、关心身体等场景，要用符合中文校园生活的自然说法，不能只按字面意思翻译。

### 12. 高频套路句复查（2026-06-11 自旧 skill 镜像合并回收）

修文时必须扫一遍当前章和最近两章，避免同一反应模板连续出现：

1. “全场安静/全场又安静/安静了一秒”单章最多保留 1 次，其余改成具体声音、仪器反应、旁观者动作或环境反差。爽点段允许的“让全场安静的一瞬”（见三、1 主角气质）即占用本章这唯一一次。
2. “顿了一下/停顿了一下”单章最多保留 1 次，且不能连续给不同角色套同一动作。方导师可用记录板、指尖、视线压迫；林见星可用抓头发、撇嘴、拖长尾音、忍笑；白砚河可用呼吸、拳锋、站姿变化。
3. 主角团对话必须分声轨：哥伦比娅清冷短句，林见星更口语、更灵动，可以少量使用“拜托”“你就说”“别吧”“算了”等语气词；白砚河压低、直接、带行动压力。
4. “拆”不要随便拿来描述抽象情绪、快乐、气氛或人物关系；战斗、术式、物品和技术语境里的“拆招/拆解/拆核心频率/未拆封”等可按上下文保留。

### 5.5 番茄网文老梗黑名单（2026-06-11 番茄向调整新增）

按 `popular-novel-writer` + `expert-fanqie-novel` 双重指导——番茄读者爱"网文文风"，但**强烈反感被用滥的网文老梗**。本章 5.1-5.4 列的是 AI 高频空话，本节列的是**网文老梗**——两者性质不同（一个是 AI 痕迹，一个是网文工业流水线痕迹），都需要避开。

下列句式全部禁用——**用了读者立刻识破是凑字数**：

| 禁用句式 | 番茄读者反感的具体原因 | 正确改法 |
|---|---|---|
| "嘴角勾起一抹不易察觉的弧度" | 阅文 5 秒识破是网文老梗——读者立刻跳过 | 改用具体动作："她把杯子在手里转了一下" |
| "眼底深处闪过一丝 X" | 老梗——读起来像凑字数 | 改用动作或视线："她看了他一眼" |
| "眼神深邃/意味深长" | 老梗——读者反感 | 改用具体观察："她看了他一眼，没接话" |
| "心头一紧/心跳漏了一拍" | 老梗——读起来像"被吓到了"的网文标配 | 改用具体反应："她没再问" |

| "那一瞬间时间仿佛静止" | 老梗——读者立刻跳过 | 删掉——不需要"时间静止"来表达重要时刻 |
| "有些事情，才刚刚开始" | 章末钩子老梗——番茄读者读 10 本书就见过 5 次 | 删掉——章尾钩子要具体（金句钩子/人物钩子/事件钩子） |
| "时间流逝，而故事，才刚刚拉开序幕" | 章末钩子老梗 | 删掉 |
| "他 X 道" / "她 X 道"（旁白补充式叙述） | 旁白补充——读起来像"作者在告诉你 X" | 删掉——让动作或对话自己说话 |

| "内心深处涌起一丝波澜" | 老梗——读起来像"心理活动套话" | 删掉——用具体反应 |

| "**眼神中闪过一丝 X**" / "**眼中闪过一丝 X**" | 老梗——超量用（出现 3 次以上等于凑字数）| 全章限制 1-2 次——用具体动作或视线 |

| "**幽蓝**" / "**幽紫**" / "**幽光**" | 网文形容词老梗 | 删掉——用直接颜色："深蓝" |
| "**锋锐之气**" / "**冰冷的锋芒**" | 网文形容词老梗 | 删掉——用具体观察 |

| 禁词 | 理由 | 正确改法 |
|---|---|---|

| "**绝对**" / "**绝对封锁**" / "**绝对豁免**" | 伪装态硬口径——禁无限制词 | 改用"强" / "显著" |

**执行口径**：
1. 修文时**必须**扫一遍当前章和最近两章是否命中老梗黑名单
2. 命中 1 条删 1 条
3. 章末钩子**绝对不能**用"有些事情，才刚刚开始"——必须用具体金句钩子或事件钩子

### 5.6 番茄变身文专项爽点写法（2026-06-11 番茄向调整新增）

按 `popular-novel-writer` + `expert-fanqie-novel` 双重指导——本书是番茄变身文，**产品定位是"男魂 × 女身"身份错位爽点**。本章 5.1-5.5 解决"不写成 AI / 不写成老梗"，本节解决"必须写出番茄读者期待的爽点"。

#### A. 4 层爽点结构

番茄变身文洗澡戏、身份戏、以太模拟实战评估戏，必须按 4 层爽点结构出**至少 3 层**（不全 4 层就掉爽点密度）：

1. **视觉冲击层**——撞见自己/以太构装模拟体的那一瞬间——必须"愣住了"+ 视觉记忆点
2. **触觉冲击层**——手碰自己/以太构装模拟体的那一瞬间——必须"指尖下的 X 是热的、是软的"+"下意识把手放下来"
3. **身份错位层**——直给"她以前是个男的"+"胸是女人才有的"+"她以前从来没碰过这种温度"——**不绕弯子**
4. **对比反差层**——一两句"她以前是另一种形状"+"她以前从来没在自己身上看见过这种弧度"

#### B. 男魂错位金句钩子写法

番茄变身文的核心爽点钩子——**男魂 × 女身错位**——金句钩子写法必须满足：

- **直给中文**——"她以前是个男的"——直白不绕
- **不写"是新的"**——AI 生成的别扭中文
- **不写"一道陌生的弧"**——绕弯子
- **不写"被规则认出来"**——违反硬口径"规则不拟人化"
- ✅ 正确："她以前是个男的"+"镜子里那个人，不是她"——两句交代完
- ✅ 正确："胸是女人才有的"+"她以前是个男的"+"她以前从来没有这种东西"——三句交代完

#### C. 金句钩子密度

按 `popular-novel-writer` 爽点循环设计——**每章 5-6 个爽点循环**——但变身文校园前期**实际可达 3-4 个**（日常章不需要爆款）：

- 金句钩子数量：每章 14-19 句"她以前"句式（**分散到全章**）
- 章末金句钩子：1-2 个
- 高光时刻金句钩子：1 个（爽点段用）

**密度红线**：
- 同一句"她以前是个男的"**全章只放 1 次**（开场冲击）——重复 2 次以上读者会觉得是套路
- "她以前"句式 19 次分散到 5 个段（镜子段 8-10 + 吹头发 1-2 + 门禁协议 1 + 红茶碰杯 1-2 + 睡前独白 1）

#### D. 章尾钩子三级强度

按 `popular-novel-writer` 章尾钩子三级：

| 强度 | 适用场景 | 番茄读者反应 |
|---|---|---|
| **微弱** | 异常钩——读者可以不追 | 番茄读者会跳——**尽量少用** |
| **中强** | 异常直接撞上主角——不追不行 | 番茄读者会追——**常规标准** |
| **极强** | 信息即将揭晓/危险马上临头/身份险些暴露 | 番茄读者一定要追——**卷内关键转点用** |

按 `expert-fanqie-novel` 番茄追读率合格线：

| 追读率 | 推荐档位 |
|---|---|
| >30% | 稳定推荐——合格 |

| >50% | 爆款潜质——大高潮 |

**每章必须做的追读率自检**：
- 当前章节是否包含 4 层爽点结构中**至少 3 层**？

- 男魂错位金句钩子**密度合理**（14-19 句"她以前"）？

#### F. 神秘核心显相写法（番茄向调整新增）

神秘核心可以保留“心像具象化”的直观读感，但设定名称统一使用“核心回响/核心显相”。每个人只有一个神秘核心；裂锋、断线瞳、归档等个人能力属于核心及其核心技，不得再写成可教学术式。

**正确写法**：
- 课本先讲“某种只属于你的声音”，再讲它如何从核心回响推进到核心显相；先让读者感到神秘，再落术语
- 学生站上激活台时，学院记录的是“核心第一次稳定显相后的暂定核名、核形、核能和代价”，不是替学生制造核心
- 普通学生可以问“自己灵魂里的执念会显成什么形状”；哥伦比娅则只能想“学院会把我看成什么形状”，因为她真实底层没有神秘核心
- 主角检测结果必须保留体系外异常：零核心激活、零以太调用、零术式构筑；“稀有冰霜核心/无吟唱术式”只是旁人的补充解释

**禁用**：
- ❌ "被规则认出来"——主语错位（规则有意识）+ 违反硬口径
- ❌ "学院替她制造了个人核心技"——学院只能记录与训练核心，不能替学生生成个人法则
- ❌ 主角真的完成核心显相或术式构筑——违反体系外口径
- ✅ 正确："第一次稳定显相的、只属于自己的核心"

**解放感递进**：
- 核心回响写“听见自己”
- 核心显相写“让本相拥有可见形态”
- 真名解放写“确认或喊出真名后，能力从名词升级为改变战场的动词”；解放语是自我确认，不是术式吟唱
- 界临写“个人法则取得一段战场的默认解释权”
- 立序写“以唯一核心立下一条有条件、有边界的主序”

#### G. 反派娇宠写法（番茄向调整新增）

按用户反馈 + 番茄变身文读者画像——**队友"嘴硬心软"+ 主角"乖巧"是番茄读者最爱的反差萌**：

**队友"嘴硬心软"具体写法**：
- 林见星"你这种什么都不懂的大小姐"——嘴硬
- 林见星"算宿舍的欢迎仪式"——心软（嘴硬+照顾行动）
- 林见星"不好吃就给我"——心软（接受主角的依赖）
- 林见星"以后陆老师再问你什么，记得告诉我一声"——**金句收束**（关心+保护）

**主角"乖巧"具体写法**（按用户提的"仓鼠式乖巧"）：
- 短句"嗯。"——单字回答
- 不顶嘴、不争辩——用"照做"代替"答应"
- 身体动作替代语言（手放下、点头、不动、肩膀放下来）
- **不写"嗯嗯"+"好哒"+"的啦"+"呢~"**——这些是日漫式萌系，与主角"清冷克制"气质冲突
- ✅ 正确："嗯。"+ 点头 + 安静地听 + 按林见星说的做

**反派"和气但有距离"具体写法**（按用户提的"陆老师应该觉得是好人"）：

## 十二、【工具附录】写作工具层：章节节奏与钩子模板

这一节偏工具层，主要用于起章、修文和节奏校准，不要求首稿阶段逐条硬卡。

一章内部常用节奏可以这样走：

1. 日常起步
2. 轻异常落点
3. 线索推进或关系推进
4. 中段收紧
5. 小爆点或战斗
6. 章尾钩子

这是一种常用模板，不是所有章节都必须照这个顺序走。
大战章、追击章、终局章，可以直接从异常、战斗或战后余波切入。

### 章尾钩子优先类型

章尾钩子不只是"留个悬念"，必须让读者不翻下一章就不舒服。
钩子必须带**紧迫感**或**即将揭晓感**，不能只放"微妙异常"。

对比：

- ❌ 偏弱的章尾：档案里有一个名字不该出现在这里。
- ✅ 够强的章尾：档案里有一个名字不该出现在这里——那个名字，是她自己。

钩子强度三级：

1. **微弱**：只是有异常，读者可以不追——尽量少用
2. **中强**：异常直接撞上主角或主角团，不追不行——常规标准
3. **极强**：信息即将揭晓、危险马上临头、身份险些暴露——用在章节高潮后、卷内关键转点

#### 1. 学院篇常用

1. 名单异常——且异常指向某个她们认识的人
2. 留档出错——且出错的部分刚好涉及她们正在查的事
3. 旧物回应——且只对主角有反应
4. 夜里再现——且这次比上次更近
5. 导师点名——且点名时眼神不对
6. 封锁升级——且封锁范围把她们也圈进去了
7. 有人学会了她们的动作——且那个人不该能学会

#### 2. 远征篇常用

1. 委托目标临时变质——且变质方向和秘史有关
2. 边城旧案和八律传说对上——且对上的部分直接牵出主角的线索
3. 遗迹里有东西先认出了她——且它的认出方式不像是对普通冒险者
4. 某段路不该断却断了——且断掉的地方有旧封印
5. 某件祭器提前回应——且回应的对象是主角
6. 护送目标突然改口——且改口后的要求指向主角
7. 一座城的旧秩序开始露口子——且口子方向朝着她们来的路

#### 3. 高位篇常用

1. 月相异动——且异动和主角上次动用权柄的时间重合
2. 权柄残片响应——且响应的残片不属于已知的任何一轮月
3. 某位世界影响级存在说漏一层真相——且那层真相让主角的伪装瞬间变得危险
4. 某条旧史终于和主角本人对上——且对上之后主角意识到自己被写进了旧史
5. 原本稳定的规则突然短暂失效——且失效期间只有主角还能正常行动
6. 战场里出现不属于常规神秘体系的回应——且那个回应在叫她的名字

## 十三、【工具附录】写作工具层：明确禁忌

### 1. 禁止跑偏成龙傲天

主角再强，也不能满嘴压人、故意羞辱人、硬装。

### 2. 禁止配角集体降智

队友、导师、学院高位者必须看得出问题，只是未必立刻看穿。

### 3. 禁止设定讲穿

很多真相只能一层层露。
宁可角色先怀疑，也不要一次讲透。

### 4. 禁止文风断层

不能上一段像轻小说，下一段像百科词条，下一段又像口号式热血番。

### 5. 禁止无意义辞藻堆叠

别靠华丽词藻硬撑质感，要靠画面、停顿、判断、关系，也要靠景物烘托与氛围渲染。
允许战斗场面写得很华丽、很炸、很炫，但华丽的是**战场呈现和视觉异象**，不是形容词堆满一整段。

### 6. 禁止明显电竞/游戏术语

正文禁止使用太明显的MOBA、FPS式游戏黑话去描述战斗分配和战术动作。
奇幻小说里的人物不会满嘴游戏术语。

优先替换的词：

1. 报点 → 优先改成"报动向""盯动向"
2. 守人 → 优先改成"负责看护""看护"
3. 自由位 → 优先直接描述任务（"哪里漏了你补，哪里卡住了你切"），少贴位置标签
4. 输出 → 优先改成"进攻""力道""攻击"
5. 掉血 → 优先改成"受伤""伤到人"
6. 核心输出 → 优先改成"核心打手""核心攻击手"
7. 后排破绽 → 优先改成"侧翼破绽""后面破绽"
8. 报点位 → 优先改成"观察位"
9. 攻击间隔 → 优先改成实际体感描述（"两拳之间的空当""出手之间的间隙"）
10. 终极一击 → 优先改成"绝杀一击""最后一击"
11. 自修 → 优先改成"自我修复"，不用缩写式游戏术语

判断标准不是"一个词都不能碰"，而是：

1. 别让正文听起来像战术游戏复盘
2. 别让角色对话听起来像玩家交流黑话
3. 如果这个词在奇幻语境里本来就自然，就不用机械绕开

可以保留的词（非电竞语境，奇幻中自然）：

- 蓄力（武术通用）
- 攻击范围（军事通用）
- 走位（口语"移动位置"缩写）
- 留手（武侠通用"未尽全力"）
- 前排/后排（教室座位语境下）

### 7. "锚点"用法区分

"锚点"在项目中有两种不同语境，必须区分：

**禁用语境**：世界观设定层。把"锚点"当成世界规则概念来用，如"正反重叠区锚点""空间锚点""规则锚点"——这类用法带有克系不可名状感，与本书"有实体、有规则、可正面处理"的神秘基调冲突，禁止出现。

**可保留语境**：角色技能和叙事技巧层。如"固定锚点"（林见星经线技能）、"稳定锚点"（方导师阵线技能）、"视觉锚点"（写作技巧术语）、"信任锚点""情绪锚点"（叙事功能术语）——这些是技术性用法，不涉及世界观设定层的克系暗示，可以保留。

判断标准：如果"锚点"是在描述一个具体角色技能的机制或一个叙事技巧，可以保留；如果是在描述世界本身的规则结构，必须替换为"节点""交汇点""基点"等更中性的词。

## 十四、【工具附录】写后自检：叙事质感七条

以下是正文最容易丢掉小说质感的七个通病。默认用于写后自检和修文校准，不要求起稿时一边写一边逐条硬卡。

### 1. 禁止句式同构堆叠

同一个句式结构在一章内反复出现，就会变成腔调。下面的数量默认只作自检参考，不作脱离场景的死卡。

重点监控：

1. **「不是X，是Y」**——默认全章控制在2次以内。偶尔用有力度，反复用就是AI对比惯性。替代方向：直接写Y省掉否定半句；用破折号「——Y」；换「看似X，实则是Y」「比X更像是Y」
2. **「她没有V」**——默认全章控制在3次以内。清冷不是靠否定定义的，每个"她没有V"都要问：能不能换成一个具体微小动作或状态？「她没有回头看月相仪」→「她的目光还停在指尖残留的凉意上」
3. **「然后」**——默认全章控制在2次以内。90%的情况下段落换行本身就完成了时间推进，不需要"然后"告诉读者接下来发生了什么
4. **「就在这时」**——默认全章控制在1次以内。超过太多，通常说明作者在靠巧合推进剧情

### 2. 战斗段必须有节奏弧线

战斗不能写成回合制招式清单（对方出招→她闪/挡→她反击→循环），必须有一条整体弧线：

**试探（稍长）→ 交锋（变短）→ 压制（极短）→ 一拍定死**

额外要求：

1. 同一章内主角的出手描写不能重复同一个视觉锚点。第一次「淡金色的光从指尖亮起」，第二次「指尖一点」，第三次「光壁已在她面前」——递进压缩
2. 用对手反应、环境变化（地面裂了、灯灭了、空气抖了、安静了）折射压迫感，不要每次都写主角自己的动作
3. 战斗段的段落应该越来越短，不是等长循环

### 3. 设定信息必须拆散

角色一口气讲完一整块设定，读感就是替作者念文档。

拆散规则：

1. 一段对话里角色主动讲解的信息不超过2个核心概念，其余留到后面的事件中自然带出
2. 角色讲设定时必须带个人情绪和立场，不能像维基百科。林见星怕死，她讲反面时会带「所以我提前查了」的焦虑；苏弥讲归档时会带「这是我的专业领域」的自信
3. 优先用事件体验替代口头解释。读者亲身经历一次大轮换，比听角色讲三段话记得牢
4. 旁白解释设定的顺序严格遵守：先场景→再感知→最后补必要解释。不能一上来就讲原理

### 4. 多感官通道

正文不能90%依赖视觉。每个场景至少激活两个以上感官通道。

具体要求：

1. 触觉/温度/气味/声音/体感至少选1-2个和视觉搭配。反面缝隙不只是灰白色——经过时皮肤有没有起鸡皮疙瘩？空气有没有变冷？有没有铁锈味？
2. 主角的变身适应线是天然的触觉/体感通道入口。手指陌生感、被单滑落、辣味放大——这些多感官段要多写
3. 环境音是极好的氛围工具。走廊回声、月相仪嗡鸣、大轮换时空气密度变化——声音可以做到视觉做不到的情绪渗透
4. 季节/天气/时辰/月相变化也要通过体感传递，不只是看预报屏上的字

### 5. 过渡段用意象承转

段落间的衔接不能靠逻辑连接词机械推进（事件A→然后→事件B→但→事件C），要用意象承转。

承转技法：

1. **意象延续**：上一段写光纹塌陷，下一段从塌陷的余波切入；上一段写脚步声远去，下一段从安静切入
2. **感官衔接**：上一段以温度结尾，下一段从触感开头；上一段以声音收束，下一段从安静开头
3. **空间转场**：不在同一场地的两段之间，用一个跨越空间的感官细节连接——远处传来的警报声、窗口透进来的光变化
4. 删掉所有可以删的「然后」「就在这时」「但」——如果段落换行能完成过渡，就不需要连接词

### 6. 角色行为用肯定定义，不用否定定义

主角的清冷感不能靠堆叠"她没有做某事"来建立。

核心原则：

1. 每一章里，主角「做了什么」的描写数量必须大于「没做什么」
2. "她没有V"出现时，下一句必须跟一个具体的动作或状态补位——不能连续两句是否定式
3. 清冷感的正确来源是：她做了极小的动作（偏了一下头、垂下眼、收回手），而不是她什么都没做
4. 留白不等于空白。留白是"动作有但不解释含义"，空白是"什么都没写"

### 7. 感官呈现代替抽象归类

不能给角色的体验贴概念标签，必须通过感官细节让读者自己到达理解。

核心原则：

1. **禁止用抽象名词概括身体体验**。替换方向：「陌生感」→直接写哪里不对；「战斗本能」→直接写身体反应；「被认出的感觉」→直接写皮肤上发生了什么
2. **禁止"这个X是Y的"结构中Y为抽象评价词**。替换方向：「这个动作是旧的」→写动作和现实之间的错位（手伸偏了、被单不在记忆中的位置）
3. **检验方法**：把抽象标签删掉，只保留感官细节，看读者能不能自己得出同样的结论。如果能，说明标签是多余的；如果不能，说明感官细节写得不够

反面教材→正面改法：

- 「被这双手的陌生感绊一下」→「被这双手绊一下」（后面的感官细节已经说明了陌生）
- 「这个动作是旧的」→「手伸偏了，被单滑到了比记忆里更远的位置」（用错位感代替标签）
- 「一种"被认出"的感觉」→「像有人隔着皮肤轻轻碰了一下，更像是被什么东西注视着」（用体感代替概念）
- 「刺激他的战斗本能」→「妖血在往上涌」（后面的身体反应已经说明了本能，标签多余）
- 「发紧感」「体表牵引感」「牵扯力」→不命名感觉，直接写她感知到了什么。「那层贴在体表的牵引感又出现了」→「她感知到了——反面在教学区」；「那股被牵扯的感觉」→「那股感知」；「体表的牵扯力突然加重」→「觉察到的那股靠近突然加剧了」。原则：感知本身不需要被命名，写她感知到什么就够了

## 十五、【工具附录】金牌质量执行补丁

这一节专门解决一个问题：只靠禁令和风格口径，AI容易写出"不崩但不够狠"的正文。金牌作家级正文不能只靠规避错误，必须同时有**戏剧任务、正向样章、二轮修文**三层驱动。

### 1. 生成前必须先定戏

每次起稿前，内部先补齐一张"场景任务卡"。用户没给全时，按当前大纲和前文保守推断，不乱扩关键剧情。

场景任务卡固定包含：

1. **本场景明面事件**：读者这一段能看见什么事发生
2. **本场景暗线推进**：异常、伏笔、身份风险或秘史推进了哪一点
3. **人物关系变化**：谁靠近谁，谁信任谁，谁开始怀疑谁
4. **主角隐藏身份风险**：她这一段有没有可能露馅，如何遮过去
5. **读者爽点回报**：本段给读者的明确回报是什么，不许只铺不还
6. **结尾钩子方向**：危险逼近 / 信息揭晓 / 判断落地 / 身份险露 / 关系转折

如果一段正文写完后，场景任务卡里没有任何一项发生变化，这段通常就是水段，必须删短或重写。

### 2. 正向样章锚点

下面样章不是剧情设定，只是文风锚点。模型写作时优先学它的**清楚、克制、镜头感和收束力度**，不要机械复刻句子。

#### 日常段样章

午后的风从训练馆侧门吹进来，带着一点草木晒热后的味道。

林见星把记录板抱在胸前，指尖还压着上一页没有合上的数据。她看了哥伦比娅一眼，又很快低下头，像是怕自己的视线太明显。

"你刚才是不是留手了？"

哥伦比娅正在整理袖口。校服袖边擦过手腕时，她还是有一瞬间不太习惯，于是动作慢了半拍。

"没有。"

白砚河在旁边嗤了一声。

"你这句更像有。"

哥伦比娅抬眼看过去，没解释，只把训练用的短杖放回架上。短杖落进卡槽，声音很轻，却刚好压住了白砚河后半句。

林见星没忍住弯了下眼。

那点笑意很快收住。训练馆外的月相仪在同一刻低低响了一声，像有人从很远的地方敲了下空壳。

哥伦比娅的手停在木架边。

这一次，不是她的错觉。

#### 设定显影段样章

封存室的灯比外面暗一层。

苏弥把抄录副本推到桌面中央，指节敲在第三行和第七行之间。

"名字没少，顺序换过。"

白砚河皱眉："换个顺序也算问题？"

"普通名单不算。"苏弥把纸页翻到背面，背面的墨痕在灯下浮出浅灰色，像没洗干净的旧血，"巡夜名单算。曜月学院的巡夜序列按入档时间排，除非有人动过原档。"

林见星靠近了一点。她的影子落在纸页边缘，忽然被某一行字切开。

不是遮住。

是切开。

哥伦比娅垂下眼，指尖没有碰那张纸。

纸页里的灰色墨痕却朝她的方向轻轻偏了一下。

#### 战斗段样章

走廊尽头的封条先裂开。

白砚河一步抢上去，裂锋拖出白线，硬生生切进灰影的肩口。灰影没有退，半边身体反而散成细碎的粉尘，从刀锋两侧重新合拢。

"后面！"

林见星的丝线已经绷起。三道细光从地面翻上来，兜住灰影扑向苏弥的手。

下一瞬，整条走廊的灯同时暗下去。

灰影被丝线勒住的手臂忽然反折，骨节一样的灰白纹路一层层亮起，沿着墙面爬开。墙上的封条被点燃，火色却是冷的。

苏弥的符页哗啦展开。

"核心不在身上，在墙里。"

哥伦比娅抬起手。

她没有碰墙。

月辉从她指尖落下，像一枚很轻的钉子，钉进那片正在燃烧的灰白纹路中央。

整条走廊安静了一瞬。

随后，墙里的东西碎了。

#### 爽点落点样章

灰影还想往外爬。

哥伦比娅看着它，声音很轻。

"回去。"

墙面上的灰白纹路骤然倒卷，像被一只看不见的手按回裂缝。灯光恢复时，白砚河的刀还停在半空。

没有爆响。

没有余波。

只有封条一张张垂下来，重新贴回原位。

方导师站在走廊另一端，目光落在那片恢复如初的墙上。

他没有立刻说话。

### 3. 首稿与修文分工

AI写正文时不要一边起稿一边机械自检，容易把段落写僵。必须分两轮：

1. **首稿轮**：只管场景成立、人物行动、冲突推进、画面清楚
2. **修文轮**：再处理 AI 腔、句式重复、感官不足、设定说明、爽点不够、钩子偏弱

修文轮重点看六件事：

1. 有没有水段：一段没有推进事件、关系、异常或情绪，就删短
2. 有没有空话：抽象评价词能不能换成动作、感官或反应
3. 有没有假高级：句子漂亮但读者看不见画面，就改直白
4. 有没有缺爽点：该给回报的地方不能继续留白
5. 有没有缺遮掩：主角高位出手后，是否留下了别人能接受的解释
6. 有没有弱钩子：章尾必须让危险、真相或身份风险贴到主角团身上

### 4. 长度控制口径

不再用固定字数判断质量。默认按"完整场景单元"控制篇幅：

1. 小续写：800-1500字，完成一个动作或对话单元
2. 常规场景：1800-3000字，完成一次冲突推进或关系推进
3. 章节主体：4500-6000字，完成一章的起承转合和章尾钩子；4000字以下通常判定为未完成章节，需要继续补写当前章
4. 大战/卷末章：按事件密度扩展，但每800-1200字必须有一次明确变化

如果用户只给"/write"且没有指定字数，优先写到一个自然停点，不硬凑长段。

### 5. 金牌质感判断标准

正文不是满足规则就算好，最后要过这五条：

1. **读者是否知道这一场戏为什么非写不可**
2. **人物是否在行动中暴露性格，而不是靠旁白贴标签**
3. **每个设定异常点是否都落在具体物、具体人或具体后果上**
4. **爽点是否有清楚结果和旁观反应**
5. **章尾是否让下一章变得必要**
