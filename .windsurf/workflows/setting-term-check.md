# /setting-term-check

Description: 按 Trae 定稿前检查 skill 执行 `/setting-term-check`。

## Source of truth
- 读取并服从 `.trae\rules\三月女神定稿前检查skill.md`。
- 术语裁决以 `设定\术语口径统一表.md`、`设定\神秘设定.md`、`设定\战斗设定.md`、`设定\当前硬口径.md`、`设定\核心技与解放归档.md` 和 `设定\术式吟唱归档.md` 为准。

## Steps
1. 只核对体系术语，不套完整 `/review`。
2. 检查魔力体系回潮、以太误用、个人核心/核心技/通用术式混淆、术式编号或吟唱漂移、真名解放/界临/立序误用、污染进阶错名、主角体系外机制被写成真实无吟唱、月相供能误写和月亮体系越界。
3. 对每处命中给出冲突口径、定稿依据和原位改法。
4. 发现术语表未覆盖的新冲突，标注为口径缺口。
5. 按 Trae skill 固定格式输出。
