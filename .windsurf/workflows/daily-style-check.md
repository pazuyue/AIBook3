# /daily-style-check

Description: 按 Trae 写作风格 skill 执行 `/daily-style-check` 日常场检查。

## Source of truth
- 读取并服从 `.trae\rules\三月女神写作风格skill.md`。

## Steps
1. 只检查日常场写法，不做完整审稿。
2. 核对关系推进、变身适应可见落点、日常里的轻微异常和人物对话辨识度。
3. 变身适应只看社交和生活层面；发现走路、步幅、重心、动作校准或诊断式旁白时，给出原位改法。
4. 避免闲聊灌水、说明书腔、同声道和普通目光被写成危险凝视。
5. 给出原位修文建议；用户授权时按风格 skill 写回。
6. 不改变设定、剧情目标或人物关系阶段。
