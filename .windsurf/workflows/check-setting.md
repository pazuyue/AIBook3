# /check-setting

Description: 按 Trae 审稿 skill 执行 `/check-setting`。

## Source of truth
- 读取并服从 `.trae\rules\三月女神审稿skill.md`。
- 设定裁决以 `设定\当前硬口径.md` 和各权威设定文件为准。

## Steps
1. 只检查设定文件或设定表达，不写正文。
2. 核对设定打架、战力失衡、技能命名、神秘学讲透、术语口径、反面边界、八律现实利益和高位战力尺度。
3. 发现权威源缺失时标注口径缺口，不自行补设定。
4. 给出是否需要调整、具体改哪里、为什么改和推荐落地文案。
5. 用户授权时可按最小改动调整设定表达。
