# /review

Description: 按 Trae 审稿 skill 执行 `/review`。

## Source of truth
- 读取并服从 `.trae\rules\三月女神审稿skill.md`。
- 审稿前先核对 `设定\当前硬口径.md`。

## Steps
1. 只做完整审稿或综合校准；单项轻量检查改用 quick-check 对应命令。
2. 判断对象类型：正文、细纲、大纲、设定、skill/规则/工作流文件。
3. 优先检查设定越界、当前卷边界、隐藏身份风险、战斗逻辑，再看人物、节奏、文风。
4. 审稿必须落到原文具体位置，给原位微调方案。
5. 按 Trae skill 的审稿模式或校准模式输出。
