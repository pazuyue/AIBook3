# /quality-card

Description: 按 Trae 新章节规划 skill 执行 `/quality-card`。

## Source of truth
- 读取并服从 `.trae\rules\三月女神新章节规划skill.md`。

## Steps
1. 只输出写前质量任务卡，不写正文。
2. 基于当前 `/chapter-plan`、当前卷状态、写作设定包和口径卡片生成质量目标。
3. 明确本章创作指标、禁区、场景重点和读者追读理由。
4. 若本章含变身适应场景，质量卡必须输出以下勾选项，用户确认全部通过后方可进入 `/write`：
   - [ ] 变身适应只写社交感知层面（被看、被对待、衣着、长发），无走路/步幅/重心等运动功能适应
   - [ ] 无"精确计算""机械感""精密仪器"等机器/怪异比喻描述身体动作
   - [ ] 无"过分端正""用力太均匀""节奏太稳""没有半点迟疑"等诊断式旁白；异常通过动作结果、旁人反应和角色补救呈现
   - [ ] 性别体验前置；归一之躯超自然精密只作附带后置，且用体感表达（如"几乎听不见脚步声"）
   - [ ] 读感是"别扭、新鲜、慢半拍"，不是"非人机体校准中"
5. 写入当前卷 `chapter_plan_cache.md` 或 Trae skill 指定缓存。
6. 提醒后续需要人工填写 `/write-brief`。
