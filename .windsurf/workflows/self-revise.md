# /self-revise

Description: 按 Trae 写作风格 skill 执行 `/self-revise` 整章自修。

## Source of truth
- 读取并服从 `.trae\rules\三月女神写作风格skill.md`。
- 必须尊重 `/word-check` 闸门。

## Steps
1. 先确认当前章已通过 `/word-check`；低于 4000 字不得自修。
2. 只做正文工艺修订，不改变剧情骨架。
3. 检查镜头、动作线、感官、对话、节奏、章尾钩子和主角出手遮掩理由。
4. 发现设定/剧情层问题时停止，提示改用 `/chapter-review`、`/plot-review` 或 `/rewrite-plan`。
5. 按 Trae skill 写回或输出。
