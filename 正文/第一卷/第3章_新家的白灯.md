# 第3章 新家的白灯

哥伦比娅在第七救助医院住了二十七天。

第一周，她主要学会了三件事。

走廊尽头的热水机要按蓝色键。

午餐车经过病房门口时，如果不出声，医官也会给她留一份。

夜里三声短促警报不是火灾，而是反面波动预警演练，听见后要待在原地，等走廊里的蓝线亮起来。

这些都是小事。

小到放在任何一份个人档案里都不值一提。可对她来说，这些微不足道的日常就像一枚枚钉子，在陌生的现实里把她牢牢钉住。有了这些，她才能确认自己是真的活在这个房间里，而不是随时会掉回那座死寂的灰白废墟。

闻澈每天都会来。

有时候是早上，有时候是深夜。他的白大褂外面常带着一点冷雨味，袖口偶尔沾着消毒水洗不掉的浅痕。来得早，他会给她带医院食堂刚出锅的豆乳和小面包；来得晚，就坐在床边那张椅子上，翻灾后安置条例，等她把当天的药吃完。

他不问她从哪里来。

也不问她为什么对这个世界的一切都生疏。

他只把该知道的东西拆成很小的份，一点点告诉她。

"曜月城在学院城外圈，靠近几处监管反面，所以救助医院多。"

"反面不是每天都会出现。大多数时候，普通人一辈子也只在新闻里见。"

"预警灯分三色。白色演练，蓝色待命，红色撤离。你只要记住这一条就够了。"

"月相仪是学校和神秘机构用来做适配检测的设备，不是玩具。"

哥伦比娅听得很认真。

她听不懂的词很多，却很少打断。遇到实在不明白的地方，她会把那个词记在医院发的便签本上，等闻澈说完，再把本子推过去。

便签本第一页写着：

反面。

以太。

神秘者。

月相仪。

曜月学院。

第二页写着：

门禁卡不能放进微波炉。

热水机蓝色键。

红色警报先找最近的蓝线。

闻澈第一次看见第二页时，沉默了三秒，说："这条很重要。"

哥伦比娅点头，把"很重要"三个字补在旁边。

她的日子就这样慢慢有了形状。

早上量体温，吃饭，做恢复训练。午后在医院安置区的小教室补基础常识，认识三月历、城市地图和公共终端。傍晚闻澈如果有空，会带她去楼下花园走一圈。

恢复训练最初也很奇怪。

医官让她沿着白线走十米，她走完以后站在原地，本能地等胸口那阵熟悉的刺痛蔓延开来。

没有疼。

让她扶着栏杆上下楼梯，她走到一半时下意识地停住，准备迎接那种随时会抽空力气的眩晕感。

也没有。

这具身体健康、轻盈，甚至有些过于陌生了。

对她来说，最需要花时间适应的，反而是一些以前从未在意过的琐碎细节。比如转身时长发会慢半拍扫过肩背，睡觉翻身时总会不小心压到发尾。

她用了好几天才慢慢习惯，在每天清晨起身前，要先抬手把压在颈后的长发拨出来；低头喝水时，需要用手指轻轻按住滑落到胸前的发丝；在走廊里遇到别人时，大家似乎总会下意识地放轻脚步，甚至主动为她让出一条更宽敞的路。

这些看起来都不像什么大事。

可她就是这样，通过一次次理顺被风吹乱的发尾，一次次感受着落在自己身上的温和视线，一点一点地，把这具陌生的少女身体，过成了真实存在于这个世界的样子。

她开始能分清医院和学院的徽记。也知道了方芷凝导师是第三区事故现场的带队人，负责跟进她那份事故档案。

但方导师很少出现。

她只在走廊尽头见过那位冷淡的女导师两次。对方没有盘问她，也没有把她当成犯人，只在医官汇报时远远看过来一眼。

那一眼没有停留。

哥伦比娅却觉得，自己已经被写进某份报告里。

她低下头，继续数便签纸上的格子。

不能让别人知道。

这个念头仍旧在。

比身份不明更深，比陌生世界更深。

因为她开始发现，这具身体偶尔会做出一些不该有的事。

第一次是在病房。

她半夜醒来，口渴，伸手去拿床头的水杯。指尖碰到玻璃杯壁的瞬间，杯子里的水面忽然平了。

不是普通的平稳。

而是连她呼吸带起的细纹都被按住，水面像一小片被抹平的白光，静得没有一丝波纹。

她盯着那杯水看了三秒，慢慢松手。

水面重新晃开。

第二天早上，闻澈拿起那个玻璃杯。杯壁上留着一圈很淡的白痕，摸上去比周围凉一点。

哥伦比娅坐在床上，手指压着被角，没有说话。

闻澈也没有问。

他只是把杯子换成了一只不透明的厚瓷杯。

"厚瓷杯不容易凉。"他说。

此时正是夏末，天气还带着闷热。

哥伦比娅低头接过杯子，轻轻说："谢谢。"

闻澈把那只玻璃杯收进柜子里面。

第二十八天，灾后安置处下发临时监护手续。

流程比她想象中简单，也比她想象中郑重。闻澈带她去了一间很小的办公室，工作人员确认她的身体恢复情况、基础识字能力和个人意愿，又把一份临时监护书推到她面前。

"你可以继续住院，也可以进入安置家庭。闻澈医官提交了监护申请，但需要你本人同意。"

工作人员说得很慢。

"这不是必须。你可以拒绝。"

哥伦比娅看向闻澈。

闻澈坐在旁边，手放在膝上，没有替她回答。

他坐得很端正，肩膀绷着，袖口还留着早上出急诊时被器械勾出的线头。他的呼吸比平常浅了一点。

哥伦比娅把视线移回纸上。

监护人：闻澈。

被监护人：哥伦比娅·希珀塞莱尼娅。

关系类型：灾害失依未成年人临时收养观察。

她读完，拿起笔。

笔尖没有抖，一笔写完，没有停顿。

闻澈带她回家的那天，曜月城下了一场小雨。

他住在医院和处置队驻地中间的一栋旧公寓里。楼下有二十四小时便利店，门口贴着反面预警说明；电梯有些慢，运行时会发出很轻的金属声。

房子不大。

客厅靠窗摆着一盏白色落地灯，灯罩边缘有被修过的痕迹。餐桌上放着两套餐具，一大一小。冰箱门上贴着一张手写清单：

牛奶。

鸡蛋。

儿童拖鞋。

发绳。

她站在玄关，脚尖抵着拖鞋边沿，没换鞋。

闻澈把雨伞收好，弯腰把那双新拖鞋放到她脚边。

"左边是你的房间。床单新洗过。"

她还是站在原地。

闻澈也不催，只把门禁卡和一串钥匙放进玄关的小碗里。

"这张卡能开楼下门禁。这把钥匙能开家门。"

他说到这里，停了一下，确认这个词不会吓到她。

最后他还是说了。

"回家用的。"

哥伦比娅低头看那把钥匙。

金属齿口很小，挂在浅蓝色钥匙扣上，钥匙扣背面写着她的名字。

她伸手拿起来。

钥匙很轻。

握进掌心时，她第一次切实体会到，这里有一双鞋、一把钥匙、一盏灯，是为她准备的。

闻澈走进厨房，打开那盏白色落地灯。

暖光铺开，照亮玄关、餐桌、冰箱上的清单，也照亮她脚边那双小小的拖鞋。

"进来吧，哥伦比娅。"

他说。

"饭快好了。"
