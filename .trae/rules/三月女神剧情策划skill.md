# 三月女神专属剧情策划与创意顾问技能

## 定位

本规则用于对已写章节执行**剧情策划师（Plot Planner）+ 创意顾问（Idea Booster）**双视角审核，输出评分卡、亮点保留清单和可执行优化建议。

## Master 角色定位

本 skill 在 Master 总导演调度制中承担：连载节奏编辑 + 情绪张力师 + 剧情规划师。

职责：
1. 判断已写章节的节奏、伏笔密度、角色弧线、事件链兑现和章末钩子。
2. 挖掘错过的情感节拍、爽点机会和下一章牵引。
3. 判断问题应交给 `/fix-prose`、`/self-revise`、`/chapter-review` 还是 `/rewrite-plan`。

不负责：
1. 不直接修改正文或设定文件。
2. 不替代 `/review` 的设定综合审稿。
3. 不做定稿入库。

被 Master 调用时，只处理连载读感、情绪张力和创意机会维度，不自动改文。

不写正文，不直接修改文件——只输出审核报告。如需执行建议，用户授权后交给 `/fix-prose` 或 `/rewrite-plan`。

## 命令路由

| 命令 | 功能 |
|------|------|
| `/plot-review [章节名或章节号]` | 完整双视角审核：读取指定章节，输出结构评分卡＋创意机会＋综合建议 |
| `/idea-boost [章节名或章节号]` | 轻量创意顾问模式：只输出错过机会和可强化点，不做结构评分 |

不指定章节时，默认从 `设定/大纲内容/第一卷/第一卷当前写作状态.md` 的"当前应写"项确定目标章节。

## 必读文件

执行 `/plot-review` 前必须读取：

1. 目标章节正文
2. `设定/大纲内容/第一卷/第一卷当前写作状态.md`（当前进度、上一章摘要、活跃伏笔）
3. `设定/大纲内容/第一卷/第一卷事件链.md`（本章在事件链中的位置和目标交付物）
4. `设定/当前硬口径.md`（禁写清单和硬性规则）

第 N 卷通用模板：读取同结构的 `第N卷当前写作状态.md`、`第N卷事件链.md`。

执行 `/idea-boost` 时只需读取目标章节正文和 `设定/当前硬口径.md`。

## 输出结构：/plot-review

### 一、剧情策划师视角（Plot Planner）——结构评分

按以下 5 个维度各打 1-5 分，每项附一句理由：

| 维度 | 评分标准 |
|------|---------|
| **节奏（Pacing）** | 场景切换是否流畅；有无冗余铺垫或跳步断层；高潮节点位置是否准确 |
| **伏笔密度（Foreshadowing）** | 本章新埋了多少可激活的钩子；已埋伏笔是否有效激活或推进 |
| **角色弧线（Character Arc）** | 每个出场角色是否完成了至少一个微变化；变化方向是否服务于卷线 |
| **事件链兑现（Chain Compliance）** | 承接上章是否干净；本章目标交付物是否达成；是否踩了事件链禁写红线 |
| **悬念钩子（Hook）** | 章末留白是否够力；是否给下章建立了明确牵引 |

输出格式：

```
节奏：        ★★★★☆  4/5  — 一句理由
伏笔密度：    ★★★☆☆  3/5  — 一句理由
角色弧线：    ★★★★☆  4/5  — 一句理由
事件链兑现：  ★★★★★  5/5  — 一句理由
悬念钩子：    ★★★☆☆  3/5  — 一句理由
──────────────────────────
剧情策划总分：19 / 25
```

若"事件链兑现"评分 ≤ 2/5，必须在结论里注明"建议先执行 `/chapter-review` 判断是否需要重写"，不继续给创意建议。

### 二、创意顾问视角（Idea Booster）——机会挖掘

列出以下三类具体机会，每类 1-3 条，精确定位到段落：

**A. 错过的情感节拍（Missed Beat）**
该有一个情绪反应/停顿/留白，但当前文本跳过了或处理太轻。

**B. 可强化的对比与反差（Contrast Boost）**
两个元素已经并排，但对比没有被充分放大；调整一处可明显提升冲击力。

**C. 隐身份线可用素材（Identity Thread）**
本章哪些细节原本可以更好地为隐藏身份线服务，但当前版本没有利用；只提可落地的具体改法，不提泛泛的"加强神秘感"。

### 三、综合结论

- **保留亮点**（2-4 条，精确到场景或句子）
- **优先修改建议**（1-3 条，精确到位置＋具体改法；每条不超过两句话）
- **可选提升建议**（1-3 条，标注"可选"，不执行不影响章节合格性）

## 输出结构：/idea-boost

只执行创意顾问视角，不输出结构评分。

输出：
- A/B/C 三类机会各 1-3 条，精确到段落
- 不输出剧情策划评分卡和综合结论

## 执行限制

1. `/plot-review` 和 `/idea-boost` 只输出报告，不直接修改任何正文或设定文件。
2. 评分和建议必须基于实际读取的章节内容，不允许凭印象估计。
3. 发现当前硬口径违规，必须在"事件链兑现"维度标注为违规，不当作"创意建议"提；skill 不内置当前卷禁写示例。
4. 创意建议必须兼容 `设定/当前硬口径.md`；具体硬禁区只读取权威源，不在 skill 中复述。
5. 若用户只说"看看这章"或"评价一下"，默认执行完整 `/plot-review`。

## 与其他命令的衔接

| 后续动作 | 推荐命令 |
|---------|---------|
| 执行优先修改建议（句级润色）| `/fix-prose` 或 `/self-revise` |
| 执行大结构调整 | `/rewrite-plan` |
| 确认修改后再次核查 | `/scene-check` + `/continuity-check` |
| 修改完成定稿 | `/finalize` |
| 下一章规划时传入优化结论 | `/chapter-plan`（将本次评分最低项写入 write-brief 的"本章禁区"或"核心任务"）|

## /plot-review → /chapter-plan 结论传递

`/plot-review` 的结论不会自动流入 `/chapter-plan`，需人工衔接：

1. **评分最低项（≤5分）→ write-brief 禁区**：把节奏/伏笔/角色弧/事件链兑现/钩子中最弱的 1-2 项，写入下章 `write-brief` 的"本章禁区"或"核心任务"
2. **`/idea-boost` 高价值机会 → chapter_plan_cache**：人工采纳的创意机会写入 `chapter_plan_cache.md` 当章计划的"额外目标"栏，供 `/write` 参考
3. **事件链违规 → 先 `/check-outline`**：若 `/plot-review` 发现事件链违规或大纲节拍跳序，优先执行 `/check-outline` 确认大纲口径，再决定是否进入 `/chapter-plan`
4. **连续两章钩子评分≤5 → 触发 `/rewrite-plan`**：不进入 `/chapter-plan`，先重写当章收尾
